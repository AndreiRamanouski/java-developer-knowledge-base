## Overview

Advanced REST features enable building high-performance, scalable APIs with async processing, real-time updates, file handling, and cross-origin support.

**Key Concepts**: Non-blocking I/O, streaming responses, long-polling, and cross-origin resource sharing.

---

## 1. Async Processing

Async processing frees up servlet threads while waiting for long-running operations, improving throughput and scalability.

### Why Async Processing?

**Traditional (Blocking):**

```
Client Request → Servlet Thread (blocked waiting for DB/API) → Response
                     ↓
               Thread tied up for entire request duration
```

**Async (Non-Blocking):**

```
Client Request → Servlet Thread → Queue Task → Release Thread
                                      ↓
                                 Worker Thread Processes
                                      ↓
                                 Complete & Send Response
```

**Benefits:**

- Better thread utilization
- Handle more concurrent requests
- Don't block on I/O operations
- Improved server throughput

**Use cases:**

- External API calls
- Database queries
- File processing
- Report generation
- Email sending
- Long-polling
- Real-time notifications

### DeferredResult

`DeferredResult` allows you to produce a response from a different thread.

#### Basic Example

```java
@RestController
@RequestMapping("/api/async")
public class AsyncController {
    
    @Autowired
    private ExecutorService executorService;
    
    @GetMapping("/deferred")
    public DeferredResult<String> getDeferredResult() {
        DeferredResult<String> deferredResult = new DeferredResult<>();
        
        // Process in background thread
        executorService.submit(() -> {
            try {
                // Simulate long-running operation
                Thread.sleep(5000);
                
                // Set result when ready
                deferredResult.setResult("Processing complete!");
            } catch (Exception e) {
                deferredResult.setErrorResult(e);
            }
        });
        
        // Return immediately - servlet thread is freed
        return deferredResult;
    }
}
```

**Request flow:**

```
1. Client sends request
2. getDeferredResult() returns DeferredResult immediately
3. Servlet thread is released
4. Background thread processes task
5. When complete, setResult() triggers response
6. Client receives response
```

#### DeferredResult with Timeout

```java
@GetMapping("/deferred-timeout")
public DeferredResult<String> getDeferredWithTimeout() {
    // Timeout after 10 seconds
    DeferredResult<String> deferredResult = new DeferredResult<>(10000L);
    
    // Handle timeout
    deferredResult.onTimeout(() -> {
        deferredResult.setErrorResult(
            new TimeoutException("Request timeout after 10 seconds")
        );
    });
    
    // Handle completion (success or error)
    deferredResult.onCompletion(() -> {
        log.info("Request completed");
    });
    
    executorService.submit(() -> {
        try {
            Thread.sleep(5000);
            deferredResult.setResult("Success");
        } catch (Exception e) {
            deferredResult.setErrorResult(e);
        }
    });
    
    return deferredResult;
}
```

#### Real-World Example: External API Call

```java
@Service
public class ExternalApiService {
    
    private final RestTemplate restTemplate;
    private final ExecutorService executorService;
    
    public DeferredResult<ApiResponse> callExternalApi(String url) {
        DeferredResult<ApiResponse> deferredResult = 
            new DeferredResult<>(30000L);  // 30 second timeout
        
        executorService.submit(() -> {
            try {
                // Call external API (blocking I/O)
                ResponseEntity<ApiResponse> response = 
                    restTemplate.getForEntity(url, ApiResponse.class);
                
                deferredResult.setResult(response.getBody());
            } catch (Exception e) {
                log.error("External API call failed", e);
                deferredResult.setErrorResult(e);
            }
        });
        
        return deferredResult;
    }
}

@RestController
@RequestMapping("/api/external")
public class ExternalApiController {
    
    @Autowired
    private ExternalApiService externalApiService;
    
    @GetMapping("/data")
    public DeferredResult<ApiResponse> getData(@RequestParam String url) {
        return externalApiService.callExternalApi(url);
    }
}
```

### Callable

`Callable` is simpler than `DeferredResult` - Spring manages the async execution.

#### Basic Example

```java
@GetMapping("/callable")
public Callable<String> getCallableResult() {
    return () -> {
        // This runs in a separate thread automatically
        Thread.sleep(5000);
        return "Processing complete!";
    };
}
```

**Spring automatically:**

1. Executes Callable in task executor thread pool
2. Handles thread management
3. Returns result to client when ready

#### Callable with Complex Logic

```java
@GetMapping("/report/{id}")
public Callable<ReportResponse> generateReport(@PathVariable Long id) {
    return () -> {
        // Long-running operation
        log.info("Starting report generation for ID: {}", id);
        
        // Fetch data from database
        List<Data> data = dataRepository.findByReportId(id);
        
        // Process data
        Report report = reportService.process(data);
        
        // Generate PDF
        byte[] pdf = pdfService.generate(report);
        
        log.info("Report generation complete for ID: {}", id);
        
        return new ReportResponse(report.getId(), pdf);
    };
}
```

#### Configure Task Executor

```java
@Configuration
@EnableAsync
public class AsyncConfig implements AsyncConfigurer {
    
    @Override
    public Executor getAsyncExecutor() {
        ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
        executor.setCorePoolSize(10);
        executor.setMaxPoolSize(50);
        executor.setQueueCapacity(100);
        executor.setThreadNamePrefix("async-");
        executor.initialize();
        return executor;
    }
    
    @Override
    public AsyncUncaughtExceptionHandler getAsyncUncaughtExceptionHandler() {
        return (ex, method, params) -> {
            log.error("Async execution error in method: " + method.getName(), ex);
        };
    }
}
```

### CompletableFuture

Most powerful async approach - composable, chainable operations.

#### Basic Example

```java
@GetMapping("/completable")
public CompletableFuture<String> getCompletableResult() {
    return CompletableFuture.supplyAsync(() -> {
        try {
            Thread.sleep(5000);
            return "Processing complete!";
        } catch (InterruptedException e) {
            throw new RuntimeException(e);
        }
    });
}
```

#### Chaining Operations

```java
@GetMapping("/users/{id}/enriched")
public CompletableFuture<EnrichedUser> getEnrichedUser(@PathVariable Long id) {
    return CompletableFuture
        .supplyAsync(() -> {
            // Step 1: Fetch user
            return userService.findById(id);
        })
        .thenApplyAsync(user -> {
            // Step 2: Fetch orders
            List<Order> orders = orderService.findByUserId(user.getId());
            user.setOrders(orders);
            return user;
        })
        .thenApplyAsync(user -> {
            // Step 3: Fetch preferences
            Preferences prefs = preferencesService.findByUserId(user.getId());
            user.setPreferences(prefs);
            return user;
        })
        .thenApply(user -> {
            // Step 4: Enrich and return
            return new EnrichedUser(user);
        });
}
```

#### Parallel Execution

```java
@GetMapping("/dashboard/{userId}")
public CompletableFuture<Dashboard> getDashboard(@PathVariable Long userId) {
    // Execute multiple operations in parallel
    CompletableFuture<User> userFuture = 
        CompletableFuture.supplyAsync(() -> userService.findById(userId));
    
    CompletableFuture<List<Order>> ordersFuture = 
        CompletableFuture.supplyAsync(() -> orderService.findByUserId(userId));
    
    CompletableFuture<Statistics> statsFuture = 
        CompletableFuture.supplyAsync(() -> statsService.getStatistics(userId));
    
    CompletableFuture<List<Notification>> notificationsFuture = 
        CompletableFuture.supplyAsync(() -> notificationService.findByUserId(userId));
    
    // Combine all results
    return CompletableFuture.allOf(
        userFuture, ordersFuture, statsFuture, notificationsFuture
    ).thenApply(v -> {
        User user = userFuture.join();
        List<Order> orders = ordersFuture.join();
        Statistics stats = statsFuture.join();
        List<Notification> notifications = notificationsFuture.join();
        
        return new Dashboard(user, orders, stats, notifications);
    });
}
```

#### Error Handling

```java
@GetMapping("/data/{id}")
public CompletableFuture<DataResponse> getData(@PathVariable Long id) {
    return CompletableFuture
        .supplyAsync(() -> {
            // Might throw exception
            return externalApiService.fetchData(id);
        })
        .exceptionally(ex -> {
            // Handle error
            log.error("Error fetching data", ex);
            return DataResponse.error("Failed to fetch data: " + ex.getMessage());
        })
        .orTimeout(10, TimeUnit.SECONDS)  // Timeout after 10 seconds
        .handle((result, ex) -> {
            if (ex != null) {
                log.error("Request failed", ex);
                return DataResponse.error("Request failed");
            }
            return result;
        });
}
```

### WebAsyncTask

`WebAsyncTask` provides more control over async execution.

```java
@GetMapping("/webasynctask")
public WebAsyncTask<String> getWebAsyncTask() {
    Callable<String> callable = () -> {
        Thread.sleep(5000);
        return "Processing complete!";
    };
    
    WebAsyncTask<String> task = new WebAsyncTask<>(10000L, callable);
    
    // Timeout handler
    task.onTimeout(() -> {
        log.warn("Request timeout");
        return "Request timeout - processing continues in background";
    });
    
    // Completion handler
    task.onCompletion(() -> {
        log.info("Request completed");
    });
    
    // Error handler
    task.onError(() -> {
        log.error("Request error");
        return "An error occurred";
    });
    
    return task;
}
```

**With custom executor:**

```java
@GetMapping("/webasynctask-custom")
public WebAsyncTask<Report> generateReport(@PathVariable Long id) {
    Callable<Report> callable = () -> {
        return reportService.generate(id);
    };
    
    // Use custom executor

        30000L,                    // Timeout
        "reportExecutor",          // Executor bean name
        callable
    );
    
    task.onTimeout(() -> {
        return Report.timeout();
    });
    
    return task;
}

        executor.setMaxPoolSize(10);
        executor.setQueueCapacity(25);
        executor.setThreadNamePrefix("report-");

        return executor;
    }
}
```

### When to Use Async

#### Use Async When:

✅ **External API calls**

```java
@GetMapping("/weather")
public CompletableFuture<Weather> getWeather(@RequestParam String city) {
    // Don't block on external API
    return CompletableFuture.supplyAsync(() -> 
        weatherApiClient.getWeather(city)
    );
}
```

✅ **Database operations (with async drivers)**

```java
@GetMapping("/users/search")
public CompletableFuture<List<User>> searchUsers(@RequestParam String query) {
    return asyncUserRepository.searchByName(query);
}
```

✅ **File processing**

```java
@PostMapping("/process-file")
public DeferredResult<ProcessingResult> processFile(@RequestParam MultipartFile file) {
    DeferredResult<ProcessingResult> result = new DeferredResult<>();
    
    executorService.submit(() -> {
        ProcessingResult processed = fileProcessor.process(file);
        result.setResult(processed);
    });
    
    return result;
}
```

✅ **Report generation**

```java
@GetMapping("/report/{id}")
public Callable<byte[]> generateReport(@PathVariable Long id) {
    return () -> reportGenerator.generatePdf(id);
}
```

✅ **Email sending**

```java
@PostMapping("/send-email")
public CompletableFuture<Void> sendEmail(@RequestBody EmailRequest request) {
    return CompletableFuture.runAsync(() -> 
        emailService.send(request)
    );
}
```

#### Don't Use Async When:

❌ **Simple CRUD operations** (fast, local database)

```java
// NO need for async
@GetMapping("/users/{id}")
public User getUser(@PathVariable Long id) {
    return userRepository.findById(id).orElseThrow();
}
```

❌ **Memory-bound operations** (already fast)

```java
// NO need for async
@GetMapping("/calculate")
public int calculate(@RequestParam int a, @RequestParam int b) {
    return a + b;
}
```

❌ **When you have blocking dependencies in the chain**

```java
// Async doesn't help if you're blocking anyway
@GetMapping("/data")
public CompletableFuture<Data> getData() {
    return CompletableFuture.supplyAsync(() -> {
        // Still blocking the thread!
        Thread.sleep(5000);
        return data;
    });
}
```

---

### Comparison: Async Approaches

|Approach|Complexity|Control|Use Case|
|---|---|---|---|
|`Callable`|Low|Low|Simple async operations|
|`DeferredResult`|Medium|High|Custom async logic, event-driven|
|`CompletableFuture`|Medium|High|Composable operations, parallel tasks|
|`WebAsyncTask`|Medium|High|Need timeout/completion handlers|

## 2. Server-Sent Events (SSE)

SSE enables real-time server-to-client streaming over HTTP.

### SseEmitter

`SseEmitter` sends events from server to client.

#### Basic Example

```java
@RestController
@RequestMapping("/api/sse")
public class SseController {
    
    @GetMapping("/stream")
    public SseEmitter streamEvents() {
        SseEmitter emitter = new SseEmitter();
        
        ExecutorService executor = Executors.newSingleThreadExecutor();
        executor.execute(() -> {
            try {
                for (int i = 0; i < 10; i++) {
                    // Send event
                    emitter.send(SseEmitter.event()
                        .name("message")
                        .data("Event " + i)
                    );
                    
                    Thread.sleep(1000);  // Send every second
                }
                
                // Complete the stream
                emitter.complete();
            } catch (Exception e) {
                emitter.completeWithError(e);
            }
        });
        
        return emitter;
    }
}
```

**Client (JavaScript):**

```javascript
const eventSource = new EventSource('/api/sse/stream');

eventSource.addEventListener('message', (event) => {
    console.log('Received:', event.data);
});

eventSource.onerror = (error) => {
    console.error('SSE error:', error);
    eventSource.close();
};
```

#### Real-Time Notifications

```java
@Service
public class NotificationService {
    
    private final Map<Long, SseEmitter> emitters = new ConcurrentHashMap<>();
    
    public SseEmitter subscribe(Long userId) {
        SseEmitter emitter = new SseEmitter(Long.MAX_VALUE);  // No timeout
        
        emitters.put(userId, emitter);
        
        // Remove on completion
        emitter.onCompletion(() -> emitters.remove(userId));
        emitter.onTimeout(() -> emitters.remove(userId));
        emitter.onError((ex) -> emitters.remove(userId));
        
        return emitter;
    }
    
    public void sendNotification(Long userId, Notification notification) {
        SseEmitter emitter = emitters.get(userId);
        
        if (emitter != null) {
            try {
                emitter.send(SseEmitter.event()
                    .name("notification")
                    .data(notification)
                );
            } catch (IOException e) {
                emitters.remove(userId);
                emitter.completeWithError(e);
            }
        }
    }
    
    public void broadcastNotification(Notification notification) {
        List<Long> deadEmitters = new ArrayList<>();
        
        emitters.forEach((userId, emitter) -> {
            try {
                emitter.send(SseEmitter.event()
                    .name("broadcast")
                    .data(notification)
                );
            } catch (IOException e) {
                deadEmitters.add(userId);
                emitter.completeWithError(e);
            }
        });
        
        // Remove dead emitters
        deadEmitters.forEach(emitters::remove);
    }
}

@RestController
@RequestMapping("/api/notifications")
public class NotificationController {
    
    @Autowired
    private NotificationService notificationService;
    
    @GetMapping("/subscribe")
    public SseEmitter subscribe(@AuthenticationPrincipal User user) {
        return notificationService.subscribe(user.getId());
    }
    
    @PostMapping("/send")
    public void sendNotification(@RequestBody NotificationRequest request) {
        Notification notification = new Notification(request.getMessage());
        notificationService.sendNotification(request.getUserId(), notification);
    }
    
    @PostMapping("/broadcast")
    public void broadcast(@RequestBody NotificationRequest request) {
        Notification notification = new Notification(request.getMessage());
        notificationService.broadcastNotification(notification);
    }
}
```

### Streaming Responses

Stream large datasets without loading everything into memory.

```java
@GetMapping("/stream-data")
public ResponseEntity<StreamingResponseBody> streamData() {
    StreamingResponseBody stream = outputStream -> {
        try (BufferedWriter writer = new BufferedWriter(
                new OutputStreamWriter(outputStream))) {
            
            // Stream data line by line
            for (int i = 0; i < 1000000; i++) {

                if (i % 1000 == 0) {
                    writer.flush();
                }
            }
        }
    };
    
    return ResponseEntity.ok()
        .header("Content-Disposition", "attachment; filename=data.txt")
        .contentType(MediaType.TEXT_PLAIN)
        .body(stream);
}
```

## 3. File Upload/Download

### MultipartFile Upload

#### Single File Upload

```java
@RestController
@RequestMapping("/api/files")
public class FileUploadController {
    
    @Autowired
    private FileStorageService fileStorageService;
    
    @PostMapping("/upload")
    public ResponseEntity<FileUploadResponse> uploadFile(
            @RequestParam("file") MultipartFile file) {
        
        // Validate file
        if (file.isEmpty()) {
            return ResponseEntity.badRequest()
                .body(new FileUploadResponse("File is empty"));
        }
        
        // Check file size (max 10MB)
        if (file.getSize() > 10 * 1024 * 1024) {
            return ResponseEntity.badRequest()
                .body(new FileUploadResponse("File size exceeds 10MB"));
        }
        
        // Check file type
        String contentType = file.getContentType();
        if (!isValidContentType(contentType)) {
            return ResponseEntity.badRequest()
                .body(new FileUploadResponse("Invalid file type"));
        }
        
        // Store file
        String filename = fileStorageService.store(file);
        
        String downloadUrl = ServletUriComponentsBuilder
            .fromCurrentContextPath()
            .path("/api/files/download/")
            .path(filename)
            .toUriString();
        
        FileUploadResponse response = new FileUploadResponse(
            filename,
            downloadUrl,
            file.getContentType(),
            file.getSize()
        );
        
        return ResponseEntity.ok(response);
    }
    
    private boolean isValidContentType(String contentType) {
        return contentType != null && (
            contentType.equals("image/jpeg") ||
            contentType.equals("image/png") ||
            contentType.equals("application/pdf")
        );
    }
}
```

#### File Storage Service

```java
@Service
public class FileStorageService {
    
    private final Path fileStorageLocation;
    
    @Autowired
    public FileStorageService(@Value("${file.upload-dir}") String uploadDir) {
        this.fileStorageLocation = Paths.get(uploadDir).toAbsolutePath().normalize();
        
        try {
            Files.createDirectories(this.fileStorageLocation);
        } catch (Exception ex) {
            throw new RuntimeException("Could not create upload directory", ex);
        }
    }
    
    public String store(MultipartFile file) {
        // Generate unique filename
        String originalFilename = StringUtils.cleanPath(file.getOriginalFilename());
        String filename = UUID.randomUUID() + "_" + originalFilename;
        
        try {
            // Check for path traversal
            if (filename.contains("..")) {
                throw new RuntimeException("Invalid filename: " + filename);
            }
            
            Path targetLocation = this.fileStorageLocation.resolve(filename);
            Files.copy(file.getInputStream(), targetLocation, StandardCopyOption.REPLACE_EXISTING);
            
            return filename;
        } catch (IOException ex) {
            throw new RuntimeException("Failed to store file " + filename, ex);
        }
    }
    
    public Resource loadAsResource(String filename) {
        try {
            Path filePath = this.fileStorageLocation.resolve(filename).normalize();
            Resource resource = new UrlResource(filePath.toUri());
            
            if (resource.exists() && resource.isReadable()) {
                return resource;
            } else {
                throw new RuntimeException("File not found: " + filename);
            }
        } catch (MalformedURLException ex) {
            throw new RuntimeException("File not found: " + filename, ex);
        }
    }
    
    public void delete(String filename) {
        try {
            Path filePath = this.fileStorageLocation.resolve(filename).normalize();
            Files.deleteIfExists(filePath);
        } catch (IOException ex) {
            throw new RuntimeException("Failed to delete file: " + filename, ex);
        }
    }
}
```

### Streaming Large Files

Stream large files to avoid memory issues.

```java
@GetMapping("/download-stream/{filename}")
public ResponseEntity<StreamingResponseBody> downloadLargeFile(
        @PathVariable String filename) {
    
    Resource resource = fileStorageService.loadAsResource(filename);
    
    StreamingResponseBody stream = outputStream -> {
        try (InputStream inputStream = resource.getInputStream()) {
            byte[] buffer = new byte[8192];
            int bytesRead;
            
            while ((bytesRead = inputStream.read(buffer)) != -1) {
                outputStream.write(buffer, 0, bytesRead);
            }
            
            outputStream.flush();
        }
    };
    
    return ResponseEntity.ok()
        .contentType(MediaType.APPLICATION_OCTET_STREAM)
        .header(HttpHeaders.CONTENT_DISPOSITION, 
            "attachment; filename=\"" + filename + "\"")
        .body(stream);
}
```

**Generate and stream on-the-fly:**

```java
    };

        .contentType(MediaType.APPLICATION_PDF)
        .header(HttpHeaders.CONTENT_DISPOSITION, 
            "attachment; filename=\"report-" + id + ".pdf\"")
        .body(stream);
}
```

## 4. CORS Configuration

Cross-Origin Resource Sharing (CORS) allows or restricts cross-origin requests.

### @CrossOrigin Annotation

#### Controller-Level

```java
@RestController
@RequestMapping("/api/users")
@CrossOrigin(origins = "http://localhost:3000")  // Allow React app
public class UserController {
    
    @GetMapping
    public List<User> getUsers() {
        return userService.findAll();
    }
}
```

#### Method-Level

```java
@RestController
@RequestMapping("/api/users")
public class UserController {
    
    @CrossOrigin(origins = "http://localhost:3000")
    @GetMapping
    public List<User> getUsers() {
        return userService.findAll();
    }
    
    @CrossOrigin(origins = "*")  // Allow all origins
    @GetMapping("/public")
    public List<User> getPublicUsers() {
        return userService.findPublicUsers();
    }
}
```

#### Advanced Configuration

```java
@CrossOrigin(
    origins = {"http://localhost:3000", "https://example.com"},

    allowedHeaders = {"Authorization", "Content-Type"},
    exposedHeaders = {"X-Total-Count"},
    allowCredentials = "true",
    maxAge = 3600  // Cache preflight for 1 hour
)
@RestController
@RequestMapping("/api/users")
public class UserController {
    // ...
```

### Pre-flight Requests

CORS pre-flight is an OPTIONS request sent before actual request.

**Pre-flight request:**

```http
OPTIONS /api/users HTTP/1.1
Origin: http://localhost:3000
Access-Control-Request-Method: POST
Access-Control-Request-Headers: Authorization, Content-Type
```

**Pre-flight response:**

```http
HTTP/1.1 200 OK
Access-Control-Allow-Origin: http://localhost:3000
Access-Control-Allow-Methods: GET, POST, PUT, DELETE
Access-Control-Allow-Headers: Authorization, Content-Type
Access-Control-Max-Age: 3600
```

**When pre-flight is triggered:**

- Custom headers (Authorization, X-Custom-Header)
- Methods other than GET, POST, HEAD
- Content-Type other than application/x-www-form-urlencoded, multipart/form-data, text/plain

**Handle pre-flight:**

```java
@Configuration
public class CorsConfig implements WebMvcConfigurer {
    
    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/api/**")
            .allowedOrigins("http://localhost:3000")
            .allowedMethods("GET", "POST", "PUT", "DELETE", "OPTIONS")  // Include OPTIONS
            .allowedHeaders("*")
            .allowCredentials(true)
            .maxAge(3600);  // Cache pre-flight for 1 hour
    }
}
```

### CORS with Spring Security

```java
@Configuration
@EnableWebSecurity
public class SecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .cors(cors -> cors.configurationSource(corsConfigurationSource()))
            .csrf(csrf -> csrf.disable())
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/api/public/**").permitAll()
                .requestMatchers("/api/**").authenticated()
            )
            .httpBasic(Customizer.withDefaults());
        
        return http.build();
    }
    
    @Bean
    public CorsConfigurationSource corsConfigurationSource() {
        CorsConfiguration configuration = new CorsConfiguration();
        configuration.setAllowedOrigins(List.of("http://localhost:3000"));
        configuration.setAllowedMethods(List.of("GET", "POST", "PUT", "DELETE", "OPTIONS"));
        configuration.setAllowedHeaders(List.of("*"));
        configuration.setAllowCredentials(true);
        configuration.setMaxAge(3600L);
        
        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/api/**", configuration);
        
        return source;
    }
}
```

## Best Practices Summary

### Async Processing ✅

1. Use async for I/O-bound operations (API calls, file processing)
2. Don't use async for CPU-bound operations
3. Configure appropriate thread pool sizes
4. Handle timeouts and errors
5. Use CompletableFuture for composable operations

### Server-Sent Events ✅

1. Set appropriate timeout (or Long.MAX_VALUE)
2. Clean up emitters on completion/timeout/error
3. Handle disconnections gracefully
4. Use for one-way server-to-client communication
5. Consider WebSockets for bi-directional communication

### File Upload/Download ✅

1. Validate file size and type
2. Use unique filenames to prevent overwrites
3. Sanitize filenames (prevent path traversal)
4. Stream large files (don't load into memory)
5. Set appropriate max file size limits

### CORS ✅

1. Be specific with allowed origins in production
2. Don't use `origins = "*"` with credentials
3. Cache pre-flight responses (maxAge)
4. Include OPTIONS in allowed methods
5. Use environment-specific configuration
