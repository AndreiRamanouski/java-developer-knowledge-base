## Overview

Spring Security is a powerful authentication and authorization framework for Java applications. Understanding its architecture deeply is crucial for implementing secure applications and troubleshooting security issues.

**Key Concept**: Spring Security uses a filter chain to intercept HTTP requests, authenticate users, and enforce authorization rules before requests reach your controllers.

---

## 1. Security Filter Chain

### Filter Chain Architecture

```java
/**
 * Security Filter Chain Overview
 */
@Configuration
@EnableWebSecurity
public class SecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .csrf().and()                    // 1. CSRF protection
            .sessionManagement().and()       // 2. Session management
            .authorizeHttpRequests()         // 3. Authorization
                .requestMatchers("/public/**").permitAll()
                .anyRequest().authenticated()
                .and()
            .formLogin().and()               // 4. Form-based authentication
            .httpBasic().and()               // 5. HTTP Basic authentication
            .logout();                       // 6. Logout handling
        
        return http.build();
    }
}
```

### Complete Filter Chain Order

Spring Security's default filter chain consists of ~15 filters. Here's the complete order:

```java
/**
 * Spring Security Filter Chain (Default Order)
 */

// 1. ChannelProcessingFilter
// Redirects to HTTPS if required

// 2. WebAsyncManagerIntegrationFilter
// Integrates SecurityContext with async requests

// 3. SecurityContextPersistenceFilter (deprecated) / SecurityContextHolderFilter
// Loads SecurityContext from session

// 4. HeaderWriterFilter
// Adds security headers (X-Frame-Options, X-XSS-Protection, etc.)

// 5. CorsFilter
// Handles CORS (Cross-Origin Resource Sharing)

// 6. CsrfFilter
// Validates CSRF tokens

// 7. LogoutFilter
// Handles logout requests (/logout)

// 8. UsernamePasswordAuthenticationFilter
// Processes form login (/login with POST)

// 9. DefaultLoginPageGeneratingFilter
// Generates default login page

// 10. DefaultLogoutPageGeneratingFilter
// Generates default logout page

// 11. BasicAuthenticationFilter
// Processes HTTP Basic authentication (Authorization: Basic ...)

// 12. RequestCacheAwareFilter
// Retrieves saved request after authentication

// 13. SecurityContextHolderAwareRequestFilter
// Wraps request with security-aware methods (getUserPrincipal, isUserInRole)

// 14. AnonymousAuthenticationFilter
// Creates anonymous authentication if no authentication exists

// 15. SessionManagementFilter
// Handles session fixation, concurrent sessions

// 16. ExceptionTranslationFilter
// Catches security exceptions (AuthenticationException, AccessDeniedException)

// 17. FilterSecurityInterceptor / AuthorizationFilter
// Makes final authorization decision
```

#### Key Filters Explained

**1. SecurityContextPersistenceFilter / SecurityContextHolderFilter**

```java
/**
 * SecurityContext Management
 */

// Loads SecurityContext from HTTP session
// Before: SecurityContextHolder is empty
// After processing: SecurityContextHolder contains authentication

@Component
public class CustomSecurityContextFilter extends OncePerRequestFilter {
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                     HttpServletResponse response, 
                                     FilterChain filterChain) throws ServletException, IOException {
        
        // 1. Load SecurityContext from session
        SecurityContext context = SecurityContextHolder.getContext();
        
        try {
            // 2. Process request
            filterChain.doFilter(request, response);
        } finally {
            // 3. Clear SecurityContext (thread safety)
            SecurityContextHolder.clearContext();
        }
    }
}
```

```java
    @Override
    protected void doFilterInternal(HttpServletRequest request,

        try {
            filterChain.doFilter(request, response);
            
        } catch (AuthenticationException e) {
            // User not authenticated → redirect to login
            response.sendRedirect("/login");
            
        } catch (AccessDeniedException e) {
            // User authenticated but lacks permission → 403
            response.sendError(HttpServletResponse.SC_FORBIDDEN);
```

**4. FilterSecurityInterceptor / AuthorizationFilter**

```java
/**
 * Final Authorization Decision
 */

// Makes final access control decision
// Checks if user has required authorities

@Component
public class CustomAuthorizationFilter extends OncePerRequestFilter {
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                     HttpServletResponse response, 
                                     FilterChain filterChain) throws ServletException, IOException {
        
        // 1. Get current authentication
        Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
        
        // 2. Check if user has required authority
        if (authentication != null && authentication.isAuthenticated()) {
            boolean hasAuthority = authentication.getAuthorities().stream()
                .anyMatch(grantedAuthority -> grantedAuthority.getAuthority().equals("ROLE_ADMIN"));
            
            if (hasAuthority) {
                filterChain.doFilter(request, response);  // ✅ Allow
            } else {
                response.sendError(HttpServletResponse.SC_FORBIDDEN);  // ❌ Deny
            }
        } else {
            response.sendError(HttpServletResponse.SC_UNAUTHORIZED);  // ❌ Not authenticated
        }
    }
}
```

```java
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            // Add custom filter BEFORE UsernamePasswordAuthenticationFilter

            .addFilterAfter(new CustomLoggingFilter(), BasicAuthenticationFilter.class)
            
            // Add custom filter at specific position

            .authorizeHttpRequests()
                .anyRequest().authenticated();

/**
 * Custom JWT Authentication Filter
 */
@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {
    
    @Autowired
    private JwtTokenProvider tokenProvider;
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                     HttpServletResponse response, 
                                     FilterChain filterChain) throws ServletException, IOException {
        
        // 1. Extract JWT from Authorization header
        String token = extractToken(request);
        
        if (token != null && tokenProvider.validateToken(token)) {
            // 2. Parse token and create authentication
            Authentication authentication = tokenProvider.getAuthentication(token);
            
            // 3. Set authentication in SecurityContext
            SecurityContextHolder.getContext().setAuthentication(authentication);
        }
        
        // 4. Continue filter chain
        filterChain.doFilter(request, response);
    }
    
    private String extractToken(HttpServletRequest request) {
        String bearerToken = request.getHeader("Authorization");
        if (bearerToken != null && bearerToken.startsWith("Bearer ")) {
            return bearerToken.substring(7);
        }
        return null;
    }
}
```

### SecurityContextHolder and SecurityContext

```java
/**
 * SecurityContextHolder: Thread-local storage for authentication
 */

// Three strategies for storing SecurityContext:
// 1. MODE_THREADLOCAL (default) - ThreadLocal storage
// 2. MODE_INHERITABLETHREADLOCAL - Inherited by child threads
// 3. MODE_GLOBAL - Global for all threads (rare)

@Service
public class SecurityService {
    
    // ✅ Get current authentication
    public Authentication getCurrentAuthentication() {
        return SecurityContextHolder.getContext().getAuthentication();
    }
    
    // ✅ Get current user
    public UserDetails getCurrentUser() {
        Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
        if (authentication != null && authentication.getPrincipal() instanceof UserDetails) {
            return (UserDetails) authentication.getPrincipal();
        }
        return null;
    }
    
    // ✅ Get username
    public String getCurrentUsername() {
        Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
        return authentication != null ? authentication.getName() : null;
    }
    
    // ✅ Check if user has role
    public boolean hasRole(String role) {
        Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
        if (authentication == null) {
            return false;
        }
        
        return authentication.getAuthorities().stream()
            .anyMatch(authority -> authority.getAuthority().equals("ROLE_" + role));
    }
    
    // ✅ Manually set authentication (e.g., for testing)
    public void setAuthentication(Authentication authentication) {
        SecurityContextHolder.getContext().setAuthentication(authentication);
    }
    
    // ✅ Clear authentication (logout)
    public void clearAuthentication() {
        SecurityContextHolder.clearContext();
    }
}
```

### Authentication vs Authorization

```java
/**
 * Authentication vs Authorization
 */

// AUTHENTICATION: Who are you?
// - Verifies identity (username + password, JWT, OAuth token)
// - Happens first in filter chain
// - Result: Authentication object in SecurityContext

// AUTHORIZATION: What can you do?
// - Verifies permissions (roles, authorities)
// - Happens after authentication
// - Result: Access granted or AccessDeniedException

@RestController
public class ExampleController {
    
    // AUTHENTICATION CHECK: Must be logged in
    @GetMapping("/authenticated")
    public String authenticated() {
        return "You are authenticated";  // Any logged-in user
    }
    
    // AUTHORIZATION CHECK: Must have specific role
    @PreAuthorize("hasRole('ADMIN')")
    @GetMapping("/admin")
    public String adminOnly() {
        return "Admin access";  // Only ADMIN role
    }
    
    // BOTH: Must be authenticated AND have role
    @PreAuthorize("isAuthenticated() and hasRole('USER')")
    @GetMapping("/user")
    public String userAccess() {
        return "User access";
    }
}
```

## 2. Authentication

### Authentication Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                     Authentication Flow                      │
└─────────────────────────────────────────────────────────────┘

1. User submits credentials (username/password)
         ↓
2. UsernamePasswordAuthenticationFilter creates 
   UsernamePasswordAuthenticationToken (unauthenticated)
         ↓
3. AuthenticationManager.authenticate(token)
         ↓
4. AuthenticationManager delegates to AuthenticationProvider(s)
         ↓
5. AuthenticationProvider calls UserDetailsService.loadUserByUsername()
         ↓
6. UserDetailsService returns UserDetails
         ↓
7. AuthenticationProvider verifies password using PasswordEncoder
         ↓
8. If valid: Return authenticated Authentication object
   If invalid: Throw AuthenticationException
         ↓
9. Authentication stored in SecurityContextHolder
         ↓
10. Success handler redirects user
```

### AuthenticationManager and AuthenticationProvider

```java
        return authConfig.getAuthenticationManager();

    // ProviderManager is the default implementation of AuthenticationManager
    // It delegates to a list of AuthenticationProviders
}

/**
 * Custom AuthenticationProvider
 */
@Component
public class CustomAuthenticationProvider implements AuthenticationProvider {
    
    @Autowired
    private UserDetailsService userDetailsService;
    
    @Autowired
    private PasswordEncoder passwordEncoder;
    
    @Override
    public Authentication authenticate(Authentication authentication) throws AuthenticationException {
        String username = authentication.getName();
        String password = authentication.getCredentials().toString();
        
        // 1. Load user
        UserDetails user = userDetailsService.loadUserByUsername(username);
        
        // 2. Verify password
        if (!passwordEncoder.matches(password, user.getPassword())) {
            throw new BadCredentialsException("Invalid credentials");
        }
        
        // 3. Check if account is enabled, not locked, not expired
        if (!user.isEnabled()) {
            throw new DisabledException("Account disabled");
        }
        if (!user.isAccountNonLocked()) {
            throw new LockedException("Account locked");
        }
        if (!user.isAccountNonExpired()) {
            throw new AccountExpiredException("Account expired");
        }
        if (!user.isCredentialsNonExpired()) {
            throw new CredentialsExpiredException("Credentials expired");
        }
        
        // 4. Return authenticated token
        return new UsernamePasswordAuthenticationToken(
            user, password, user.getAuthorities()
        );
    }
    
    @Override
    public boolean supports(Class<?> authentication) {
        // This provider supports UsernamePasswordAuthenticationToken
        return UsernamePasswordAuthenticationToken.class.isAssignableFrom(authentication);
    }
}
```

### UserDetailsService (Custom User Loading)

```java
/**
 * Custom UserDetailsService Implementation
 */
@Service
public class CustomUserDetailsService implements UserDetailsService {
    
    @Autowired
    private UserRepository userRepository;
    
    @Override
    public UserDetails loadUserByUsername(String username) throws UsernameNotFoundException {
        // 1. Load user from database
        User user = userRepository.findByUsername(username)
            .orElseThrow(() -> new UsernameNotFoundException("User not found: " + username));
        
        // 2. Convert to UserDetails
        return org.springframework.security.core.userdetails.User.builder()
            .username(user.getUsername())
            .password(user.getPassword())  // Should be BCrypt-encoded
            .authorities(user.getRoles().stream()
                .map(role -> new SimpleGrantedAuthority("ROLE_" + role.getName()))
                .collect(Collectors.toList()))
            .accountExpired(!user.isAccountNonExpired())
            .accountLocked(!user.isAccountNonLocked())
            .credentialsExpired(!user.isCredentialsNonExpired())
            .disabled(!user.isEnabled())
            .build();
    }
}
```

### PasswordEncoder

```java
/**
 * Password Encoding Configuration
 */
@Configuration
public class PasswordEncoderConfig {
    
    // ✅ RECOMMENDED: BCrypt (default)
    @Bean
    public PasswordEncoder passwordEncoder() {
        return new BCryptPasswordEncoder(12);  // Strength: 10-12 recommended
    }
    
    // Alternative: SCrypt (more secure, slower)
    @Bean
    public PasswordEncoder scryptPasswordEncoder() {
        return new SCryptPasswordEncoder();
    }
    
    // Alternative: Argon2 (most secure, newest)
    @Bean
    public PasswordEncoder argon2PasswordEncoder() {
        return new Argon2PasswordEncoder();
    }
    
    // ⚠️ For legacy systems: DelegatingPasswordEncoder
    @Bean
    public PasswordEncoder delegatingPasswordEncoder() {
        String encodingId = "bcrypt";
        
        Map<String, PasswordEncoder> encoders = new HashMap<>();
        encoders.put(encodingId, new BCryptPasswordEncoder());
        encoders.put("scrypt", new SCryptPasswordEncoder());
        encoders.put("argon2", new Argon2PasswordEncoder());
        encoders.put("sha256", new StandardPasswordEncoder());  // Legacy
        
        return new DelegatingPasswordEncoder(encodingId, encoders);
    }
}

/**
 * Password Encoding Usage
 */
@Service
public class UserService {
    
    @Autowired
    private UserRepository userRepository;
    
    @Autowired
    private PasswordEncoder passwordEncoder;
    
    // ✅ Encode password when creating user
    public User createUser(String username, String rawPassword, String email) {
        User user = new User();
        user.setUsername(username);
        user.setPassword(passwordEncoder.encode(rawPassword));  // Encode!
        user.setEmail(email);
        
        return userRepository.save(user);
    }
    
    // ✅ Verify password (Spring Security does this automatically)
    public boolean verifyPassword(String rawPassword, String encodedPassword) {
        return passwordEncoder.matches(rawPassword, encodedPassword);
    }
    
    // ✅ Update password
    public void changePassword(Long userId, String oldPassword, String newPassword) {
        User user = userRepository.findById(userId).orElseThrow();
        
        // Verify old password
        if (!passwordEncoder.matches(oldPassword, user.getPassword())) {
            throw new BadCredentialsException("Old password incorrect");
        }
        
        // Update with new password
        user.setPassword(passwordEncoder.encode(newPassword));
        userRepository.save(user);
    }
}
```

#### Password Encoder Comparison

|Encoder|Security|Speed|Use Case|
|---|---|---|---|
|**BCrypt**|✅ Good|⚡⚡⚡ Fast|General purpose (recommended)|
|**SCrypt**|✅✅ Better|⚡⚡ Moderate|High-security applications|
|**Argon2**|✅✅✅ Best|⚡ Slower|Maximum security required|
|**PBKDF2**|✅ Good|⚡⚡⚡ Fast|Compliance requirements (FIPS)|
|**SHA-256**|❌ Weak|⚡⚡⚡⚡ Fastest|❌ Legacy only, not recommended|

### Authentication Flow (Step-by-Step)

```java
/**
 * Complete Authentication Flow Example
 */

// 1️⃣ User submits login form
// POST /login
// username=john&password=secret

// 2️⃣ UsernamePasswordAuthenticationFilter intercepts
@Component
public class LoginFilter extends UsernamePasswordAuthenticationFilter {
    
    @Override
    public Authentication attemptAuthentication(HttpServletRequest request, 
                                                 HttpServletResponse response) {
        // Extract credentials
        String username = request.getParameter("username");  // "john"
        String password = request.getParameter("password");  // "secret"
        
        // Create unauthenticated token
        UsernamePasswordAuthenticationToken authRequest = 
            new UsernamePasswordAuthenticationToken(username, password);
        
        // Delegate to AuthenticationManager
        return getAuthenticationManager().authenticate(authRequest);
    }
}

// 3️⃣ AuthenticationManager (ProviderManager) receives token
@Component
public class CustomAuthenticationManager implements AuthenticationManager {
    
    @Autowired
    private List<AuthenticationProvider> providers;
    
    @Override
    public Authentication authenticate(Authentication authentication) {
        // Try each provider
        for (AuthenticationProvider provider : providers) {
            if (provider.supports(authentication.getClass())) {
                try {
                    return provider.authenticate(authentication);
                } catch (AuthenticationException e) {
                    // Try next provider
                }
            }
        }
        throw new ProviderNotFoundException("No provider found");
    }
}

// 4️⃣ DaoAuthenticationProvider processes authentication
@Component
public class CustomDaoAuthenticationProvider extends DaoAuthenticationProvider {
    
    @Override
    public Authentication authenticate(Authentication authentication) {
        String username = authentication.getName();
        String password = (String) authentication.getCredentials();
        
        // Load user
        UserDetails user = getUserDetailsService().loadUserByUsername(username);
        
        // Check password
        if (!getPasswordEncoder().matches(password, user.getPassword())) {
            throw new BadCredentialsException("Invalid credentials");
        }
        
        // Additional checks
        additionalAuthenticationChecks(user, 
            (UsernamePasswordAuthenticationToken) authentication);
        
        // Return authenticated token
        return createSuccessAuthentication(user, authentication, user);
    }
}
```

### Multiple Authentication Providers

```java
/**
 * Multiple Authentication Providers
 */
@Configuration
@EnableWebSecurity
public class MultiProviderSecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .authenticationProvider(databaseAuthenticationProvider())
            .authenticationProvider(ldapAuthenticationProvider())
            .authenticationProvider(customAuthenticationProvider())
            .authorizeHttpRequests()
                .anyRequest().authenticated()
                .and()
            .formLogin();
        
        return http.build();
    }
    
    // Provider 1: Database authentication
    @Bean
    public AuthenticationProvider databaseAuthenticationProvider() {
        DaoAuthenticationProvider provider = new DaoAuthenticationProvider();
        provider.setUserDetailsService(userDetailsService());
        provider.setPasswordEncoder(passwordEncoder());
        return provider;
    }
    
    // Provider 2: LDAP authentication
    @Bean
    public AuthenticationProvider ldapAuthenticationProvider() {
        LdapAuthenticationProvider provider = new LdapAuthenticationProvider(
            new BindAuthenticator(contextSource()),
            new DefaultLdapAuthoritiesPopulator(contextSource(), "ou=groups")
        );
        return provider;
    }
    
    // Provider 3: Custom token authentication
    @Bean
    public AuthenticationProvider customAuthenticationProvider() {
        return new CustomTokenAuthenticationProvider();
    }
}

/**
 * Custom Authentication Provider for API Tokens
 */
@Component
public class CustomTokenAuthenticationProvider implements AuthenticationProvider {
    
    @Autowired
    private ApiTokenRepository tokenRepository;
    
    @Override
    public Authentication authenticate(Authentication authentication) throws AuthenticationException {
        String token = (String) authentication.getCredentials();
        
        // Validate API token
        ApiToken apiToken = tokenRepository.findByToken(token)
            .orElseThrow(() -> new BadCredentialsException("Invalid token"));
        
        if (apiToken.isExpired()) {
            throw new CredentialsExpiredException("Token expired");
        }
        
        // Return authenticated token
        return new UsernamePasswordAuthenticationToken(
            apiToken.getUsername(), 
            token, 
            apiToken.getAuthorities()
        );
    }
    
    @Override
    public boolean supports(Class<?> authentication) {
        // Support custom token authentication
        return ApiTokenAuthenticationToken.class.isAssignableFrom(authentication);
    }
}
```

```java
    }
    
    @Override
    public boolean supports(Class<?> clazz) {
        return true;
    }

    public int vote(Authentication authentication, Object object, Collection<ConfigAttribute> attributes) {
        if (authentication == null) {
            return ACCESS_DENIED;
        }
        
        // Custom voting logic
        String username = authentication.getName();
        
        if (username.equals("superadmin")) {

            }
        }
        
        return ACCESS_ABSTAIN;  // Let other voters decide
    }
```

```java
/**
 * JWT Token Provider
 */
@Component
public class JwtTokenProvider {
    
    @Value("${jwt.secret}")
    private String secretKey;
    
    @Value("${jwt.expiration}")
    private long expirationTime;  // milliseconds
    
    // ✅ Generate JWT token
    public String generateToken(Authentication authentication) {
        UserDetails user = (UserDetails) authentication.getPrincipal();
        
        Date now = new Date();
        Date expiryDate = new Date(now.getTime() + expirationTime);
        
        return Jwts.builder()
            .setSubject(user.getUsername())
            .setIssuedAt(now)
            .setExpiration(expiryDate)
            .claim("authorities", user.getAuthorities().stream()
                .map(GrantedAuthority::getAuthority)
                .collect(Collectors.toList()))
            .signWith(SignatureAlgorithm.HS512, secretKey)
            .compact();
    }
    
    // ✅ Parse JWT and get username
    public String getUsernameFromToken(String token) {
        Claims claims = Jwts.parser()
            .setSigningKey(secretKey)
            .parseClaimsJws(token)
            .getBody();
        
        return claims.getSubject();
    }
    
    // ✅ Get authorities from token
    public Collection<? extends GrantedAuthority> getAuthorities(String token) {
        Claims claims = Jwts.parser()
            .setSigningKey(secretKey)
            .parseClaimsJws(token)
            .getBody();
        
        List<String> authorities = claims.get("authorities", List.class);
        return authorities.stream()
            .map(SimpleGrantedAuthority::new)
            .collect(Collectors.toList());
    }
    
    // ✅ Validate JWT token
    public boolean validateToken(String token) {
        try {
            Jwts.parser()
                .setSigningKey(secretKey)
                .parseClaimsJws(token);
            return true;
        } catch (SignatureException e) {
            log.error("Invalid JWT signature");
        } catch (MalformedJwtException e) {
            log.error("Invalid JWT token");
        } catch (ExpiredJwtException e) {
            log.error("Expired JWT token");
        } catch (UnsupportedJwtException e) {
            log.error("Unsupported JWT token");
        } catch (IllegalArgumentException e) {
            log.error("JWT claims string is empty");
        }
        return false;
    }
    
    // ✅ Create Authentication from token
    public Authentication getAuthentication(String token) {
        String username = getUsernameFromToken(token);
        Collection<? extends GrantedAuthority> authorities = getAuthorities(token);
        
        return new UsernamePasswordAuthenticationToken(username, null, authorities);
    }
}

/**
 * JWT Authentication Filter
 */
@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {
    
    @Autowired
    private JwtTokenProvider tokenProvider;
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                     HttpServletResponse response, 
                                     FilterChain filterChain) throws ServletException, IOException {
        try {
            // 1. Extract JWT from Authorization header
            String jwt = getJwtFromRequest(request);
            
            // 2. Validate and parse token
            if (jwt != null && tokenProvider.validateToken(jwt)) {
                // 3. Create Authentication
                Authentication authentication = tokenProvider.getAuthentication(jwt);
                
                // 4. Set in SecurityContext
                SecurityContextHolder.getContext().setAuthentication(authentication);
            }
        } catch (Exception e) {
            log.error("Could not set user authentication", e);
        }
        
        filterChain.doFilter(request, response);
    }
    
    private String getJwtFromRequest(HttpServletRequest request) {
        String bearerToken = request.getHeader("Authorization");
        
        if (bearerToken != null && bearerToken.startsWith("Bearer ")) {
            return bearerToken.substring(7);  // Remove "Bearer " prefix
        }
        
        return null;
    }
}

 * Authentication Controller (Login)

@RestController
@RequestMapping("/api/auth")
public class AuthController {
    
    @Autowired
    private AuthenticationManager authenticationManager;
    
    @Autowired
    private JwtTokenProvider tokenProvider;
    
    @PostMapping("/login")
    public ResponseEntity<JwtAuthenticationResponse> login(@RequestBody LoginRequest loginRequest) {
        // 1. Authenticate user
        Authentication authentication = authenticationManager.authenticate(
            new UsernamePasswordAuthenticationToken(
                loginRequest.getUsername(),
                loginRequest.getPassword()
            )
        );
        
        // 2. Generate JWT token
        String token = tokenProvider.generateToken(authentication);
        
        // 3. Return token to client
        return ResponseEntity.ok(new JwtAuthenticationResponse(token));
    }
    
    @PostMapping("/refresh")
    public ResponseEntity<JwtAuthenticationResponse> refresh(@RequestHeader("Authorization") String token) {
        // Extract token
        String jwt = token.substring(7);  // Remove "Bearer "
        
        // Validate and refresh
        if (tokenProvider.validateToken(jwt)) {
            Authentication authentication = tokenProvider.getAuthentication(jwt);
            String newToken = tokenProvider.generateToken(authentication);
            
            return ResponseEntity.ok(new JwtAuthenticationResponse(newToken));
        }
        
        return ResponseEntity.status(HttpStatus.UNAUTHORIZED).build();
    }
}
```

**Stateless JWT Flow**:

```
1. Client sends credentials → POST /api/auth/login
2. Server validates → generates JWT
3. Server returns JWT → {"accessToken": "eyJhbGci..."}
4. Client stores JWT (localStorage, cookie)
5. Client sends JWT in every request → Authorization: Bearer eyJhbGci...
6. Server validates JWT → extracts user info → processes request
7. NO server-side session storage ✅
```

**JWT vs Session Comparison**:

|Aspect|Session-Based|JWT (Stateless)|
|---|---|---|
|**Storage**|Server (session store)|Client (token)|
|**Scalability**|Requires sticky sessions|Fully scalable|
|**Revocation**|Easy (delete session)|Difficult (token valid until expiry)|
|**Security**|CSRF vulnerable|CSRF immune (no cookies)|
|**Size**|Small (session ID)|Large (contains data)|
|**Use Case**|Traditional web apps|APIs, microservices, SPAs|

## Summary & Best Practices

### Security Filter Chain

✅ **15+ filters** process every request in specific order  
✅ **SecurityContextHolder** stores authentication (ThreadLocal)  
✅ **Custom filters** can be added before/after/at specific positions  
✅ **ExceptionTranslationFilter** handles security exceptions  
✅ **FilterSecurityInterceptor** makes final authorization decision

### Authentication

✅ **AuthenticationManager** delegates to **AuthenticationProvider(s)**  
✅ **UserDetailsService** loads user from database  
✅ **PasswordEncoder** (BCrypt recommended) encodes passwords  
✅ **Multiple providers** supported for different auth mechanisms  
✅ **Authentication** stored in **SecurityContext** after successful login

### Authorization

✅ **@PreAuthorize** most flexible (SpEL support)  
✅ **@PostAuthorize** for result-based checks  
✅ **Custom security expressions** via @Service beans  
✅ **PermissionEvaluator** for domain object permissions  
✅ **Method security** must be enabled explicitly

### Session Management

✅ **Session fixation** protection enabled by default (migrateSession)  
✅ **Concurrent sessions** controlled via maximumSessions()  
✅ **Session timeout** configurable per user role  
✅ **Stateless JWT** for APIs (SessionCreationPolicy.STATELESS)  
✅ **CSRF protection** required for session-based auth

### Common Best Practices

1. **Always use HTTPS** in production
2. **BCrypt passwords** with strength 12
3. **Session fixation protection** enabled
4. **CSRF protection** for state-changing operations
5. **Secure session cookies** (HttpOnly, Secure, SameSite)
6. **JWT** for stateless APIs, sessions for traditional web apps
7. **Rate limiting** on authentication endpoints
8. **Account lockout** after failed login attempts
9. **Monitor security events** (failed logins, access denials)
10. **Regular security audits** and dependency updates
