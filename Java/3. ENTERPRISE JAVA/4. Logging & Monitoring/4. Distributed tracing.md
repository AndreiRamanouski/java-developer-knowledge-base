# Distributed Tracing

## Overview

Distributed tracing reveals the invisible journey of requests through microservices—essential for understanding latency, debugging failures, and optimizing performance in distributed systems. This guide covers seven critical tracing concepts: OpenTelemetry (unified standard for observability—vendor-neutral instrumentation, automatic and manual instrumentation, supports traces/metrics/logs, replaces OpenTracing and OpenCensus), Trace Context Propagation (passing trace information across services—W3C Trace Context standard, propagates trace-id and span-id via HTTP headers, maintains parent-child relationships, enables end-to-end visibility), Span Creation and Attributes (building blocks of traces—spans represent operations, contain timing/status/metadata, parent-child relationships form trace tree, attributes provide context like http.method or db.statement), Jaeger and Zipkin (tracing backends—collect/store/visualize traces, service dependency graphs, latency analysis, root cause analysis for failures), Correlation IDs (request tracking across services—unique ID per request, propagated in headers and logs, enables log correlation, crucial for troubleshooting), Performance Impact (overhead of tracing—instrumentation adds 1-5% latency, async reporters minimize impact, sampling reduces load, measure before deploying), and Sampling Strategies (controlling trace volume—always-on for errors, probability-based for normal traffic, rate-limiting for high volume, adaptive sampling based on latency). The key insight: distributed tracing makes the invisible visible—a single user request might touch 20+ services, and tracing shows exactly where time is spent and where failures occur. Common mistakes: not propagating context (broken traces), too many spans (overhead), no sampling (overwhelming backend), missing attributes (lack context), forgetting error recording, not correlating with logs. The relationship between concepts: OpenTelemetry provides instrumentation → creates spans with attributes → propagates context across services → Jaeger/Zipkin collect and visualize → correlation IDs enable log correlation → sampling controls volume → performance impact guides configuration. Real-world tracing strategy: use OpenTelemetry auto-instrumentation, propagate W3C Trace Context, sample 1-10% of traces, always trace errors, enrich spans with business context, correlate traces with logs via MDC, visualize in Jaeger.

**The Problem Distributed Tracing Solves**:

```
User request: "Why is my order taking 5 seconds?"

Without tracing:
Frontend → API Gateway → Order Service → Payment Service → Inventory Service
  ???         ???            ???              ???                ???

You know the total time (5s) but not:
- Which service is slow?
- Which database query is the bottleneck?
- Are there unnecessary sequential calls?
- Where did errors occur?

With distributed tracing:
Frontend (50ms) → API Gateway (20ms) → Order Service (500ms) → 
  Payment Service (3000ms) ← SLOW! → Inventory Service (100ms)

Now you see: Payment Service is the bottleneck (3 seconds!)
You can drill down to see which payment API call is slow
```

## 1. OpenTelemetry

### The Modern Standard for Observability

```java
/**
 * OPENTELEMETRY: Unified observability framework
 * 
 * Provides:
 * - Traces: Request flow across services
 * - Metrics: Measurements over time
 * - Logs: Event records
 * 
 * Replaces: OpenTracing + OpenCensus

import io.opentelemetry.api.OpenTelemetry;
import io.opentelemetry.api.trace.*;
import io.opentelemetry.sdk.OpenTelemetrySdk;
import io.opentelemetry.sdk.trace.SdkTracerProvider;
import io.opentelemetry.sdk.trace.export.BatchSpanProcessor;
import io.opentelemetry.exporter.jaeger.JaegerGrpcSpanExporter;

// ========================================
// OPENTELEMETRY SETUP
// ========================================

public class OpenTelemetryConfiguration {
    
    public static OpenTelemetry initializeOpenTelemetry(String serviceName) {
        
        // Create Jaeger exporter
        JaegerGrpcSpanExporter jaegerExporter = JaegerGrpcSpanExporter.builder()
                .setEndpoint("http://localhost:14250")
                .build();
        
        // Create tracer provider
        SdkTracerProvider tracerProvider = SdkTracerProvider.builder()
                .addSpanProcessor(BatchSpanProcessor.builder(jaegerExporter).build())
                .setResource(Resource.create(Attributes.of(
                    ResourceAttributes.SERVICE_NAME, serviceName,
                    ResourceAttributes.SERVICE_VERSION, "1.0.0",
                    ResourceAttributes.DEPLOYMENT_ENVIRONMENT, "production"
                )))
                .build();
        
        // Build OpenTelemetry SDK
        OpenTelemetrySdk sdk = OpenTelemetrySdk.builder()
                .setTracerProvider(tracerProvider)
                .setPropagators(ContextPropagators.create(
                    W3CTraceContextPropagator.getInstance()
                ))
                .buildAndRegisterGlobal();
        
        // Shutdown hook
        Runtime.getRuntime().addShutdownHook(new Thread(tracerProvider::close));
        
        return sdk;
    }
}

// ========================================
// BASIC TRACING
// ========================================

public class OrderService {
    
    private final Tracer tracer;
    
    public OrderService(OpenTelemetry openTelemetry) {
        this.tracer = openTelemetry.getTracer("order-service", "1.0.0");
    }
    
    public Order createOrder(OrderRequest request) {
        // Create a span
        Span span = tracer.spanBuilder("createOrder")
                .setSpanKind(SpanKind.SERVER)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            // Add attributes
            span.setAttribute("order.id", request.getOrderId());
            span.setAttribute("user.id", request.getUserId());
            span.setAttribute("order.total", request.getTotal());
            
            // Business logic
            Order order = processOrder(request);
            
            // Mark as successful
            span.setStatus(StatusCode.OK);
            
            return order;
            
        } catch (Exception e) {
            // Record exception
            span.recordException(e);
            span.setStatus(StatusCode.ERROR, "Order creation failed");
            throw e;
            
        } finally {
            // Always end the span
            span.end();
        }
    }
    
    private Order processOrder(OrderRequest request) {
        // Nested span
        Span span = tracer.spanBuilder("processOrder")
                .setSpanKind(SpanKind.INTERNAL)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            // Validate
            validateOrder(request);
            
            // Check inventory
            checkInventory(request);
            
            // Process payment
            processPayment(request);
            
            return new Order(request);
            
        } finally {
            span.end();
        }
    }
    
    private void validateOrder(OrderRequest request) {
        Span span = tracer.spanBuilder("validateOrder").startSpan();
        try (Scope scope = span.makeCurrent()) {
            // Validation logic
            if (request.getItems().isEmpty()) {
                span.addEvent("Validation failed: empty order");
                throw new ValidationException("Empty order");
            }
            span.addEvent("Validation successful");
        } finally {
            span.end();
        }
    }
}

// MANUAL INSTRUMENTATION
// ========================================

@Service
public class PaymentService {
    
    private final Tracer tracer;
    
    @Autowired
    public PaymentService(OpenTelemetry openTelemetry) {
        this.tracer = openTelemetry.getTracer("payment-service");
    }
    
    public PaymentResult processPayment(PaymentRequest request) {
        Span span = tracer.spanBuilder("processPayment")
                .setSpanKind(SpanKind.CLIENT)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            // Add business context
            span.setAttribute("payment.method", request.getMethod());
            span.setAttribute("payment.amount", request.getAmount());
            span.setAttribute("payment.currency", request.getCurrency());
            
            // Call external payment gateway
            PaymentResult result = callPaymentGateway(request);
            
            // Record result
            span.setAttribute("payment.status", result.getStatus());
            span.setAttribute("payment.transaction_id", result.getTransactionId());
            
            if (result.isSuccessful()) {
                span.setStatus(StatusCode.OK);
            } else {
                span.setStatus(StatusCode.ERROR, "Payment declined");
                span.setAttribute("payment.decline_reason", result.getReason());
            }
            
            return result;
            
        } catch (PaymentException e) {
            span.recordException(e);
            span.setStatus(StatusCode.ERROR, "Payment processing failed");
            throw e;
        } finally {
            span.end();
        }
    }
    
    private PaymentResult callPaymentGateway(PaymentRequest request) {
        Span span = tracer.spanBuilder("http.payment.gateway")
                .setSpanKind(SpanKind.CLIENT)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            // HTTP attributes
            span.setAttribute(SemanticAttributes.HTTP_METHOD, "POST");
            span.setAttribute(SemanticAttributes.HTTP_URL, "https://payment.gateway.com/api/charge");
            span.setAttribute(SemanticAttributes.HTTP_REQUEST_CONTENT_LENGTH, request.getSize());
            
            long startTime = System.currentTimeMillis();
            
            // Make HTTP call
            HttpResponse response = httpClient.post(request);
            
            long duration = System.currentTimeMillis() - startTime;
            
            // Response attributes
            span.setAttribute(SemanticAttributes.HTTP_STATUS_CODE, response.getStatusCode());
            span.setAttribute(SemanticAttributes.HTTP_RESPONSE_CONTENT_LENGTH, response.getSize());
            span.setAttribute("http.duration", duration);
            
            if (response.getStatusCode() >= 400) {
                span.setStatus(StatusCode.ERROR);
            }
            
            return parseResponse(response);
            
        } finally {
            span.end();
        }
    }
}

// ========================================
// SPAN EVENTS
// ========================================

public void processWithEvents(Order order) {
    Span span = tracer.spanBuilder("processOrder").startSpan();
    
    try (Scope scope = span.makeCurrent()) {
        span.addEvent("Order validation started");
        validateOrder(order);
        span.addEvent("Order validation completed");
        
        span.addEvent("Inventory check started");
        checkInventory(order);
        span.addEvent("Inventory check completed");
        
        span.addEvent("Payment processing started", Attributes.of(
            AttributeKey.stringKey("payment.method"), order.getPaymentMethod(),
            AttributeKey.doubleKey("payment.amount"), order.getTotal()
        ));
        processPayment(order);
        span.addEvent("Payment processing completed");
        
    } finally {
        span.end();
    }
}
```

## 2. Trace Context Propagation

### Passing Traces Across Service Boundaries

```java
/**
 * TRACE CONTEXT PROPAGATION
 * 
 * W3C Trace Context standard:
 * - traceparent: 00-{trace-id}-{span-id}-{flags}
 * - tracestate: vendor-specific data
 * 
 * Example:
 * traceparent: 00-4bf92f3577b34da6a3ce929d0e0e4736-00f067aa0ba902b7-01
 *               |  |                                |                | |
 *               |  trace-id (32 hex chars)         span-id (16)    | sampled
 *               version                                             flags
 */

// ========================================
// HTTP CLIENT PROPAGATION (Automatic)
// ========================================

@Service
public class OrderService {
    
    private final RestTemplate restTemplate;
    
    // RestTemplate automatically propagates trace context
    public InventoryStatus checkInventory(String productId) {
        Span span = tracer.spanBuilder("checkInventory")
                .setSpanKind(SpanKind.CLIENT)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            // This HTTP call automatically includes trace headers:
            // traceparent: 00-{trace-id}-{new-span-id}-01
            ResponseEntity<InventoryStatus> response = restTemplate.getForEntity(
                "http://inventory-service/api/inventory/" + productId,
                InventoryStatus.class
            );
            
            return response.getBody();
        } finally {
            span.end();
        }
    }
}

// HTTP SERVER EXTRACTION (Automatic)
// ========================================

@RestController
@RequestMapping("/api/inventory")
public class InventoryController {
    
    // Incoming request headers are automatically extracted
    // Parent span context is restored
    @GetMapping("/{productId}")

        Span span = tracer.spanBuilder("getInventory")

                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            return inventoryService.check(productId);
        } finally {
            span.end();
        }
    }
}

import io.opentelemetry.context.Context;
import io.opentelemetry.context.propagation.TextMapGetter;
import io.opentelemetry.context.propagation.TextMapSetter;

public class MessageQueuePropagation {
    
    private final TextMapPropagator propagator;
    
    public MessageQueuePropagation(OpenTelemetry openTelemetry) {
        this.propagator = openTelemetry.getPropagators().getTextMapPropagator();
    }
    
    // ========================================
    // INJECT (Sender side)
    // ========================================
    
    public void sendMessage(Message message) {
        Span span = tracer.spanBuilder("sendMessage")
                .setSpanKind(SpanKind.PRODUCER)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            // Inject trace context into message headers
            propagator.inject(Context.current(), message.getHeaders(), setter);
            
            // Send message
            kafkaProducer.send(message);
            
            span.setAttribute("messaging.destination", message.getTopic());
            span.setAttribute("messaging.message_id", message.getId());
            
        } finally {
            span.end();
        }
    }
    
    // Setter for injecting into headers
    private static final TextMapSetter<Map<String, String>> setter = 
        (carrier, key, value) -> carrier.put(key, value);
    
    // ========================================
    // EXTRACT (Receiver side)
    // ========================================
    
    public void receiveMessage(Message message) {
        // Extract trace context from message headers
        Context extractedContext = propagator.extract(
            Context.current(), 
            message.getHeaders(), 
            getter
        );
        
        Span span = tracer.spanBuilder("receiveMessage")
                .setParent(extractedContext)
                .setSpanKind(SpanKind.CONSUMER)
                .startSpan();
        
        try (Scope scope = span.makeCurrent()) {
            processMessage(message);
            
            span.setAttribute("messaging.destination", message.getTopic());
            span.setAttribute("messaging.message_id", message.getId());
            
        } finally {
            span.end();
        }
    }
    
    // Getter for extracting from headers
    private static final TextMapGetter<Map<String, String>> getter = 
        new TextMapGetter<>() {
            @Override
            public Iterable<String> keys(Map<String, String> carrier) {
                return carrier.keySet();
            }
            
            @Override
            public String get(Map<String, String> carrier, String key) {
                return carrier.get(key);
            }
        };
}

// ASYNC PROCESSING
// ========================================

public class AsyncProcessing {
    
    private final Tracer tracer;
    private final ExecutorService executor;
    
    public CompletableFuture<Result> processAsync(Request request) {
        Span span = tracer.spanBuilder("processAsync").startSpan();
        
        // Capture current context
        Context context = Context.current().with(span);
        
        return CompletableFuture.supplyAsync(() -> {
            // Restore context in async thread
            try (Scope scope = context.makeCurrent()) {
                return doAsyncWork(request);
            } finally {
                span.end();
            }
        }, executor);
    }
}

// ========================================
// MULTI-THREADED PROPAGATION
// ========================================

public class ParallelProcessing {
    
    public List<Result> processInParallel(List<Item> items) {
        Span parentSpan = tracer.spanBuilder("processInParallel").startSpan();
        
        try (Scope scope = parentSpan.makeCurrent()) {
            // Capture parent context
            Context parentContext = Context.current();
            
            return items.parallelStream()
                .map(item -> {
                    // Create child span with parent context
                    Span childSpan = tracer.spanBuilder("processItem")
                        .setParent(parentContext)
                        .startSpan();
                    
                    try (Scope childScope = childSpan.makeCurrent()) {
                        return processItem(item);
                    } finally {
                        childSpan.end();
                    }
                })
                .collect(Collectors.toList());
                
        } finally {
            parentSpan.end();
        }
    }
}
```

## 3. Span Creation and Attributes

### Building Meaningful Traces

```java
/**
 * SPAN HIERARCHY
 * 
 * Trace: Full request journey
 *   Span: Single operation
 *     Attributes: Key-value metadata
 *     Events: Point-in-time occurrences
 *     Status: Success/Error
 */

// ========================================
// SPAN KINDS
// ========================================

public class SpanKinds {
    
    // SERVER: Incoming request
    public void serverSpan() {
        Span span = tracer.spanBuilder("handleRequest")
                .setSpanKind(SpanKind.SERVER)  // Incoming HTTP/RPC request
                .startSpan();
        
        span.setAttribute(SemanticAttributes.HTTP_METHOD, "POST");
        span.setAttribute(SemanticAttributes.HTTP_ROUTE, "/api/orders");
        span.end();
    }
    
    // CLIENT: Outgoing request
    public void clientSpan() {
        Span span = tracer.spanBuilder("callPaymentService")
                .setSpanKind(SpanKind.CLIENT)  // Outgoing HTTP/RPC call
                .startSpan();
        
        span.setAttribute(SemanticAttributes.HTTP_URL, "https://payment.service.com");
        span.end();
    }
    
    // INTERNAL: Within-service operation
    public void internalSpan() {
        Span span = tracer.spanBuilder("calculateTotal")
                .setSpanKind(SpanKind.INTERNAL)  // Internal operation
                .startSpan();
        
        span.setAttribute("order.items", 5);
        span.end();
    }
    
    // PRODUCER: Message sent
    public void producerSpan() {
        Span span = tracer.spanBuilder("sendOrderEvent")
                .setSpanKind(SpanKind.PRODUCER)  // Send to queue
                .startSpan();
        
        span.setAttribute(SemanticAttributes.MESSAGING_DESTINATION, "orders-topic");
        span.end();
    }
    
    // CONSUMER: Message received
    public void consumerSpan() {
        Span span = tracer.spanBuilder("processOrderEvent")
                .setSpanKind(SpanKind.CONSUMER)  // Receive from queue
                .startSpan();
        
        span.setAttribute(SemanticAttributes.MESSAGING_DESTINATION, "orders-topic");
        span.end();
    }
}

public class SpanStatus {
    
    public void recordSuccess() {
        Span span = tracer.spanBuilder("operation").startSpan();
        
        try {
            doWork();
            span.setStatus(StatusCode.OK);
        } finally {
            span.end();
        }
    }
    
    public void recordError() {
        Span span = tracer.spanBuilder("operation").startSpan();
        
        try {
            riskyOperation();
        } catch (Exception e) {
            // Record the exception
            span.recordException(e, Attributes.of(
                AttributeKey.stringKey("error.type"), e.getClass().getName(),
                AttributeKey.stringKey("error.message"), e.getMessage()
            ));
            
            // Set error status
            span.setStatus(StatusCode.ERROR, "Operation failed: " + e.getMessage());
            
            throw e;
        } finally {
            span.end();
        }
    }
}
```

## 5. Correlation IDs

### Tracking Requests Across Services

```java
/**
 * CORRELATION ID: Unique identifier per request
 * 
 * Propagated through:
 * - HTTP headers
 * - Message queue headers
 * - MDC (for logging)
 * 
 * Enables:
 * - Log aggregation
 * - Request tracking
 * - Debugging across services

// ========================================
// CORRELATION ID GENERATION
// ========================================

import org.slf4j.MDC;

@Component
public class CorrelationIdFilter implements Filter {
    
    private static final String CORRELATION_ID_HEADER = "X-Correlation-ID";
    private static final String CORRELATION_ID_MDC_KEY = "correlationId";
    
    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain)
            throws IOException, ServletException {
        
        HttpServletRequest httpRequest = (HttpServletRequest) request;
        HttpServletResponse httpResponse = (HttpServletResponse) response;
        
        // Get or generate correlation ID
        String correlationId = httpRequest.getHeader(CORRELATION_ID_HEADER);
        if (correlationId == null || correlationId.isEmpty()) {
            correlationId = UUID.randomUUID().toString();
        }
        
        // Add to MDC (will appear in all logs)
        MDC.put(CORRELATION_ID_MDC_KEY, correlationId);
        
        // Add to response header
        httpResponse.setHeader(CORRELATION_ID_HEADER, correlationId);
        
        try {
            chain.doFilter(request, response);
        } finally {
            // Always clean up MDC
            MDC.remove(CORRELATION_ID_MDC_KEY);
        }
    }
}

// ========================================
// PROPAGATING CORRELATION ID
// ========================================

@Configuration
public class RestTemplateConfiguration {
    
    @Bean
    public RestTemplate restTemplate() {
        RestTemplate restTemplate = new RestTemplate();
        
        // Add interceptor to propagate correlation ID
        restTemplate.getInterceptors().add(new CorrelationIdInterceptor());
        
        return restTemplate;
    }
}

public class CorrelationIdInterceptor implements ClientHttpRequestInterceptor {
    
    private static final String CORRELATION_ID_HEADER = "X-Correlation-ID";
    private static final String CORRELATION_ID_MDC_KEY = "correlationId";
    
    @Override
    public ClientHttpResponse intercept(
            HttpRequest request, 
            byte[] body, 
            ClientHttpRequestExecution execution) throws IOException {
        
        // Get correlation ID from MDC
        String correlationId = MDC.get(CORRELATION_ID_MDC_KEY);
        
        if (correlationId != null) {
            // Add to outgoing request
            request.getHeaders().set(CORRELATION_ID_HEADER, correlationId);
        }
        
        return execution.execute(request, body);
    }
}

// ========================================
// CORRELATION WITH TRACING
// ========================================

@Component
public class TracingCorrelationFilter implements Filter {
    
    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain)
            throws IOException, ServletException {
        
        // Get trace ID from current span
        Span currentSpan = Span.current();
        String traceId = currentSpan.getSpanContext().getTraceId();
        String spanId = currentSpan.getSpanContext().getSpanId();
        
        // Add to MDC for logging
        MDC.put("traceId", traceId);
        MDC.put("spanId", spanId);
        
        // Also propagate correlation ID
        String correlationId = ((HttpServletRequest) request).getHeader("X-Correlation-ID");
        if (correlationId == null) {
            correlationId = UUID.randomUUID().toString();
        }
        MDC.put("correlationId", correlationId);
        
        try {
            chain.doFilter(request, response);
        } finally {
            MDC.clear();
        }
    }
}
```

### Measuring and Minimizing Overhead

```java
/**
 * TRACING OVERHEAD
 * 
 * Typical impact:
 * - Instrumentation: 1-5% latency increase
 * - Span creation: ~5 microseconds per span
 * - Context propagation: ~2 microseconds
 * - Export to backend: async, negligible
 * 
 * For 1000 req/sec with 10 spans each:
 * - 10,000 spans/sec
 * - ~50ms total span creation time
 * - 0.05% CPU overhead (on 8-core machine)
 */

// ========================================
// MEASURING PERFORMANCE IMPACT
// ========================================

public class PerformanceMeasurement {
    
    @Test
    public void measureTracingOverhead() {
        int iterations = 100_000;
        
        // Without tracing
        long startWithout = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            doWork();
        }
        long durationWithout = System.nanoTime() - startWithout;
        
        // With tracing
        long startWith = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            Span span = tracer.spanBuilder("work").startSpan();
            try (Scope scope = span.makeCurrent()) {
                doWork();
            } finally {
                span.end();
            }
        }
        long durationWith = System.nanoTime() - startWith;
        
        long overhead = durationWith - durationWithout;
        double percentIncrease = (overhead * 100.0) / durationWithout;
        
        System.out.println("Without tracing: " + durationWithout / 1_000_000 + "ms");
        System.out.println("With tracing: " + durationWith / 1_000_000 + "ms");
        System.out.println("Overhead: " + percentIncrease + "%");
        
        // Typical result: 2-3% overhead
    }
}

// ========================================
// MINIMIZING OVERHEAD
// ========================================

public class PerformanceOptimization {
    
    // 1. USE ASYNC EXPORT
    @Bean
    public SdkTracerProvider tracerProvider(SpanExporter exporter) {
        return SdkTracerProvider.builder()
                // Batch spans asynchronously
                .addSpanProcessor(BatchSpanProcessor.builder(exporter)
                    .setScheduleDelay(Duration.ofSeconds(5))  // Batch every 5s
                    .setMaxQueueSize(2048)
                    .setMaxExportBatchSize(512)
                    .setExporterTimeout(Duration.ofSeconds(30))
                    .build())
                .build();
    }
    
    // 2. AVOID EXCESSIVE SPANS
    public void avoidExcessiveSpans() {
        // ✗ BAD: Too many spans
        Span parentSpan = tracer.spanBuilder("processItems").startSpan();
        try (Scope scope = parentSpan.makeCurrent()) {
            for (Item item : items) {
                Span itemSpan = tracer.spanBuilder("processItem").startSpan();  // 1000+ spans!
                try {
                    process(item);
                } finally {
                    itemSpan.end();
                }
            }
        } finally {
            parentSpan.end();
        }
        
        // ✓ GOOD: One span with count
        Span span = tracer.spanBuilder("processItems").startSpan();
        try (Scope scope = span.makeCurrent()) {
            int processed = 0;
            for (Item item : items) {
                process(item);
                processed++;
            }
            span.setAttribute("items.count", processed);
        } finally {
            span.end();
        }
    }
    
    // 3. REDUCE ATTRIBUTE CARDINALITY
    public void reduceCardinality() {
        // ✗ BAD: High cardinality
        span.setAttribute("user.id", userId);  // Millions of values!
        
        // ✓ GOOD: Low cardinality
        span.setAttribute("user.tier", getUserTier(userId));  // "free", "premium", "enterprise"
    }
    
    // 4. SAMPLE APPROPRIATELY
    // (See next section)
}
```

## 7. Sampling Strategies

### Controlling Trace Volume

```java
/**
 * SAMPLING: Decide which traces to record
 * 
 * Why sample?
 * - Reduce storage costs
 * - Reduce network bandwidth
 * - Reduce backend load
 * - Still get representative data
 * 
 * Sampling happens at:
 * - Head: Decision at trace start (propagated downstream)
 * - Tail: Decision after trace complete (requires buffering)
 */

// ========================================
// PROBABILITY SAMPLER (Simple)
// ========================================

import io.opentelemetry.sdk.trace.samplers.Sampler;

@Configuration
public class SamplingConfiguration {
    
    @Bean
    public SdkTracerProvider tracerProvider() {
        return SdkTracerProvider.builder()
                // Sample 10% of traces
                .setSampler(Sampler.traceIdRatioBased(0.1))
                .build();
    }
}

/**
 * Probability sampling:
 * - 0.1 = 10% of traces
 * - 0.01 = 1% of traces
 * - 1.0 = 100% (all traces)
 * 
 * Pros: Simple, predictable
 * Cons: Might miss important traces
 */

// ========================================
// PARENT-BASED SAMPLER
// ========================================

@Bean
public SdkTracerProvider parentBasedSampler() {
    return SdkTracerProvider.builder()
            // Respect parent's sampling decision
            .setSampler(Sampler.parentBased(
                Sampler.traceIdRatioBased(0.1)
            ))
            .build();
}

/**
 * Parent-based sampling:
 * - If parent sampled: sample this span
 * - If parent not sampled: don't sample
 * - If no parent: use root sampler (10%)
 * 
 * Ensures complete traces (no partial traces)
 */

// ========================================
// ALWAYS-ON FOR ERRORS
// ========================================

public class ErrorSampler implements Sampler {
    
    private final Sampler delegate;
    
    public ErrorSampler(Sampler delegate) {
        this.delegate = delegate;
    }
    
    @Override
    public SamplingResult shouldSample(
            Context parentContext,
            String traceId,
            String name,
            SpanKind spanKind,
            Attributes attributes,
            List<LinkData> parentLinks) {
        
        // Check if this is an error span
        String httpStatus = attributes.get(SemanticAttributes.HTTP_STATUS_CODE);
        if (httpStatus != null && Integer.parseInt(httpStatus) >= 500) {
            // Always sample errors
            return SamplingResult.recordAndSample();
        }
        
        // Check for exceptions
        String exceptionType = attributes.get(AttributeKey.stringKey("exception.type"));
        if (exceptionType != null) {
            return SamplingResult.recordAndSample();
        }
        
        // Otherwise, use delegate sampler
        return delegate.shouldSample(parentContext, traceId, name, spanKind, attributes, parentLinks);
    }
}

// ========================================
// RATE-LIMITING SAMPLER
// ========================================

public class RateLimitingSampler implements Sampler {
    
    private final Sampler delegate;
    private final AtomicInteger counter = new AtomicInteger(0);
    private final int maxTracesPerSecond;
    
    public RateLimitingSampler(Sampler delegate, int maxTracesPerSecond) {
        this.delegate = delegate;
        this.maxTracesPerSecond = maxTracesPerSecond;
        
        // Reset counter every second
        ScheduledExecutorService scheduler = Executors.newScheduledThreadPool(1);
        scheduler.scheduleAtFixedRate(() -> counter.set(0), 1, 1, TimeUnit.SECONDS);
    }
    
    @Override
    public SamplingResult shouldSample(...) {
        if (counter.incrementAndGet() <= maxTracesPerSecond) {
            return delegate.shouldSample(...);
        }
        return SamplingResult.drop();
    }
}

// Usage: Limit to 100 traces/second
Sampler sampler = new RateLimitingSampler(
    Sampler.traceIdRatioBased(0.1), 
    100
);

// ========================================
// ADAPTIVE SAMPLING
// ========================================

public class AdaptiveSampler implements Sampler {
    
    private final AtomicReference<Double> samplingRate = new AtomicReference<>(0.1);
    
    @Override
    public SamplingResult shouldSample(...) {
        // Check if slow trace (>1s)
        Duration duration = getDuration(attributes);
        if (duration != null && duration.toMillis() > 1000) {
            // Sample 100% of slow traces
            return SamplingResult.recordAndSample();
        }
        
        // Use current sampling rate for normal traces
        double rate = samplingRate.get();
        if (Math.random() < rate) {
            return SamplingResult.recordAndSample();
        }
        
        return SamplingResult.drop();
    }
    
    // Adjust sampling rate based on load
    public void adjustSamplingRate(double newRate) {
        samplingRate.set(Math.max(0.01, Math.min(1.0, newRate)));
    }
}

// ========================================
// RECOMMENDED SAMPLING STRATEGIES
// ========================================

/**
 * DEVELOPMENT:
 * - 100% sampling
 * - See every trace
 * 
 * STAGING:
 * - 50% sampling
 * - Good coverage for testing
 * 
 * PRODUCTION (LOW TRAFFIC <1000 req/s):
 * - 10% sampling
 * - Enough for analysis
 * - Always sample errors
 * 
 * PRODUCTION (HIGH TRAFFIC >10,000 req/s):
 * - 1% sampling (or less)
 * - Rate limiting (e.g., 100 traces/sec)
 * - Always sample errors
 * - Adaptive based on latency
 * 
 * CRITICAL PATHS:
 * - Payment processing: 100%
 * - User registration: 100%
 * - Checkout flow: 50%
 * - Read-only APIs: 1%
 */

@Configuration
public class ProductionSamplingStrategy {
    
    @Bean
    public SdkTracerProvider productionTracer() {
        // Composite sampler
        Sampler sampler = Sampler.parentBased(
            new ErrorSampler(
                new RateLimitingSampler(
                    Sampler.traceIdRatioBased(0.01),  // 1% base rate
                    100  // Max 100 traces/second
                )
            )
        );
        
        return SdkTracerProvider.builder()
                .setSampler(sampler)
                .build();
    }
}
```
