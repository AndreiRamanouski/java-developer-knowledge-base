# Static Analysis Tools

## Overview

Static analysis tools automatically detect code quality issues, bugs, and security vulnerabilities without executing code—essential for maintaining high-quality, maintainable codebases. This guide covers six powerful static analysis tools: SonarQube (comprehensive quality platform—analyzes code quality, security, maintainability, duplications, coverage, integrates with CI/CD, provides quality gates, tracks technical debt, supports 25+ languages, enterprise-ready), SpotBugs (FindBugs successor—detects bugs in bytecode, finds null pointer exceptions, resource leaks, concurrency issues, security vulnerabilities, 400+ bug patterns, Maven/Gradle plugins), PMD (source code analyzer—detects code smells, unused variables, empty catch blocks, complex code, naming violations, best practice violations, custom rules with XPath), Checkstyle (code style enforcer—enforces naming conventions, whitespace rules, Javadoc requirements, import organization, code formatting, customizable rulesets, Google/Sun style presets), Error Prone (Google's bug detector—catches common mistakes at compile time, null pointer bugs, collection issues, string bugs, pluggable checkers, integrated with javac), and Build Integration (automated quality gates—Maven/Gradle plugins, CI/CD integration, fail builds on violations, quality reports, pull request analysis, continuous monitoring). The key insight: static analysis catches bugs before they reach production—40-70% of bugs found by static analysis, cheaper to fix early (1x vs 10x vs 100x in production), enforces consistency across teams, automates code review. Common mistakes: ignoring tool warnings (they exist for a reason!), too many rules initially (overwhelming), no customization (one size doesn't fit all), not failing builds (warnings without consequences), analyzing only on CI (should run locally), treating all warnings equally (prioritize severity). The relationship between concepts: Code written → Local analysis (IDE) → Checkstyle validates style → Error Prone catches at compile → SpotBugs analyzes bytecode → PMD analyzes source → SonarQube aggregates all → Quality gate passes/fails → Code merged or rejected. Real-world quality strategy: start with default rules, gradually add more, fail builds on high severity, run locally before commit, analyze pull requests, track metrics over time, fix existing issues incrementally, prioritize security vulnerabilities.

/**
 * Examples of issues SonarQube detects:
 */

public class SonarQubeExamples {
    
    // ❌ BUG: Null pointer exception risk
    public String processName(User user) {
        return user.getName().toUpperCase();  // NPE if user or name is null
        // SonarQube: "NullPointerException might be thrown"
    }
    
    // ✅ FIXED: Null check
    public String processNameFixed(User user) {
        if (user == null || user.getName() == null) {
            return "";
        }
        return user.getName().toUpperCase();
    }
    
    // ❌ CODE SMELL: Unused variable
    public void processData() {
        int count = 0;  // Never used
        // SonarQube: "Remove this unused 'count' local variable"
        
        System.out.println("Processing...");
    }
    
    // ❌ CODE SMELL: Empty catch block
    public void readFile(String path) {
        try {
            Files.readString(Paths.get(path));
        } catch (IOException e) {
            // Empty catch - exception ignored!
            // SonarQube: "Either log or rethrow this exception"
        }
    }
    
    // ✅ FIXED: Proper exception handling
    public void readFileFixed(String path) {
        try {
            Files.readString(Paths.get(path));
        } catch (IOException e) {
            log.error("Failed to read file: {}", path, e);
            throw new UncheckedIOException(e);
        }
    }
    
    // ❌ SECURITY: SQL Injection risk
    public List<User> findUsers(String name) {
        String sql = "SELECT * FROM users WHERE name = '" + name + "'";
        // SonarQube: "Use prepared statement to prevent SQL injection"
        return jdbcTemplate.query(sql, userMapper);
    }
    
    // ✅ FIXED: Prepared statement
    public List<User> findUsersFixed(String name) {
        String sql = "SELECT * FROM users WHERE name = ?";
        return jdbcTemplate.query(sql, userMapper, name);
    }
    
    // ❌ CODE SMELL: Cognitive Complexity too high
    public void complexMethod(Order order) {
        if (order != null) {
            if (order.getItems() != null) {
                for (Item item : order.getItems()) {
                    if (item.isValid()) {
                        if (item.getPrice() > 0) {
                            if (item.getQuantity() > 0) {
                                // Deep nesting = high complexity
                                // SonarQube: "Refactor this method to reduce complexity"
                            }
                        }
                    }
                }
            }
        }
    }
    
    // ✅ FIXED: Simplified with early returns
    public void simpleMethod(Order order) {
        if (order == null || order.getItems() == null) {
            return;
        }
        
        for (Item item : order.getItems()) {
            if (item.isValid() && item.getPrice() > 0 && item.getQuantity() > 0) {
                processValidItem(item);
            }
        }
    }
    
    // ❌ CODE SMELL: Magic numbers
    public double calculateDiscount(double price) {
        if (price > 100) {
            return price * 0.1;  // Magic number
        } else if (price > 50) {
            return price * 0.05;  // Magic number
        }
        return 0;
    }
    
    // ✅ FIXED: Named constants
    private static final double LARGE_ORDER_THRESHOLD = 100.0;
    private static final double MEDIUM_ORDER_THRESHOLD = 50.0;
    private static final double LARGE_ORDER_DISCOUNT = 0.1;
    private static final double MEDIUM_ORDER_DISCOUNT = 0.05;
    
    public double calculateDiscountFixed(double price) {
        if (price > LARGE_ORDER_THRESHOLD) {
            return price * LARGE_ORDER_DISCOUNT;
        } else if (price > MEDIUM_ORDER_THRESHOLD) {
            return price * MEDIUM_ORDER_DISCOUNT;
        }
        return 0;
    }
}

// ========================================
// COMMON SPOTBUGS ISSUES
// ========================================

/**
 * Examples of bugs SpotBugs detects:
 */

public class SpotBugsExamples {
    
    // ❌ BUG: Null pointer dereference
    public int getLength(String str) {
        return str.length();  // NPE if str is null
        // SpotBugs: NP_NULL_PARAM_DEREF
    }
    
    // ❌ BUG: Resource not closed
    public String readFile(String path) throws IOException {
        FileInputStream fis = new FileInputStream(path);
        return new String(fis.readAllBytes());  // Resource leak!
        // SpotBugs: OBL_UNSATISFIED_OBLIGATION
    }
    
    // ✅ FIXED: Use try-with-resources
    public String readFileFixed(String path) throws IOException {
        try (FileInputStream fis = new FileInputStream(path)) {
            return new String(fis.readAllBytes());
        }
    }
    
    // ❌ BUG: Mutable static field
    public static final List<String> ITEMS = new ArrayList<>();
    // SpotBugs: MS_MUTABLE_COLLECTION
    // Problem: Anyone can modify ITEMS.add("x")
    
    // ✅ FIXED: Immutable collection
    public static final List<String> ITEMS_FIXED = 
        Collections.unmodifiableList(Arrays.asList("item1", "item2"));
    
    // ❌ BUG: Equals method compares incompatible types
    @Override
    public boolean equals(Object obj) {
        if (obj instanceof String) {  // Wrong type comparison!
            return this.name.equals(obj);
        }
        return false;
        // SpotBugs: EQ_UNUSUAL
    }
    
    // ❌ BUG: Inconsistent synchronization
    private int counter = 0;
    
    public synchronized void increment() {
        counter++;
    }
    
    public int getCounter() {  // Not synchronized!
        return counter;  // Race condition
        // SpotBugs: IS_FIELD_NOT_GUARDED
    }
    
    // ✅ FIXED: Consistent synchronization
    public synchronized int getCounterFixed() {
        return counter;
    }
    
    // ❌ BUG: Dead store
    public int calculate(int x) {
        int result = x * 2;
        result = x * 3;  // Previous value never used
        return result;
        // SpotBugs: DLS_DEAD_LOCAL_STORE
    }
    
    // ❌ BUG: Float equality comparison
    public boolean compareFloats(float a, float b) {
        return a == b;  // Dangerous with floating point
        // SpotBugs: FE_FLOATING_POINT_EQUALITY
    }
    
    // ✅ FIXED: Use epsilon comparison
    public boolean compareFloatsFixed(float a, float b) {
        return Math.abs(a - b) < 0.0001f;
    }
}

// SECURITY BUGS (FindSecBugs)
// ========================================

public class SecurityBugs {
    
    // ❌ SECURITY: SQL Injection
    public List<User> findUsers(String name) {
        String sql = "SELECT * FROM users WHERE name = '" + name + "'";
        // FindSecBugs: SQL_INJECTION_JDBC
        return jdbcTemplate.query(sql, rowMapper);
    }
    
    // ❌ SECURITY: Path Traversal
    public File getFile(String filename) {
        return new File("/uploads/" + filename);
        // FindSecBugs: PATH_TRAVERSAL_IN
        // Attack: filename = "../../etc/passwd"
    }
    
    // ❌ SECURITY: Weak cryptography
    public byte[] encrypt(byte[] data) throws Exception {
        Cipher cipher = Cipher.getInstance("DES");  // Weak algorithm
        // FindSecBugs: DES_USAGE
        cipher.init(Cipher.ENCRYPT_MODE, key);
        return cipher.doFinal(data);
    }
    
    // ✅ FIXED: Strong cryptography
    public byte[] encryptFixed(byte[] data) throws Exception {
        Cipher cipher = Cipher.getInstance("AES/GCM/NoPadding");
        cipher.init(Cipher.ENCRYPT_MODE, key, new GCMParameterSpec(128, iv));
        return cipher.doFinal(data);
    }
    
    // ❌ SECURITY: Hardcoded password
    private static final String PASSWORD = "admin123";
    // FindSecBugs: HARD_CODE_PASSWORD
    
    // ❌ SECURITY: Predictable random
    public String generateToken() {
        Random random = new Random();  // Not cryptographically secure
        // FindSecBugs: PREDICTABLE_RANDOM
        return String.valueOf(random.nextInt());
    }
    
    // ✅ FIXED: Secure random
    public String generateTokenFixed() {
        SecureRandom random = new SecureRandom();
        byte[] bytes = new byte[16];
        random.nextBytes(bytes);
        return Base64.getEncoder().encodeToString(bytes);
    }
}

 * Examples of bugs Error Prone catches at compile time:

public class ErrorProneExamples {
    
    // ❌ ERROR PRONE: Object equality with ==
    public boolean compareStrings(String a, String b) {
        return a == b;  // Should use equals()
        // Error Prone: [StringEquality] String comparison using reference equality instead of value equality
    }
    
    // ✅ FIXED
    public boolean compareStringsFixed(String a, String b) {
        return a.equals(b);
    }
    
    // ❌ ERROR PRONE: Missing @Override
    public boolean equals(User other) {  // Wrong parameter type!
        // Error Prone: [MissingOverride] Method overrides method in superclass but doesn't have @Override
        return this.name.equals(other.name);
    }
    
    // ✅ FIXED
    @Override
    public boolean equals(Object obj) {
        if (!(obj instanceof User)) {
            return false;
        }
        User other = (User) obj;
        return this.name.equals(other.name);
    }
    
    // ❌ ERROR PRONE: Collection size comparison
    public boolean hasItems(List<String> items) {
        return items.size() > 0;  // Use isEmpty()
        // Error Prone: [CollectionUndefinedEquality] Avoid calling Collection.size() > 0
    }
    
    // ✅ FIXED
    public boolean hasItemsFixed(List<String> items) {
        return !items.isEmpty();
    }
    
    // ❌ ERROR PRONE: Catching Throwable
    public void process() {
        try {
            doWork();
        } catch (Throwable t) {  // Too broad!
            // Error Prone: [CatchAndPrintStackTrace] Catching Throwable/Error masks errors
            t.printStackTrace();
        }
    }
    
    // ✅ FIXED: Catch specific exceptions
    public void processFixed() {
        try {
            doWork();
        } catch (IOException e) {
            log.error("IO error", e);
        } catch (SQLException e) {
            log.error("Database error", e);
        }
    }
    
    // ❌ ERROR PRONE: Optional.get() without isPresent()
    public String getValue(Optional<String> optional) {
        return optional.get();  // May throw NoSuchElementException
        // Error Prone: [OptionalNotPresent] Optional.get() called without checking isPresent()
    }
    
    // ✅ FIXED
    public String getValueFixed(Optional<String> optional) {
        return optional.orElse("");
    }
    
    // ❌ ERROR PRONE: Modifying collection during iteration
    public void removeNulls(List<String> items) {
        for (String item : items) {
            if (item == null) {
                items.remove(item);  // ConcurrentModificationException!
                // Error Prone: [ModifyCollectionInEnhancedForLoop]
            }
        }
    }
    
    // ✅ FIXED: Use Iterator.remove()
    public void removeNullsFixed(List<String> items) {
        items.removeIf(Objects::isNull);
    }
    
    // ❌ ERROR PRONE: Thread.sleep() in test
    @Test
    public void testAsync() throws Exception {
        executor.submit(() -> doWork());
        Thread.sleep(1000);  // Flaky test!
        // Error Prone: [ThreadSleepInTests] Thread.sleep() in tests is bad practice
        assertProcessed();
    }
    
    // ✅ FIXED: Use proper synchronization
    @Test
    public void testAsyncFixed() throws Exception {
        CompletableFuture<Void> future = CompletableFuture.runAsync(() -> doWork(), executor);
        future.get(1, TimeUnit.SECONDS);
        assertProcessed();
    }
