# SSL/TLS in Java

## Overview

SSL/TLS is the foundation of secure internet communication—essential for protecting data in transit between clients and servers. This guide covers seven critical TLS concepts: SSLContext Configuration (setting up secure connections—SSLContext is entry point for TLS, requires KeyManager for client certificates, TrustManager for server validation, SecureRandom for randomness, configure protocols and cipher suites), TrustManager and KeyManager (certificate management—TrustManager validates server certificates against trusted CAs, KeyManager provides client certificates for authentication, custom implementations for special cases, default managers for standard scenarios), Certificate Validation (verifying identity—check certificate chain to trusted root CA, validate expiration dates, verify hostname matches certificate, check for revocation, trust only valid certificates), Hostname Verification (preventing man-in-the-middle—ensure certificate CN or SAN matches server hostname, default verifier sufficient for most cases, custom verifiers for special scenarios, NEVER disable hostname verification in production), Cipher Suite Selection (choosing encryption algorithms—prefer AEAD ciphers like GCM, avoid weak ciphers like RC4/DES/3DES, forward secrecy with ECDHE/DHE, modern TLS 1.3 simplifies cipher selection), TLS Version Selection (protocol version—ALWAYS use TLS 1.2+ minimum, prefer TLS 1.3 for best security, NEVER use SSL 2.0/3.0 or TLS 1.0/1.1, configure supported protocols explicitly), and Mutual TLS/mTLS (two-way authentication—server validates client certificate, client validates server certificate, common in microservices and B2B, requires client certificates in KeyStore, server certificates in TrustStore). The key insight: TLS security depends on proper configuration—default settings are often too permissive, one misconfiguration can break security completely, trust all certificates = no security. Common mistakes: disabling certificate validation (CRITICAL vulnerability), accepting all hostnames (man-in-the-middle attack), using weak cipher suites (broken encryption), allowing old TLS versions (protocol vulnerabilities), trusting self-signed certificates in production (no chain of trust), not validating certificate expiration (expired = insecure), using default Java trust store for production (may contain unexpected CAs). The relationship between concepts: SSLContext creates TLS connection → KeyManager provides client cert → TrustManager validates server cert → Hostname verification checks domain → Protocol version negotiated → Cipher suite selected → Secure channel established. Real-world TLS strategy: use TLS 1.2+ only, strong cipher suites (AES-GCM, ChaCha20), validate certificates properly, enable hostname verification, use mutual TLS for microservices, monitor certificate expiration, rotate certificates before expiry, follow Mozilla SSL Configuration Generator recommendations.

## 1. SSLContext Configuration

### Setting Up Secure TLS Connections

```java
/**
 * SSLContext: Entry point for TLS configuration
 * 
 * Components:
 * - Protocol: TLS version (TLS 1.2, TLS 1.3)
 * - KeyManager: Client certificates
 * - TrustManager: Server certificate validation
 * - SecureRandom: Randomness source
 */

import javax.net.ssl.*;
import java.security.KeyStore;
import java.security.SecureRandom;

// ========================================
// BASIC SSLCONTEXT SETUP
// ========================================

public class SSLContextConfiguration {
    
    /**
     * ✅ SECURE: Basic SSLContext with system defaults
     */
    public SSLContext createDefaultSSLContext() throws Exception {
        // Uses system trust store and default settings
        SSLContext sslContext = SSLContext.getInstance("TLS");
        sslContext.init(null, null, null);
        return sslContext;
    }
    
    /**
     * ✅ SECURE: SSLContext with custom trust store
     */
    public SSLContext createCustomSSLContext(String trustStorePath, char[] trustStorePassword) 
            throws Exception {
        
        // Load trust store (contains trusted CA certificates)
        KeyStore trustStore = KeyStore.getInstance("PKCS12");
        try (FileInputStream fis = new FileInputStream(trustStorePath)) {
            trustStore.load(fis, trustStorePassword);
        }
        
        // Initialize TrustManagerFactory
        TrustManagerFactory tmf = TrustManagerFactory.getInstance(
            TrustManagerFactory.getDefaultAlgorithm()
        );
        tmf.init(trustStore);
        
        // Create SSLContext
        SSLContext sslContext = SSLContext.getInstance("TLS");
        sslContext.init(null, tmf.getTrustManagers(), new SecureRandom());
        
        return sslContext;
    }
    
    /**
     * ✅ SECURE: SSLContext for mutual TLS (client authentication)
     */
    public SSLContext createMutualTLSContext(
            String keyStorePath, char[] keyStorePassword,
            String trustStorePath, char[] trustStorePassword) throws Exception {
        
        // Load key store (contains client certificate and private key)
        KeyStore keyStore = KeyStore.getInstance("PKCS12");
        try (FileInputStream fis = new FileInputStream(keyStorePath)) {
            keyStore.load(fis, keyStorePassword);
        }
        
        // Initialize KeyManagerFactory
        KeyManagerFactory kmf = KeyManagerFactory.getInstance(
            KeyManagerFactory.getDefaultAlgorithm()
        );
        kmf.init(keyStore, keyStorePassword);
        
        // Load trust store
        KeyStore trustStore = KeyStore.getInstance("PKCS12");
        try (FileInputStream fis = new FileInputStream(trustStorePath)) {
            trustStore.load(fis, trustStorePassword);
        }
        
        // Initialize TrustManagerFactory
        TrustManagerFactory tmf = TrustManagerFactory.getInstance(
            TrustManagerFactory.getDefaultAlgorithm()
        );
        tmf.init(trustStore);
        
        // Create SSLContext with both KeyManager and TrustManager
        SSLContext sslContext = SSLContext.getInstance("TLS");
        sslContext.init(
            kmf.getKeyManagers(),
            tmf.getTrustManagers(),
            new SecureRandom()
        );
        
        return sslContext;
    }
}
```

```java
public class CertificatePinningTrustManager implements X509TrustManager {
    
    private final X509TrustManager defaultTrustManager;
    private final Set<String> pinnedCertificates;
    
    public CertificatePinningTrustManager(
            X509TrustManager defaultTrustManager,
            Set<String> pinnedCertificates) {
        this.defaultTrustManager = defaultTrustManager;
        this.pinnedCertificates = pinnedCertificates;
    }
    
    @Override
    public void checkServerTrusted(X509Certificate[] chain, String authType) 
            throws CertificateException {
        
        // First, perform standard validation
        defaultTrustManager.checkServerTrusted(chain, authType);
        
        // Then, check if certificate is pinned
        boolean found = false;
        for (X509Certificate cert : chain) {
            String fingerprint = getCertificateFingerprint(cert);
            if (pinnedCertificates.contains(fingerprint)) {
                found = true;
                break;
            }
        }
        
        if (!found) {
            throw new CertificateException("Certificate not pinned!");
        }
    }
    
    @Override
    public void checkClientTrusted(X509Certificate[] chain, String authType) 
            throws CertificateException {
        defaultTrustManager.checkClientTrusted(chain, authType);
    }
    
    @Override
    public X509Certificate[] getAcceptedIssuers() {
        return defaultTrustManager.getAcceptedIssuers();
    }
    
    private String getCertificateFingerprint(X509Certificate cert) throws CertificateException {
        try {
            MessageDigest digest = MessageDigest.getInstance("SHA-256");
            byte[] fingerprint = digest.digest(cert.getEncoded());
            return Base64.getEncoder().encodeToString(fingerprint);
        } catch (Exception e) {
            throw new CertificateException("Failed to compute fingerprint", e);
        }
    }
}
```

## 3. Certificate Validation

### Verifying Certificate Chains

```java
/**
 * CERTIFICATE VALIDATION: Ensure certificate is trustworthy
 * 
 * Validation checks:
 * 1. Certificate chain to trusted root CA
 * 2. Certificate not expired
 * 3. Certificate not revoked (OCSP/CRL)
 * 4. Hostname matches certificate
 * 5. Certificate purpose matches usage
 */

import java.security.cert.*;
import java.security.KeyStore;
import java.util.*;

// ========================================
// CERTIFICATE VALIDATION
// ========================================

public class CertificateValidation {
    
    /**
     * ✅ Validate certificate chain
     */
    public boolean validateCertificateChain(
            X509Certificate[] chain, 
            KeyStore trustStore) throws Exception {
        
        // Create CertPath from certificate chain
        CertificateFactory certFactory = CertificateFactory.getInstance("X.509");
        CertPath certPath = certFactory.generateCertPath(Arrays.asList(chain));
        
        // Create PKIXParameters with trust anchors
        PKIXParameters params = new PKIXParameters(trustStore);
        
        // Enable revocation checking
        params.setRevocationEnabled(true);
        
        // Create validator
        CertPathValidator validator = CertPathValidator.getInstance("PKIX");
        
        try {
            PKIXCertPathValidatorResult result = 
                (PKIXCertPathValidatorResult) validator.validate(certPath, params);
            
            System.out.println("Certificate chain is valid");
            System.out.println("Trust anchor: " + result.getTrustAnchor());
            
            return true;
            
        } catch (CertPathValidatorException e) {
            System.err.println("Certificate validation failed: " + e.getMessage());
            System.err.println("Index: " + e.getIndex());
            System.err.println("Reason: " + e.getReason());
            
            return false;
        }
    }
    
    /**
     * ✅ Check certificate expiration
     */
    public boolean isCertificateValid(X509Certificate cert) {
        try {
            cert.checkValidity();  // Throws if expired or not yet valid
            
            Date notAfter = cert.getNotAfter();
            Date now = new Date();
            
            // Warn if expiring soon (within 30 days)
            long daysUntilExpiry = (notAfter.getTime() - now.getTime()) / (1000 * 60 * 60 * 24);
            if (daysUntilExpiry < 30) {
                System.out.println("WARNING: Certificate expires in " + daysUntilExpiry + " days");
            }
            
            return true;
            
        } catch (CertificateExpiredException e) {
            System.err.println("Certificate has expired");
            return false;
            
        } catch (CertificateNotYetValidException e) {
            System.err.println("Certificate is not yet valid");
            return false;
        }
    }
    
    /**
     * ✅ Extract certificate information
     */
    public void inspectCertificate(X509Certificate cert) {
        System.out.println("=== Certificate Information ===");
        
        // Subject (who the certificate is issued to)
        System.out.println("Subject: " + cert.getSubjectDN());
        
        // Issuer (who issued the certificate)
        System.out.println("Issuer: " + cert.getIssuerDN());
        
        // Serial number
        System.out.println("Serial: " + cert.getSerialNumber());
        
        // Validity period
        System.out.println("Valid from: " + cert.getNotBefore());
        System.out.println("Valid until: " + cert.getNotAfter());
        
        // Subject Alternative Names (SANs)
        try {
            Collection<List<?>> sans = cert.getSubjectAlternativeNames();
            if (sans != null) {
                System.out.println("Subject Alternative Names:");
                for (List<?> san : sans) {
                    System.out.println("  Type: " + san.get(0) + ", Value: " + san.get(1));
                }
            }
        } catch (CertificateParsingException e) {
            System.err.println("Error parsing SANs: " + e.getMessage());
        }
        
        // Key usage
        boolean[] keyUsage = cert.getKeyUsage();
        if (keyUsage != null) {
            System.out.println("Key Usage:");
            String[] usageNames = {
                "Digital Signature", "Non Repudiation", "Key Encipherment",
                "Data Encipherment", "Key Agreement", "Key Cert Sign",
                "CRL Sign", "Encipher Only", "Decipher Only"
            };
            for (int i = 0; i < keyUsage.length; i++) {
                if (keyUsage[i]) {
                    System.out.println("  " + usageNames[i]);
                }
            }
        }
    }
}
```

## 4. Hostname Verification

### Preventing Man-in-the-Middle Attacks

```java
/**
 * HOSTNAME VERIFICATION: Ensure certificate matches server hostname
 * 
 * Without this: Attacker can use valid certificate for different domain
 * Example: Certificate for attacker.com used to impersonate bank.com
 */

import javax.net.ssl.SSLSession;

     */
    public HostnameVerifier createTrustAllHostnameVerifier() {

        // 1. Attacker gets valid certificate for evil.com
        // 2. Sets up man-in-the-middle for bank.com
```

```java
        response.put("client", clientDN);
        response.put("serial", serialNumber);
        response.put("message", "Authenticated via mTLS");
        
        return ResponseEntity.ok(response);

public class MtlsClient {

     * ✅ Configure HttpClient for mutual TLS
     */
    public CloseableHttpClient createMtlsClient(
            String clientKeyStorePath, char[] clientKeyStorePassword,
            String serverTrustStorePath, char[] serverTrustStorePassword) 
            throws Exception {
        
        // Load client key store (client certificate + private key)
        KeyStore clientKeyStore = KeyStore.getInstance("PKCS12");
        try (FileInputStream fis = new FileInputStream(clientKeyStorePath)) {
            clientKeyStore.load(fis, clientKeyStorePassword);
        }
        
        // Load server trust store (trusted server certificates)
        KeyStore serverTrustStore = KeyStore.getInstance("PKCS12");
        try (FileInputStream fis = new FileInputStream(serverTrustStorePath)) {
            serverTrustStore.load(fis, serverTrustStorePassword);
        }
        
        // Create SSLContext
        SSLContext sslContext = SSLContexts.custom()
            .loadKeyMaterial(clientKeyStore, clientKeyStorePassword)
            .loadTrustMaterial(serverTrustStore, null)
            .build();
        
        // Create SSL socket factory
        SSLConnectionSocketFactory sslSocketFactory = new SSLConnectionSocketFactory(
            sslContext,
            new String[]{"TLSv1.2", "TLSv1.3"},
            null,
            SSLConnectionSocketFactory.getDefaultHostnameVerifier()
        );
        
        // Build HttpClient
        return HttpClients.custom()
            .setSSLSocketFactory(sslSocketFactory)
            .build();
    }
    
    /**
     * Make request with client certificate
     */
    public void makeSecureRequest() throws Exception {
        try (CloseableHttpClient httpClient = createMtlsClient(
                "client-keystore.p12", "changeit".toCharArray(),
                "server-truststore.p12", "changeit".toCharArray())) {
            
            HttpGet request = new HttpGet("https://api.example.com/secure");
            
            try (CloseableHttpResponse response = httpClient.execute(request)) {
                System.out.println("Status: " + response.getCode());
                
                String responseBody = EntityUtils.toString(response.getEntity());
                System.out.println("Response: " + responseBody);
            }
        }
    }
```
