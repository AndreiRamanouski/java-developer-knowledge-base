# Foreign Function & Memory API (Java 22)

## Overview

The Foreign Function & Memory API (FFM API), finalized in Java 22 (JEP 454), provides a modern, safe, and efficient way to interact with native code and memory outside the Java heap. Before FFM, developers had to use JNI (Java Native Interface), which required writing C wrapper code, dealing with complex memory management, and accepting poor performance due to crossing the JNI boundary. FFM eliminates these problems by providing direct access to native memory through MemorySegment (spatial safety), automatic cleanup through Arena (temporal safety), and direct native function calls through Linker (no JNI overhead). The API is designed around three pillars: (1) Memory management with MemorySegment and Arena that prevents dangling pointers and use-after-free bugs, (2) Foreign function calls that compile to machine code for performance, and (3) MemoryLayout that describes native data structures in a type-safe way. This guide covers the entire API from basic memory allocation to calling C libraries, comparing performance with JNI, understanding safety guarantees, and following best practices. The FFM API represents a fundamental shift in how Java interacts with the native world, making it safer, faster, and more ergonomic.

## 1. Why FFM API? (The Problem with JNI)

### Moving Beyond JNI

```java
/**
 * THE PROBLEM WITH JNI
 * 
 * Why we needed a better solution
 */

public class WhyFFMAPI {
    
    /**
     * JNI problems
     */
    public static void jniProblems() {
        /**
         * PROBLEM 1: REQUIRES C CODE
         * 
         * To call a C function like strlen:
         * 
         * 1. Write Java native method:
         *    public native long strlen(String str);
         * 
         * 2. Generate header:
         *    javah -jni MyClass
         * 
         * 3. Write C wrapper:
         *    JNIEXPORT jlong JNICALL Java_MyClass_strlen
         *      (JNIEnv *env, jobject obj, jstring str) {
         *      const char *cstr = (*env)->GetStringUTFChars(env, str, NULL);
         *      size_t len = strlen(cstr);
         *      (*env)->ReleaseStringUTFChars(env, str, cstr);
         *      return (jlong)len;
         *    }
         * 
         * 4. Compile C code:
         *    gcc -shared -I$JAVA_HOME/include ...
         * 
         * 5. Load library:
         *    System.loadLibrary("mylib");
         * 
         * Problems:
         * ✗ Lots of boilerplate
         * ✗ Need C compiler
         * ✗ Platform-specific builds
         * ✗ Complex build process
         * ✗ Hard to maintain
         * 
         * 
         * PROBLEM 2: POOR PERFORMANCE
         * 
         * Every JNI call:
         * - Crosses JVM/native boundary
         * - Argument marshalling
         * - Reference tracking
         * - Can't be inlined
         * - Prevents JIT optimizations
         * 
         * Performance overhead: 10-100x slower than direct call
         * 
         * 
         * PROBLEM 3: MEMORY MANAGEMENT COMPLEXITY
         * 
         * Manual reference management:
         * - GetStringUTFChars / ReleaseStringUTFChars
         * - NewGlobalRef / DeleteGlobalRef
         * - Easy to leak memory
         * - Easy to cause crashes
         * 
         * No safety guarantees:
         * - Use after free
         * - Dangling pointers
         * - Buffer overruns
         * 
         * 
         * PROBLEM 4: ERROR-PRONE
         * 
         * Common mistakes:
         * - Forgetting to release resources
         * - Type mismatches
         * - Incorrect signatures
         * - Platform differences
         * 
         * Runtime crashes, not compile errors
         */
    }
    
    /**
     * FFM API solution
     */
    public static void ffmSolution() {
        /**
         * FFM API BENEFITS:
         * 
         * 1. NO C CODE NEEDED:
         *    Just describe the function signature in Java
         *    Call it directly
         * 
         * 2. BETTER PERFORMANCE:
         *    Direct calls (no JNI boundary)
         *    Can be inlined by JIT
         *    10-100x faster than JNI
         * 
         * 3. SAFE MEMORY MANAGEMENT:
         *    MemorySegment: spatial safety (bounds checking)
         *    Arena: temporal safety (automatic cleanup)
         *    No manual reference management
         * 
         * 4. TYPE SAFE:
         *    Compile-time checking
         *    MemoryLayout describes structures
         *    Linker validates signatures
         * 
         * 5. MODERN API:
         *    Java-style (try-with-resources)
         *    Designed for Java 21+ features
         *    Clean and ergonomic
         * 
         * 
         * Example: Calling strlen
         * 
         * Before (JNI): 50+ lines of C code
         * 
         * After (FFM): 5 lines of Java:
         * 
         * Linker linker = Linker.nativeLinker();
         * SymbolLookup stdlib = linker.defaultLookup();
         * MemorySegment strlen = stdlib.find("strlen").orElseThrow();
         * MethodHandle strlenHandle = linker.downcallHandle(
         *   strlen, 
         *   FunctionDescriptor.of(JAVA_LONG, ADDRESS)
         * );
         * 
         * Done! No C code, no compilation, just works.
         */
    }
}
```

## 2. MemorySegment and MemoryLayout

### Safe Access to Off-Heap Memory

```java
/**
 * MEMORYSEGMENT AND MEMORYLAYOUT
 * 
 * Core abstractions for memory access
 */

import java.lang.foreign.*;
import java.nio.ByteOrder;

import static java.lang.foreign.ValueLayout.*;

public class MemorySegmentBasics {
    
    /**
     * Basic MemorySegment usage
     */
    public static void basicMemorySegment() {
        System.out.println("=== Basic MemorySegment ===");
        
        // Allocate native memory (10 bytes)
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment segment = arena.allocate(10);
            
            // Write values
            segment.set(JAVA_BYTE, 0, (byte) 65);  // 'A'
            segment.set(JAVA_BYTE, 1, (byte) 66);  // 'B'
            segment.set(JAVA_BYTE, 2, (byte) 67);  // 'C'
            
            // Read values
            byte b0 = segment.get(JAVA_BYTE, 0);
            byte b1 = segment.get(JAVA_BYTE, 1);
            byte b2 = segment.get(JAVA_BYTE, 2);
            
            System.out.printf("%c %c %c%n", b0, b1, b2);
            
            /**
             * MEMORYSEGMENT:
             * 
             * - Represents a contiguous memory region
             * - Can be on-heap or off-heap
             * - Bounds-checked (spatial safety)
             * - Lifecycle-managed (temporal safety)
             * - Access via ValueLayout constants
             * 
             * ValueLayout constants:
             * - JAVA_BYTE, JAVA_SHORT, JAVA_INT, JAVA_LONG
             * - JAVA_FLOAT, JAVA_DOUBLE
             * - JAVA_CHAR, JAVA_BOOLEAN
             * - ADDRESS (pointers)
             */
        } // Memory freed automatically
    }
    
    /**
     * Working with different types
     */
    public static void differentTypes() {
        System.out.println("\n=== Different Types ===");
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment segment = arena.allocate(100);
            
            // Integers
            segment.set(JAVA_INT, 0, 42);
            segment.set(JAVA_INT, 4, 100);
            
            // Floats
            segment.set(JAVA_FLOAT, 8, 3.14f);
            segment.set(JAVA_DOUBLE, 12, 2.718);
            
            // Longs
            segment.set(JAVA_LONG, 20, 123456789L);
            
            // Read back
            int i1 = segment.get(JAVA_INT, 0);
            int i2 = segment.get(JAVA_INT, 4);
            float f = segment.get(JAVA_FLOAT, 8);
            double d = segment.get(JAVA_DOUBLE, 12);
            long l = segment.get(JAVA_LONG, 20);
            
            System.out.printf("int: %d, %d%n", i1, i2);
            System.out.printf("float: %f, double: %f%n", f, d);
            System.out.printf("long: %d%n", l);
            
            /**
             * OFFSETS:
             * 
             * Must calculate offsets manually:
             * - int at 0, size 4
             * - int at 4, size 4
             * - float at 8, size 4
             * - double at 12, size 8
             * - long at 20, size 8
             * 
             * Or use MemoryLayout (better way!)
             */
        }
    }
    
    /**
     * MemoryLayout - structured access
     */
    public static void memoryLayout() {
        System.out.println("\n=== MemoryLayout ===");
        
        // Define a C struct:
        // struct Point {
        //   int x;
        //   int y;
        // };
        
        StructLayout pointLayout = MemoryLayout.structLayout(
            JAVA_INT.withName("x"),
            JAVA_INT.withName("y")
        );
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment point = arena.allocate(pointLayout);
            
            // Get offset paths
            VarHandle xHandle = pointLayout.varHandle(
                MemoryLayout.PathElement.groupElement("x")
            );
            VarHandle yHandle = pointLayout.varHandle(
                MemoryLayout.PathElement.groupElement("y")
            );
            
            // Set values using handles
            xHandle.set(point, 0L, 100);
            yHandle.set(point, 0L, 200);
            
            // Get values
            int x = (int) xHandle.get(point, 0L);
            int y = (int) yHandle.get(point, 0L);
            
            System.out.printf("Point: (%d, %d)%n", x, y);
            
            /**
             * MEMORYLAYOUT BENEFITS:
             * 
             * ✓ Type-safe access
             * ✓ Named fields
             * ✓ Automatic offset calculation
             * ✓ Platform-independent
             * ✓ Alignment handling
             * ✓ Portable code
             */
        }
    }
    
    /**
     * Complex struct example
     */
    public static void complexStruct() {
        System.out.println("\n=== Complex Struct ===");
        
        // C struct:
        // struct Person {
        //   char name[20];
        //   int age;
        //   double salary;
        // };
        
        SequenceLayout nameLayout = MemoryLayout.sequenceLayout(20, JAVA_BYTE);
        
        StructLayout personLayout = MemoryLayout.structLayout(
            nameLayout.withName("name"),
            JAVA_INT.withName("age"),
            MemoryLayout.paddingLayout(4),  // Padding for alignment
            JAVA_DOUBLE.withName("salary")
        );
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment person = arena.allocate(personLayout);
            
            // Set name (as bytes)
            byte[] nameBytes = "Alice".getBytes();
            MemorySegment.copy(nameBytes, 0, person, JAVA_BYTE, 0, nameBytes.length);
            
            // Set age
            VarHandle ageHandle = personLayout.varHandle(
                MemoryLayout.PathElement.groupElement("age")
            );
            ageHandle.set(person, 0L, 30);
            
            // Set salary
            VarHandle salaryHandle = personLayout.varHandle(
                MemoryLayout.PathElement.groupElement("salary")
            );
            salaryHandle.set(person, 0L, 75000.0);
            
            // Read back
            int age = (int) ageHandle.get(person, 0L);
            double salary = (double) salaryHandle.get(person, 0L);
            
            System.out.printf("Age: %d, Salary: $%.2f%n", age, salary);
        }
    }
    
    /**
     * Array layouts
     */
    public static void arrayLayouts() {
        System.out.println("\n=== Array Layouts ===");
        
        // int[10] array
        SequenceLayout intArray = MemoryLayout.sequenceLayout(10, JAVA_INT);
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment array = arena.allocate(intArray);
            
            // Set values
            for (int i = 0; i < 10; i++) {
                array.setAtIndex(JAVA_INT, i, i * 10);
            }
            
            // Read values
            System.out.print("Array: ");
            for (int i = 0; i < 10; i++) {
                int value = array.getAtIndex(JAVA_INT, i);
                System.out.print(value + " ");
            }
            System.out.println();
            
            /**
             * ARRAY ACCESS:
             * 
             * setAtIndex(layout, index, value)
             * getAtIndex(layout, index)
             * 
             * Much cleaner than manual offset calculation!
             */
        }
    }
    
    /**
     * Byte order (endianness)
     */
    public static void byteOrder() {
        /**
         * BYTE ORDER:
         * 
         * Native layouts use native byte order:
         * - JAVA_INT (native order)
         * - JAVA_LONG (native order)
         * 
         * Explicit byte order:
         * - JAVA_INT.withOrder(ByteOrder.BIG_ENDIAN)
         * - JAVA_INT.withOrder(ByteOrder.LITTLE_ENDIAN)
         * 
         * Important for:
         * - Network protocols
         * - File formats
         * - Cross-platform compatibility
         */
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment segment = arena.allocate(8);
            
            // Native order
            segment.set(JAVA_INT, 0, 0x12345678);
            
            // Big endian
            ValueLayout.OfInt bigEndian = 
                JAVA_INT.withOrder(ByteOrder.BIG_ENDIAN);
            segment.set(bigEndian, 4, 0x12345678);
            
            // Read as bytes
            System.out.print("Native: ");
            for (int i = 0; i < 4; i++) {
                System.out.printf("%02x ", segment.get(JAVA_BYTE, i));
            }
            System.out.println();
            
            System.out.print("BigEndian: ");
            for (int i = 4; i < 8; i++) {
                System.out.printf("%02x ", segment.get(JAVA_BYTE, i));
            }
            System.out.println();
        }
    }
    
    public static void main(String[] args) {
        basicMemorySegment();
        differentTypes();
        memoryLayout();
        complexStruct();
        arrayLayouts();
    }
}
```

## 3. Arena-Based Memory Management

### Automatic Cleanup and Safety

```java
/**
 * ARENA-BASED MEMORY MANAGEMENT
 * 
 * Safe, automatic memory lifecycle
 */

import java.lang.foreign.*;
import static java.lang.foreign.ValueLayout.*;

public class ArenaMemoryManagement {
    
    /**
     * Arena basics
     */
    public static void arenaBasics() {
        System.out.println("=== Arena Basics ===");
        
        // Confined arena - single thread
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment seg1 = arena.allocate(100);
            MemorySegment seg2 = arena.allocate(200);
            MemorySegment seg3 = arena.allocate(300);
            
            // All segments share same lifecycle
            // All freed when arena closes
            
            seg1.set(JAVA_INT, 0, 42);
            System.out.println("Value: " + seg1.get(JAVA_INT, 0));
            
        } // All memory freed here automatically!
        
        /**
         * ARENA:
         * 
         * - Manages lifecycle of memory segments
         * - All segments allocated from arena share lifecycle
         * - When arena closes, all memory freed
         * - No manual free() calls
         * - No memory leaks
         * 
         * Like try-with-resources for native memory!
         */
    }
    
    /**
     * Confined arena - single thread
     */
    public static void confinedArena() {
        System.out.println("\n=== Confined Arena ===");
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment segment = arena.allocate(100);
            
            // Can only access from thread that created it
            segment.set(JAVA_INT, 0, 42);
            
            // Cannot access from another thread:
            /*
            new Thread(() -> {
                segment.get(JAVA_INT, 0);  // IllegalStateException!
            }).start();
            */
            
            /**
             * CONFINED ARENA:
             * 
             * Arena.ofConfined()
             * 
             * - Single-threaded access only
             * - Fastest (no synchronization)
             * - Most common use case
             * - Thread-safe enforcement
             */
        }
    }
    
    /**
     * Shared arena - multi-threaded
     */
    public static void sharedArena() throws InterruptedException {
        System.out.println("\n=== Shared Arena ===");
        
        try (Arena arena = Arena.ofShared()) {
            MemorySegment segment = arena.allocate(100);
            segment.set(JAVA_INT, 0, 0);
            
            // Can access from multiple threads
            Thread t1 = new Thread(() -> {
                for (int i = 0; i < 1000; i++) {
                    int val = segment.get(JAVA_INT, 0);
                    segment.set(JAVA_INT, 0, val + 1);
                }
            });
            
            Thread t2 = new Thread(() -> {
                for (int i = 0; i < 1000; i++) {
                    int val = segment.get(JAVA_INT, 0);
                    segment.set(JAVA_INT, 0, val + 1);
                }
            });
            
            t1.start();
            t2.start();
            t1.join();
            t2.join();
            
            int result = segment.get(JAVA_INT, 0);
            System.out.println("Final value: " + result);
            
            /**
             * SHARED ARENA:
             * 
             * Arena.ofShared()
             * 
             * - Multi-threaded access allowed
             * - Slower (synchronization overhead)
             * - Use when needed for concurrency
             * 
             * Note: Memory access not synchronized!
             * Use proper synchronization for concurrent writes
             */
        }
    }
    
    /**
     * Global arena - application lifetime
     */
    public static void globalArena() {
        System.out.println("\n=== Global Arena ===");
        
        // Never closed - lives until JVM shutdown
        Arena arena = Arena.global();
        
        MemorySegment segment = arena.allocate(100);
        segment.set(JAVA_INT, 0, 42);
        
        // No try-with-resources needed
        // Memory never freed (until JVM exit)
        
        /**
         * GLOBAL ARENA:
         * 
         * Arena.global()
         * 
         * - Application lifetime
         * - Never closed
         * - Use for long-lived native resources
         * - No automatic cleanup
         * 
         * Use sparingly!
         * Prefer confined or shared arenas
         */
    }
    
    /**
     * Arena scope and lifecycle
     */
    public static void arenaScope() {
        System.out.println("\n=== Arena Scope ===");
        
        MemorySegment outer;
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment segment = arena.allocate(100);
            segment.set(JAVA_INT, 0, 42);
            
            System.out.println("Inside: " + segment.get(JAVA_INT, 0));
            
            outer = segment;  // Reference escapes
            
        } // Arena closed, segment invalidated
        
        // Accessing now would throw:
        // outer.get(JAVA_INT, 0);  // IllegalStateException!
        
        /**
         * TEMPORAL SAFETY:
         * 
         * Arena tracks all segments
         * When arena closes:
         * - All segments invalidated
         * - Memory freed
         * - Access throws exception
         * 
         * Prevents:
         * ✓ Use-after-free bugs
         * ✓ Dangling pointers
         * ✓ Memory corruption
         */
    }
    
    /**
     * Allocation strategies
     */
    public static void allocationStrategies() {
        System.out.println("\n=== Allocation Strategies ===");
        
        try (Arena arena = Arena.ofConfined()) {
            // Single allocation
            MemorySegment seg1 = arena.allocate(100);
            
            // Allocation with layout
            StructLayout pointLayout = MemoryLayout.structLayout(
                JAVA_INT.withName("x"),
                JAVA_INT.withName("y")
            );
            MemorySegment point = arena.allocate(pointLayout);
            
            // Array allocation
            MemorySegment array = arena.allocateArray(JAVA_INT, 10);
            
            // With initial values
            int[] values = {1, 2, 3, 4, 5};
            MemorySegment initialized = arena.allocateArray(JAVA_INT, values);
            
            /**
             * ALLOCATION METHODS:
             * 
             * arena.allocate(size)
             * arena.allocate(layout)
             * arena.allocate(layout, count)
             * arena.allocateArray(layout, count)
             * arena.allocateArray(layout, values)
             * 
             * All allocations freed when arena closes
             */
        }
    }
    
    /**
     * Best practices
     */
    public static void bestPractices() {
        /**
         * BEST PRACTICES:
         * 
         * 1. Use confined arenas when possible
         *    - Fastest
         *    - Safest
         *    - Most common
         * 
         * 2. Keep arena scope tight
         *    try (Arena arena = ...) {
         *      // Allocate and use
         *    } // Freed
         * 
         * 3. Don't let segments escape
         *    - Return values from arena scope is risky
         *    - Segments invalid after arena closes
         * 
         * 4. Use shared arena for concurrency
         *    - When multiple threads need access
         *    - Add your own synchronization
         * 
         * 5. Avoid global arena
         *    - Memory never freed
         *    - Use only for app-lifetime resources
         * 
         * 6. One arena per logical operation
         *    - Easy to reason about
         *    - Clear lifecycle
         *    - Automatic cleanup
         */
    }
    
    public static void main(String[] args) throws InterruptedException {
        arenaBasics();
        confinedArena();
        sharedArena();
        globalArena();
        arenaScope();
        allocationStrategies();
    }
}
```

## 4. Calling Native Functions

### Direct Foreign Function Calls

```java
/**
 * CALLING NATIVE FUNCTIONS
 * 
 * Using Linker to call C functions
 */

import java.lang.foreign.*;
import java.lang.invoke.MethodHandle;

import static java.lang.foreign.ValueLayout.*;

public class CallingNativeFunctions {
    
    /**
     * Basic function call - strlen
     */
    public static void callingStrlen() throws Throwable {
        System.out.println("=== Calling strlen ===");
        
        // Get linker and lookup
        Linker linker = Linker.nativeLinker();
        SymbolLookup stdlib = linker.defaultLookup();
        
        // Find strlen function
        MemorySegment strlenAddress = stdlib.find("strlen")
            .orElseThrow(() -> new RuntimeException("strlen not found"));
        
        // Describe function: size_t strlen(const char *s)
        FunctionDescriptor strlenDescriptor = FunctionDescriptor.of(
            JAVA_LONG,   // Return type: size_t
            ADDRESS      // Parameter: const char*
        );
        
        // Create method handle
        MethodHandle strlen = linker.downcallHandle(
            strlenAddress,
            strlenDescriptor
        );
        
        // Call it!
        try (Arena arena = Arena.ofConfined()) {
            // Allocate C string
            MemorySegment cString = arena.allocateUtf8String("Hello, World!");
            
            // Call strlen
            long length = (long) strlen.invoke(cString);
            
            System.out.println("Length: " + length);
        }
        
        /**
         * STEPS:
         * 
         * 1. Get Linker
         * 2. Look up function symbol
         * 3. Describe function signature (FunctionDescriptor)
         * 4. Create MethodHandle (downcallHandle)
         * 5. Invoke!
         * 
         * No C code needed!
         * No JNI!
         * Direct call!
         */
    }
    
    /**
     * Function with multiple parameters
     */
    public static void multipleParameters() throws Throwable {
        System.out.println("\n=== Multiple Parameters ===");
        
        Linker linker = Linker.nativeLinker();
        SymbolLookup stdlib = linker.defaultLookup();
        
        // memcpy: void* memcpy(void *dest, const void *src, size_t n)
        MemorySegment memcpyAddr = stdlib.find("memcpy")
            .orElseThrow(() -> new RuntimeException("memcpy not found"));
        
        FunctionDescriptor memcpyDescriptor = FunctionDescriptor.of(
            ADDRESS,     // Return: void*
            ADDRESS,     // dest
            ADDRESS,     // src
            JAVA_LONG    // n
        );
        
        MethodHandle memcpy = linker.downcallHandle(
            memcpyAddr,
            memcpyDescriptor
        );
        
        try (Arena arena = Arena.ofConfined()) {
            // Source buffer
            byte[] data = {1, 2, 3, 4, 5};
            MemorySegment src = arena.allocateArray(JAVA_BYTE, data);
            
            // Destination buffer
            MemorySegment dest = arena.allocate(5);
            
            // Call memcpy
            memcpy.invoke(dest, src, 5L);
            
            // Verify
            System.out.print("Copied: ");
            for (int i = 0; i < 5; i++) {
                System.out.print(dest.get(JAVA_BYTE, i) + " ");
            }
            System.out.println();
        }
    }
    
    /**
     * Function returning struct
     */
    public static void returningStruct() throws Throwable {
        /**
         * RETURNING STRUCTS:
         * 
         * C function:
         * struct Point { int x; int y; };
         * struct Point getPoint();
         * 
         * FunctionDescriptor:
         * FunctionDescriptor.of(
         *   pointLayout  // Return struct by value
         * )
         * 
         * Result is MemorySegment with struct data
         */
    }
    
    /**
     * Variadic functions
     */
    public static void variadicFunctions() throws Throwable {
        System.out.println("\n=== Variadic Functions ===");
        
        Linker linker = Linker.nativeLinker();
        SymbolLookup stdlib = linker.defaultLookup();
        
        // printf: int printf(const char *format, ...)
        MemorySegment printfAddr = stdlib.find("printf")
            .orElseThrow(() -> new RuntimeException("printf not found"));
        
        // For variadic functions, use VaList
        FunctionDescriptor printfDescriptor = FunctionDescriptor.of(
            JAVA_INT,    // Return: int
            ADDRESS      // format string
            // Variadic arguments passed separately
        );
        
        MethodHandle printf = linker.downcallHandle(
            printfAddr,
            printfDescriptor,
            Linker.Option.firstVariadicArg(1)  // Varargs start at index 1
        );
        
        try (Arena arena = Arena.ofConfined()) {
            MemorySegment format = arena.allocateUtf8String("Number: %d\n");
            
            // Call with variadic argument
            printf.invoke(format, 42);
        }
    }
    
    /**
     * Callbacks (upcalls)
     */
    public static void callbacks() throws Throwable {
        System.out.println("\n=== Callbacks (Upcalls) ===");
        
        Linker linker = Linker.nativeLinker();
        
        // Java method to be called from C
        class MyComparator {
            static int compare(MemorySegment a, MemorySegment b) {
                int valA = a.get(JAVA_INT, 0);
                int valB = b.get(JAVA_INT, 0);
                return Integer.compare(valA, valB);
            }
        }
        
        // Get MethodHandle for Java method
        MethodHandle compareHandle = MethodHandles.lookup()
            .findStatic(
                MyComparator.class,
                "compare",
                MethodType.methodType(int.class, MemorySegment.class, MemorySegment.class)
            );
        
        // Describe callback signature: int (*)(const void*, const void*)
        FunctionDescriptor comparatorDesc = FunctionDescriptor.of(
            JAVA_INT,
            ADDRESS,
            ADDRESS
        );
```

```java
                this.objectSize = objectSize;

                    objectSize

                nextIndex++;
                return result;
            }
```
