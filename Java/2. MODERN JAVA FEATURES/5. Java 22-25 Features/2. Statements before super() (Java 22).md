# Statements before super() (Java 22)

## Overview

Statements before super(), finalized in Java 22 (JEP 447), removes a long-standing restriction in Java that required the super() or this() call to be the very first statement in a constructor. Before Java 22, if you needed to perform any computation, validation, or preparation before calling the superclass constructor, you had to resort to workarounds like static helper methods, instance initializers, or factory methods. This limitation was particularly painful when superclass constructors required parameters that needed preprocessing, validation, or transformation. Java 22 now allows you to write any statements before super() as long as they don't reference the instance being constructed (this). This enables natural pre-construction validation, cleaner parameter transformation, better error messages, and more flexible constructor design. The feature maintains Java's safety guarantees—you still cannot access instance members before super() completes—while giving developers the flexibility to prepare arguments and validate inputs naturally. This guide covers the old restrictions, new capabilities, common patterns, important limitations, and best practices for using this feature effectively.

## 1. The Problem Before Java 22

### Historical Restriction and Workarounds

```java
/**
 * THE PROBLEM BEFORE JAVA 22
 * 
 * super() had to be first statement
 */

import java.util.*;

public class TheProblem {
    
    /**
     * Problem 1: Cannot validate before super
     */
    static class Before_ValidationProblem {
        
        static class Person {
            protected final String name;
            
            Person(String name) {
                this.name = name;
            }
        }
        
        static class Employee extends Person {
            private final String employeeId;
            
            Employee(String name, String employeeId) {
                // Want to validate BEFORE calling super
                // But can't - super must be first!
                
                super(name);  // Must be first statement
                
                // Validation happens AFTER super
                if (employeeId == null || employeeId.isEmpty()) {
                    throw new IllegalArgumentException("Employee ID required");
                }
                this.employeeId = employeeId;
                
                /**
                 * PROBLEM:
                 * If validation fails, superclass is already constructed!
                 * Person object partially initialized
                 * Wasteful and confusing
                 */
            }
        }
    }
    
    /**
     * Problem 2: Cannot transform parameters before super
     */
    static class Before_TransformationProblem {
        
        static class Base {
            protected final String value;
            
            Base(String value) {
                this.value = value;
            }
        }
        
        static class Derived extends Base {
            Derived(String input) {
                // Want to transform input before super
                // String transformed = input.trim().toUpperCase();
                // But can't - super must be first!
                
                super(transform(input));  // Must use helper method
            }
            
            // Workaround: static helper method
            private static String transform(String input) {
                return input == null ? "" : input.trim().toUpperCase();
            }
            
            /**
             * PROBLEM:
             * - Extra static method needed
             * - Less readable
             * - Cannot use local variables
             * - Cannot use complex logic
             */
        }
    }
    
    /**
     * Problem 3: Cannot use complex expressions
     */
    static class Before_ComplexExpressionProblem {
        
        static class Configuration {
            protected final Map<String, String> settings;
            
            Configuration(Map<String, String> settings) {
                this.settings = settings;
            }
        }
        
        static class AppConfig extends Configuration {
            AppConfig(String configFile) {
                // Want to:
                // Map<String, String> settings = new HashMap<>();
                // settings.put("file", configFile);
                // settings.putAll(loadDefaults());
                // super(settings);
                
                // But can't - super must be first!
                super(buildSettings(configFile));  // Workaround
            }
            
            private static Map<String, String> buildSettings(String file) {
                Map<String, String> settings = new HashMap<>();
                settings.put("file", file);
                // settings.putAll(loadDefaults());
                return settings;
            }
            
            /**
             * PROBLEM:
             * - Complex logic in static method
             * - Harder to read and maintain
             * - Cannot share local state
             */
        }
    }
    
    /**
     * Problem 4: Poor error messages
     */
    static class Before_ErrorMessageProblem {
        
        static class User {
            protected final String username;
            
            User(String username) {
                this.username = username;
            }
        }
        
        static class AdminUser extends User {
            AdminUser(String username, String adminCode) {
                super(username);
                
                // Validate after super - bad for error messages
                if (adminCode == null || adminCode.length() < 8) {
                    throw new IllegalArgumentException(
                        "Admin code must be at least 8 characters for user: " + username
                        // ^ Already constructed User!
                    );
                }
                
                /**
                 * PROBLEM:
                 * - Error mentions constructed object
                 * - Confusing - object exists but is invalid
                 * - Better to fail BEFORE construction
                 */
            }
        }
    }
    
    /**
     * Common workarounds
     */
    public static void commonWorkarounds() {
        /**
         * WORKAROUND 1: Static helper methods
         * 
         * class Derived extends Base {
         *   Derived(String input) {
         *     super(transform(input));
         *   }
         *   
         *   private static String transform(String input) {
         *     // Complex logic here
         *   }
         * }
         * 
         * Problems:
         * - Extra methods clutter
         * - Less readable
         * - Cannot use instance context
         * 
         * 
         * WORKAROUND 2: Factory methods
         * 
         * class Derived extends Base {
         *   private Derived(String value) {
         *     super(value);
         *   }
         *   
         *   static Derived create(String input) {
         *     String transformed = transform(input);
         *     validate(transformed);
         *     return new Derived(transformed);
         *   }
         * }
         * 
         * Problems:
         * - Cannot use new keyword
         * - More complex API
         * - Extra indirection
         * 
         * 
         * WORKAROUND 3: Builder pattern
         * 
         * class Derived extends Base {
         *   Derived(Builder builder) {
         *     super(builder.value);
         *   }
         *   
         *   static class Builder {
         *     // Build and validate
         *   }
         * }
         * 
         * Problems:
         * - Heavy machinery for simple case
         * - More code to maintain
         * 
         * 
         * All workarounds are verbose and awkward!
         */
    }
}
```

## 2. Pre-Construction Validation

```java
/**
 * PRE-CONSTRUCTION VALIDATION
 * 
 * Validate arguments before super() - Java 22

import java.util.*;

public class PreConstructionValidation {
    
    /**
     * Basic validation before super
     */
    static class BasicValidation {
        
        static class Person {
            protected final String name;
            
            Person(String name) {
                this.name = Objects.requireNonNull(name);
            }
        }
        
        static class Employee extends Person {
            private final String employeeId;
            
            // Java 22: Can validate BEFORE super!
            Employee(String name, String employeeId) {
                // Validate before construction
                if (name == null || name.isBlank()) {
                    throw new IllegalArgumentException("Name required");
                }
                if (employeeId == null || employeeId.isBlank()) {
                    throw new IllegalArgumentException("Employee ID required");
                }
                
                super(name);  // Call super after validation
                this.employeeId = employeeId;
                
                /**
                 * BENEFITS:
                 * ✓ Validation before any construction
                 * ✓ Fail fast - no partial objects
                 * ✓ Clearer error messages
                 * ✓ More efficient (no wasted construction)
                 */
            }
        }
        
        public static void demo() {
            try {
                new Employee("", "EMP001");
            } catch (IllegalArgumentException e) {
                System.out.println("Failed before super: " + e.getMessage());
            }
        }
    }
    
    /**
     * Complex validation
     */
    static class ComplexValidation {
        
        static class Account {
            protected final String accountNumber;
            
            Account(String accountNumber) {
                this.accountNumber = accountNumber;
            }
        }
        
        static class BankAccount extends Account {
            private final String routingNumber;
            
            BankAccount(String accountNumber, String routingNumber) {
                // Complex validation before super
                validateAccountNumber(accountNumber);
                validateRoutingNumber(routingNumber);
                validateCompatibility(accountNumber, routingNumber);
                
                super(accountNumber);
                this.routingNumber = routingNumber;
            }
            
            private static void validateAccountNumber(String number) {
                if (number == null || number.length() != 10) {
                    throw new IllegalArgumentException(
                        "Account number must be 10 digits"
                    );
                }
                if (!number.matches("\\d{10}")) {
                    throw new IllegalArgumentException(
                        "Account number must contain only digits"
                    );
                }
            }
            
            private static void validateRoutingNumber(String number) {
                if (number == null || number.length() != 9) {
                    throw new IllegalArgumentException(
                        "Routing number must be 9 digits"
                    );
                }
                // Checksum validation
                int sum = 0;
                for (int i = 0; i < 9; i++) {
                    int digit = number.charAt(i) - '0';
                    sum += digit * (3 - (i % 3));
                }
                if (sum % 10 != 0) {
                    throw new IllegalArgumentException(
                        "Invalid routing number checksum"
                    );
                }
            }
            
            private static void validateCompatibility(String account, String routing) {
                // Cross-field validation
                // Example: certain routing numbers only work with certain account patterns
            }
        }
        
        public static void demo() {
            try {
                new BankAccount("1234567890", "123456789");
            } catch (IllegalArgumentException e) {
                System.out.println("Validation failed: " + e.getMessage());
            }
        }
    }
    
    /**
     * Null checking patterns
     */
    static class NullCheckingPatterns {
        
        static class Base {
            protected final String value;
            
            Base(String value) {
                this.value = value;
            }
        }
        
        static class Derived extends Base {
            private final String extra;
            
            Derived(String value, String extra) {
                // Check nulls before super
                Objects.requireNonNull(value, "value cannot be null");
                Objects.requireNonNull(extra, "extra cannot be null");
                
                super(value);
                this.extra = extra;
                
                /**
                 * BENEFIT:
                 * If validation fails, no superclass construction
                 * Clear which parameter was null
                 */
            }
            
            // Alternative: Use local variable
            Derived(String value, String extra, boolean alt) {
                // Store validated values
                String validValue = Objects.requireNonNull(value);
                String validExtra = Objects.requireNonNull(extra);
                
                super(validValue);
                this.extra = validExtra;
            }
        }
    }
    
    /**
     * Business rule validation
     */
    static class BusinessRuleValidation {
        
        static class Order {
            protected final double amount;
            
            Order(double amount) {
                this.amount = amount;
            }
        }
        
        static class DiscountedOrder extends Order {
            private final double discount;
            
            DiscountedOrder(double amount, double discount) {
                // Validate business rules before super
                if (amount < 0) {
                    throw new IllegalArgumentException(
                        "Amount cannot be negative"
                    );
                }
                if (discount < 0 || discount > 1) {
                    throw new IllegalArgumentException(
                        "Discount must be between 0 and 1"
                    );
                }
                if (amount < 100 && discount > 0.1) {
                    throw new IllegalArgumentException(
                        "Orders under $100 cannot have discount > 10%"
                    );
                }
                
                super(amount);
                this.discount = discount;
            }
            
            /**
             * PATTERN:
             * Validate all business rules before construction
             * Ensures object is valid from creation
             */
        }
    }
    
    public static void main(String[] args) {
        System.out.println("=== Pre-Construction Validation ===");
        BasicValidation.demo();
```

```java
import java.util.*;
import java.time.*;

public class FieldInitializationBeforeSuper {
    
    /**
     * Transform parameters before super
     */
    static class ParameterTransformation {
        
        static class Person {
            protected final String name;
            
            Person(String name) {
                this.name = name;
            }
        }
        
        static class FormattedPerson extends Person {
            FormattedPerson(String name) {
                // Transform before super - natural and clean!
                String formatted = name.trim().toUpperCase();
                
                super(formatted);
                
                /**
                 * Before Java 22:
                 * super(name.trim().toUpperCase());
                 * or
                 * super(transform(name));  // static helper
                 * 
                 * After Java 22:
                 * Direct transformation in constructor
                 * More readable!
                 */
            }
        }
    }
    
    /**
     * Compute derived values
     */
    static class DerivedValues {
        
        static class Rectangle {
            protected final double area;
            
            Rectangle(double area) {
                this.area = area;
            }
        }
        
        static class Square extends Rectangle {
            private final double side;
            
            Square(double side) {
                // Compute area before super
                double area = side * side;
                
                super(area);
                this.side = side;
            }
        }
        
        static class Circle extends Rectangle {
            private final double radius;
            
            Circle(double radius) {
                // Compute area with local variables
                double pi = Math.PI;
                double area = pi * radius * radius;
                
                super(area);
                this.radius = radius;
            }
        }
    }
    
    /**
     * Build complex objects
     */
    static class ComplexObjectConstruction {
        
        static class Configuration {
            protected final Map<String, String> settings;
            
            Configuration(Map<String, String> settings) {
                this.settings = settings;
            }
        }
        
        static class AppConfiguration extends Configuration {
            private final String appName;
            
            AppConfiguration(String appName, String environment) {
                // Build complex map before super
                Map<String, String> settings = new HashMap<>();
                settings.put("app.name", appName);
                settings.put("app.environment", environment);
                settings.put("app.timestamp", Instant.now().toString());
                
                // Add environment-specific settings
                if ("production".equals(environment)) {
                    settings.put("app.debug", "false");
                    settings.put("app.logging", "error");
                } else {
                    settings.put("app.debug", "true");
                    settings.put("app.logging", "debug");
                }
                
                super(settings);
                this.appName = appName;
                
                /**
                 * BENEFIT:
                 * Complex initialization logic inline
                 * No need for static helper methods
                 * Clear and readable
                 */
            }
        }
    }
    
    /**
     * Default values and fallbacks
     */
    static class DefaultValues {
        
        static class Server {
            protected final String host;
            protected final int port;
            
            Server(String host, int port) {
                this.host = host;
                this.port = port;
            }
        }
        
        static class WebServer extends Server {
            WebServer(String host, Integer port) {
                // Apply defaults before super
                String actualHost = host != null ? host : "localhost";
                int actualPort = port != null ? port : 8080;
                
                super(actualHost, actualPort);
            }
        }
    }
    
    /**
     * Type conversion

    static class TypeConversion {
        
        static class NumericData {
            protected final double value;
            
            NumericData(double value) {
                this.value = value;
            }
        }
        
        static class IntegerData extends NumericData {
            IntegerData(String stringValue) {
                // Parse before super
                int intValue = Integer.parseInt(stringValue);
                double doubleValue = intValue;
                
                super(doubleValue);
            }
        }
        
        static class CurrencyData extends NumericData {
            CurrencyData(String currencyString) {
                // Remove currency symbols and parse
                String cleaned = currencyString.replace("$", "")
                                              .replace(",", "")
                                              .trim();
                double value = Double.parseDouble(cleaned);
                
                super(value);
            }
        }
    }
    
    /**
     * Conditional initialization
     */
    static class ConditionalInitialization {
        
        static class Logger {
            protected final String output;
            
            Logger(String output) {
                this.output = output;
            }
        }
        
        static class FileLogger extends Logger {
            FileLogger(String filename, boolean useTimestamp) {

                String actualFilename;
                if (useTimestamp) {
                    String timestamp = Instant.now().toString()

                    actualFilename = timestamp + "_" + filename;
                } else {
                    actualFilename = filename;
                }
                
                super(actualFilename);
            }
        }
    }
    
    /**
     * Multiple transformations
     */
    static class MultipleTransformations {
        
        static class User {

            User(String email) {

                 * Clear pipeline of transformations
                 * Each step visible
                 * Easy to debug
                 */
            }
        }
    }
```

```java
import java.util.*;
import java.util.function.*;

public class ConstructorFlexibility {
    
    /**
     * Factory-like logic in constructor
     */
    static class FactoryLikeLogic {
        
        static class Connection {
            protected final String connectionString;
            
            Connection(String connectionString) {
                this.connectionString = connectionString;
            }
        }
        
        static class DatabaseConnection extends Connection {
            private final String database;
            
            DatabaseConnection(String host, int port, String database) {
                // Build connection string inline
                String connectionString = String.format(
                    "jdbc:postgresql://%s:%d/%s",
                    host, port, database
                );
                
                super(connectionString);
                this.database = database;
                
                /**
                 * Before: Needed static factory
                 * After: Logic in constructor
                 */
            }
        }
    }
    
    /**
     * Resource loading
     */
    static class ResourceLoading {
        
        static class Config {
            protected final Properties properties;
            
            Config(Properties properties) {
                this.properties = properties;
            }
        }
        
        static class FileConfig extends Config {
            FileConfig(String filename) {
                // Load resource before super
                Properties props = new Properties();
                try (var input = getClass().getResourceAsStream(filename)) {
                    if (input != null) {
                        props.load(input);
                    }
                } catch (Exception e) {
                    // Handle error before construction
                    throw new RuntimeException("Failed to load config: " + filename, e);
                }
                
                super(props);
            }
        }
    }
    
    /**
     * Builder-style initialization
     */
    static class BuilderStyle {
        
        static class HttpRequest {
            protected final Map<String, String> headers;
            protected final String body;
            
            HttpRequest(Map<String, String> headers, String body) {
                this.headers = headers;
                this.body = body;
            }
        }
        
        static class JsonRequest extends HttpRequest {
            JsonRequest(String jsonBody) {
                // Build headers map
                Map<String, String> headers = new HashMap<>();
                headers.put("Content-Type", "application/json");
                headers.put("Accept", "application/json");
                headers.put("Content-Length", String.valueOf(jsonBody.length()));
                
                super(headers, jsonBody);
            }
        }
    }
    
    /**
     * Callback-based initialization
     */
    static class CallbackInitialization {
        
        static class Widget {
            protected final String config;
            
            Widget(String config) {
                this.config = config;
            }
        }
        
        static class CustomWidget extends Widget {
            CustomWidget(Consumer<StringBuilder> configBuilder) {
                // Use callback to build config
                StringBuilder sb = new StringBuilder();
                configBuilder.accept(sb);
                String config = sb.toString();
                
                super(config);
            }
        }
        
        public static void demo() {
            new CustomWidget(sb -> {
                sb.append("theme=dark;");
                sb.append("size=large;");
                sb.append("position=top;");
            });
        }
    }
    
    /**
     * Validation with custom logic
     */
    static class CustomValidation {
        
        static class Account {
            protected final String id;
            
            Account(String id) {
                this.id = id;
            }
        }
        
        static class SecureAccount extends Account {
            private static final Set<String> RESERVED_IDS = 
                Set.of("admin", "root", "system");
            
            SecureAccount(String id) {
                // Custom validation before super
                if (RESERVED_IDS.contains(id.toLowerCase())) {
                    throw new IllegalArgumentException(
                        "ID '" + id + "' is reserved"
                    );
                }
                
                // Validate format
                if (!id.matches("[a-zA-Z][a-zA-Z0-9_]{2,19}")) {
                    throw new IllegalArgumentException(
                        "ID must start with letter, 3-20 chars, alphanumeric and underscore"
                    );
                }
                
                super(id);
            }
        }
    }
    
    /**
     * Lazy computation
     */
    static class LazyComputation {
        
        static class ExpensiveComputation {
            protected final String result;
            
            ExpensiveComputation(String result) {
                this.result = result;
            }
        }
        
        static class CachedComputation extends ExpensiveComputation {
            private static final Map<String, String> cache = new HashMap<>();
            
            CachedComputation(String input) {
                // Check cache before expensive computation
                String result = cache.get(input);
                
                if (result == null) {
                    // Expensive computation
                    result = performExpensiveComputation(input);
                    cache.put(input, result);
                }
                
                super(result);
            }
            
            private static String performExpensiveComputation(String input) {
                // Simulate expensive work
                return input.toUpperCase();
            }
        }
    }
    
    /**
     * Error context building
     */
    static class ErrorContextBuilding {
        
        static class DataRecord {
            protected final String data;
            
            DataRecord(String data) {
                this.data = data;
            }
        }

                    source, lineNumber

                    throw new IllegalArgumentException(

                if (!data.matches("\\d+")) {
                    throw new IllegalArgumentException(

                    );
                }
                
                super(data);
                
                /**
                 * BENEFIT:
                 * Error messages include full context
                 * Easier debugging
                 */
            }
        }
    }
```

```java
/**

 * Real-world applications

import java.util.*;
import java.time.*;

public class UseCasesAndPatterns {
    
    /**
     * Use case 1: Normalizing user input
     */
    static class NormalizingInput {
        
        static class User {
            protected final String email;
            
            User(String email) {
                this.email = email;
            }
        }
        
        static class RegisteredUser extends User {
            private final String username;
            
            RegisteredUser(String email, String username) {
                // Normalize inputs before super
                String normalizedEmail = email.trim()
                                             .toLowerCase()
                                             .replace(" ", "");
                String normalizedUsername = username.trim();
                
                // Validate normalized values
                if (!normalizedEmail.contains("@")) {
                    throw new IllegalArgumentException("Invalid email");
                }
                if (normalizedUsername.length() < 3) {
                    throw new IllegalArgumentException("Username too short");
                }
                
                super(normalizedEmail);
                this.username = normalizedUsername;
            }
        }
    }
    
    /**
     * Use case 2: Parsing and validating
     */
    static class ParsingAndValidating {
        
        static class Coordinate {
            protected final double latitude;
            protected final double longitude;
            
            Coordinate(double latitude, double longitude) {
                this.latitude = latitude;
                this.longitude = longitude;
            }
        }
        
        static class ParsedCoordinate extends Coordinate {
            ParsedCoordinate(String coordinateString) {
                // Parse: "40.7128,-74.0060"
                String[] parts = coordinateString.split(",");
                if (parts.length != 2) {
                    throw new IllegalArgumentException(
                        "Invalid coordinate format: " + coordinateString
                    );
                }
                
                double lat = Double.parseDouble(parts[0].trim());
                double lon = Double.parseDouble(parts[1].trim());
                
                // Validate ranges
                if (lat < -90 || lat > 90) {
                    throw new IllegalArgumentException(
                        "Latitude must be between -90 and 90: " + lat
                    );
                }
                if (lon < -180 || lon > 180) {
                    throw new IllegalArgumentException(
                        "Longitude must be between -180 and 180: " + lon
                    );
                }
                
                super(lat, lon);
            }
        }
    }
    
    /**
     * Use case 3: Environment-based configuration
     */
    static class EnvironmentBasedConfig {
        
        static class Service {
            protected final String endpoint;
            protected final int timeout;
            
            Service(String endpoint, int timeout) {
                this.endpoint = endpoint;
                this.timeout = timeout;
            }
        }
        
        static class EnvironmentAwareService extends Service {
            EnvironmentAwareService(String baseEndpoint) {
                // Determine environment
                String env = System.getenv("APP_ENV");
                if (env == null) {
                    env = "development";
                }
                
                // Configure based on environment
                String endpoint;
                int timeout;
                
                switch (env) {
                    case "production":
                        endpoint = "https://" + baseEndpoint;
                        timeout = 30000;
                        break;
                    case "staging":
                        endpoint = "https://staging." + baseEndpoint;
                        timeout = 60000;
                        break;
                    default:
```

## 6. Limitations and Rules

### What You Cannot Do

```java
/**
 * LIMITATIONS AND RULES
 * 
 * Understanding the constraints
 */

public class LimitationsAndRules {
    
    /**
     * Rule 1: Cannot access 'this'
     */
    static class CannotAccessThis {
        
        static class Base {
            protected final String value;
            
            Base(String value) {
                this.value = value;
            }
        }
        
        static class Derived extends Base {
            private int count;
            
            Derived(String value) {
                // ✗ CANNOT do this:
                // this.count = 0;  // ERROR: cannot reference 'this' before super
                // String v = this.toString();  // ERROR
                // this.someMethod();  // ERROR
                
                // ✓ CAN do this:
                String processed = value.toUpperCase();  // OK - no 'this'
                
                super(processed);
                
                // ✓ After super, can use 'this':
                this.count = 0;  // OK
            }
            
            /**
             * RULE:
             * Cannot use 'this' before super() completes
             * This is the fundamental safety guarantee
             */
        }
    }
    
    /**
     * Rule 2: Cannot access instance fields
     */
    static class CannotAccessInstanceFields {
        
        static class Base {
            Base(String value) {}
        }
        
        static class Derived extends Base {
            private String prefix = "default";
            
            Derived(String value) {
                // ✗ CANNOT do this:
                // String v = prefix + value;  // ERROR: cannot access instance field
                // String v = this.prefix + value;  // ERROR
                
                // ✓ CAN do this:
                String defaultPrefix = "default";  // Local variable OK
                String v = defaultPrefix + value;
                
                super(v);
            }
            
            /**
             * RULE:
             * Instance fields don't exist yet
             * Use local variables instead
             */
        }
    }
    
    /**
     * Rule 3: Cannot call instance methods
     */
    static class CannotCallInstanceMethods {
        
        static class Base {
            Base(String value) {}
        }
        
        static class Derived extends Base {
            Derived(String value) {
                // ✗ CANNOT do this:
                // String processed = process(value);  // ERROR: instance method
                // String processed = this.process(value);  // ERROR
                
                // ✓ CAN do this:
                String processed = staticProcess(value);  // Static method OK
                
                super(processed);
            }
            
            // ✗ Instance method
            private String process(String s) {
                return s.toUpperCase();
            }
            
            // ✓ Static method
            private static String staticProcess(String s) {
                return s.toUpperCase();
            }
            
            /**
             * RULE:
             * Can only call static methods before super
             * Instance methods require 'this'
             */
        }
    }
    
    /**
     * Rule 4: Must eventually call super() or this()
     */
    static class MustCallSuperOrThis {
        
        static class Base {
            Base(String value) {}
        }
        
        static class Derived extends Base {
            // ✗ WRONG: No super() call
            /*
            Derived(String value) {
                String processed = value.toUpperCase();
                // ERROR: must call super() or this()
            }
            */
            
            // ✓ CORRECT: Calls super()
            Derived(String value) {
                String processed = value.toUpperCase();
                super(processed);  // Required
            }
            
            // ✓ CORRECT: Calls this()
            Derived(String value, boolean flag) {
                String processed = value.toUpperCase();
                this(processed);  // Calls other constructor
            }
            
            /**
             * RULE:
             * Every constructor path must call super() or this()
             * Cannot just return without calling one
             */
        }
    }
    
    /**
     * Rule 5: Control flow restrictions
     */
    static class ControlFlowRestrictions {
        
        static class Base {
            Base(String value) {}
        }
        
        static class Derived extends Base {
            Derived(String value, boolean condition) {
                // ✓ CAN have conditionals
                String processed;
                if (condition) {
                    processed = value.toUpperCase();
                } else {
                    processed = value.toLowerCase();
                }
                
                super(processed);
            }
            
            Derived(String value, int mode) {
                // ✓ CAN have loops
                StringBuilder sb = new StringBuilder();
                for (int i = 0; i < mode; i++) {
                    sb.append(value);
                }
                
                super(sb.toString());
            }
            
            Derived(String value, String extra) {
                // ✗ CANNOT return before super
                /*
                if (value == null) {
                    return;  // ERROR: must call super() before return
                }
                */
                
                // ✓ Must throw or call super
                if (value == null) {
                    throw new IllegalArgumentException("null value");
                }
                
                super(value);
            }
            
            /**
             * RULE:
             * Can use normal control flow (if, loops)
             * But all paths must reach super() or throw
             * Cannot return before super()
             */
        }
    }
    
    /**
     * Rule 6: Exception handling
     */
    static class ExceptionHandling {
        
        static class Base {
            Base(String value) {}
        }
        
        static class Derived extends Base {
            Derived(String value) {
                // ✓ CAN have try-catch before super
                String processed;
                try {
                    processed = parseAndValidate(value);
                } catch (Exception e) {
                    throw new IllegalArgumentException("Invalid value", e);
                }
                
                super(processed);
            }
            
            Derived(String value, boolean flag) {
                // ✓ CAN use try-with-resources
                String result;
                try (var scanner = new java.util.Scanner(value)) {
                    result = scanner.nextLine();
                } catch (Exception e) {
                    result = "default";
                }
                
                super(result);
            }
            
            private static String parseAndValidate(String s) {
                return s;
            }
            
            /**
             * RULE:
             * Exception handling works normally
             * Can catch, rethrow, use try-with-resources
             * As long as all paths reach super() or throw
             */
        }
    }
    
    /**
     * What IS allowed
     */
    public static void whatIsAllowed() {
        /**
         * ALLOWED BEFORE SUPER:
         * 
         * ✓ Local variable declarations
         * ✓ Static method calls
         * ✓ Static field access
         * ✓ Method parameters
         * ✓ Literals and constants
         * ✓ Control flow (if, loops, switch)
         * ✓ Exception handling
         * ✓ Lambda expressions (that don't capture 'this')
         * ✓ Creating new objects
         * ✓ Array access
         * ✓ Arithmetic operations
         * 
         * 
         * NOT ALLOWED BEFORE SUPER:
         * 
         * ✗ this keyword
         * ✗ Instance field access
         * ✗ Instance method calls
         * ✗ super keyword (except super() call)
         * ✗ Returning from constructor
         * ✗ Anything that requires 'this'
         */
    }
    
    /**
     * Compiler enforcement
     */
    public static void compilerEnforcement() {
        /**
         * COMPILER CHECKS:
         * 
         * The compiler ensures:
         * 1. 'this' not used before super()
         * 2. All code paths call super() or this()
         * 3. super() called at most once
         * 4. No return before super()
         * 
         * These are compile-time errors:
         * - Clear error messages
         * - Cannot accidentally violate
         * - Type-safe
         * 
         * 
         * SAFETY GUARANTEE:
         * 
         * Object is not accessible until fully constructed
         * - Superclass constructor completes
         * - Then subclass fields initialized
         * - Then 'this' available
         * 
         * This prevents:
         * - Accessing uninitialized fields
         * - Publishing partially constructed objects
         * - Calling methods on incomplete objects
         */
    }
}
```

## Summary

### Quick Reference

**Before Java 22:**

```java
// ✗ Error: statements before super
class Derived extends Base {
    Derived(String value) {
        String trimmed = value.trim();  // ERROR!
        super(trimmed);
    }
}

// Workaround: static helper
class Derived extends Base {
    Derived(String value) {
        super(transform(value));
    }
    
    private static String transform(String s) {
        return s.trim();
    }
}
```

**After Java 22:**

```java
// ✓ Allowed: statements before super
class Derived extends Base {
    Derived(String value) {
        String trimmed = value.trim();  // OK!
        super(trimmed);
    }
}

// ✓ Can validate before super
class Derived extends Base {
    Derived(String value) {
        if (value == null) {
            throw new IllegalArgumentException();
        }
        String processed = value.trim();
        super(processed);
    }
}
```

**What's Allowed:**

```java
// ✓ Local variables
String processed = value.trim();

// ✓ Static method calls
String result = transform(value);

// ✓ Control flow
if (condition) {
    value = "A";
} else {
    value = "B";
}

// ✓ Exception handling
try {
    value = parse(input);
} catch (Exception e) {
    throw new IllegalArgumentException(e);
}
```

**What's NOT Allowed:**

```java
// ✗ Cannot use 'this'
this.field = value;

// ✗ Cannot access instance fields
value = this.defaultValue;

// ✗ Cannot call instance methods
value = this.process(input);

// ✗ Cannot return before super
if (invalid) return;  // ERROR
```

- **Cleaner code**: No more static helper method workarounds
- **Fail fast**: Validation errors before any object creation

- **Flexible**: Enables new constructor patterns
