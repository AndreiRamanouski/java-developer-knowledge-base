# Process API Enhancements (Java 9+)

## Overview

Java 9 introduced comprehensive enhancements to the Process API, making it much easier to manage and monitor operating system processes. The new ProcessHandle interface provides access to process IDs (PIDs), process information, parent-child relationships, and process lifecycle management. Combined with improvements to ProcessBuilder, developers can now create, control, destroy, and monitor processes with platform-independent code. This guide covers ProcessHandle for fine-grained process control, retrieving detailed process information (PID, arguments, command, user, etc.), ProcessBuilder enhancements for easier process creation, various methods for destroying processes (gracefully and forcefully), and techniques for monitoring process state and receiving lifecycle notifications.

## 1. ProcessHandle for Process Control

### Managing Processes with ProcessHandle

```java
/**
 * PROCESSHANDLE
 * 
 * New Java 9 API for process management
 */

import java.io.*;
import java.util.*;
import java.util.concurrent.*;
import java.time.*;

public class ProcessHandleBasics {
    
    /**
     * Getting current process
     */
    public static void currentProcess() {
        System.out.println("=== Current Process ===");
        
        // Get handle to current JVM process
        ProcessHandle current = ProcessHandle.current();
        
        System.out.println("PID: " + current.pid());
        System.out.println("Is alive: " + current.isAlive());
        
        /**
         * PROCESSHANDLE.CURRENT():
         * 
         * Returns: ProcessHandle for current JVM process
         * 
         * Use to:
         * - Get current process PID
         * - Access current process info
         * - Compare with other processes
         * - Check if current process is alive
         */
    }
    
    /**
     * Getting process by PID
     */
    public static void processByPid() {
        System.out.println("\n=== Process by PID ===");
        
        // Get current PID
        long currentPid = ProcessHandle.current().pid();
        System.out.println("Current PID: " + currentPid);
        
        // Get handle by PID
        Optional<ProcessHandle> handle = ProcessHandle.of(currentPid);
        
        if (handle.isPresent()) {
            System.out.println("Found process: " + handle.get().pid());
        } else {
            System.out.println("Process not found");
        }
        
        // Try non-existent PID
        Optional<ProcessHandle> nonExistent = ProcessHandle.of(999999);
        System.out.println("Non-existent: " + nonExistent.isPresent());
        
        /**
         * PROCESSHANDLE.OF(PID):
         * 
         * Returns: Optional<ProcessHandle>
         * 
         * Present if:
         * - Process exists
         * - Process is accessible (permissions)
         * 
         * Empty if:
         * - Process doesn't exist
         * - No permission to access
         */
    }
    
    /**
     * All running processes
     */
    public static void allProcesses() {
        System.out.println("\n=== All Processes ===");
        
        // Get all accessible processes
        Stream<ProcessHandle> allProcesses = ProcessHandle.allProcesses();
        
        // Count processes
        long count = allProcesses.count();
        System.out.println("Total accessible processes: " + count);
        
        // First 10 processes
        ProcessHandle.allProcesses()
            .limit(10)
            .forEach(p -> System.out.println("PID: " + p.pid()));
        
        /**
         * PROCESSHANDLE.ALLPROCESSES():
         * 
         * Returns: Stream<ProcessHandle> of all processes
         * 
         * Notes:
         * - Platform-dependent (OS security)
         * - May not include all system processes
         * - Stream is lazy (processes may come/go)
         * - Includes current process
         * 
         * Use for:
         * - Finding processes
         * - Monitoring system
         * - Collecting statistics
         */
    }
    
    /**
     * Creating and getting process handle
     */
    public static void createdProcess() throws IOException {
        System.out.println("\n=== Created Process ===");
        
        // Create a process
        ProcessBuilder pb = new ProcessBuilder("ping", "-c", "3", "localhost");
        Process process = pb.start();
        
        // Get ProcessHandle from Process
        ProcessHandle handle = process.toHandle();
        
        System.out.println("Created PID: " + handle.pid());
        System.out.println("Is alive: " + handle.isAlive());
        
        // Get same handle via PID
        Optional<ProcessHandle> sameHandle = ProcessHandle.of(handle.pid());
        System.out.println("Same process: " + 
            sameHandle.map(h -> h.equals(handle)).orElse(false));
        
        /**
         * PROCESS.TOHANDLE():
         * 
         * Process (Java 5):
         * - Created via ProcessBuilder
         * - Has I/O streams
         * - Limited API
         * 
         * ProcessHandle (Java 9):
         * - Can be obtained from any PID
         * - Rich API for process info
         * - No I/O streams
         * 
         * Conversion:
         * Process → ProcessHandle: process.toHandle()
         * ProcessHandle → Process: Not possible
         */
    }
    
    /**
     * Parent and children
     */
    public static void parentAndChildren() throws IOException {
        System.out.println("\n=== Parent and Children ===");
        
        // Current process
        ProcessHandle current = ProcessHandle.current();
        
        // Parent process
        Optional<ProcessHandle> parent = current.parent();
        parent.ifPresent(p -> 
            System.out.println("Parent PID: " + p.pid()));
        
        // Create child processes
        ProcessBuilder pb = new ProcessBuilder("sleep", "10");
        Process child1 = pb.start();
        Process child2 = pb.start();
        
        // Children of current process
        Stream<ProcessHandle> children = current.children();
        System.out.println("Children:");
        children.forEach(child -> 
            System.out.println("  Child PID: " + child.pid()));
        
        // Cleanup
        child1.destroy();
        child2.destroy();
        
        /**
         * PROCESS TREE:
         * 
         * parent():
         * - Returns Optional<ProcessHandle>
         * - Empty if parent no longer exists
         * - Empty if no permission
         * 
         * children():
         * - Returns Stream<ProcessHandle>
         * - Direct children only (not grandchildren)
         * - Snapshot at call time
         * 
         * descendants():
         * - Returns Stream<ProcessHandle>
         * - All descendants (children, grandchildren, etc.)
         * - Entire subtree
         */
    }
    
    /**
     * Process descendants
     */
    public static void processDescendants() throws IOException {
        System.out.println("\n=== Process Descendants ===");
        
        // Create process hierarchy
        // Current → child → grandchild
        ProcessBuilder pb = new ProcessBuilder(
            "bash", "-c", "sleep 5 & sleep 5 & wait"
        );
        Process child = pb.start();
        
        Thread.sleep(1000);  // Let children spawn
        
        ProcessHandle childHandle = child.toHandle();
        
        // Direct children
        long childrenCount = childHandle.children().count();
        System.out.println("Direct children: " + childrenCount);
        
        // All descendants
        long descendantsCount = childHandle.descendants().count();
        System.out.println("All descendants: " + descendantsCount);
        
        // Cleanup
        child.destroyForcibly();
        
        /**
         * CHILDREN VS DESCENDANTS:
         * 
         * children():
         * Process
         *   └─ Child 1
         *   └─ Child 2
         * Returns: [Child 1, Child 2]
         * 
         * descendants():
         * Process
         *   └─ Child 1
         *       └─ Grandchild 1
         *       └─ Grandchild 2
         *   └─ Child 2
         * Returns: [Child 1, Grandchild 1, Grandchild 2, Child 2]
         */
    } catch (InterruptedException e) {
        Thread.currentThread().interrupt();
    }
    
    /**
     * Comparing ProcessHandles
     */
    public static void comparingHandles() {
        System.out.println("\n=== Comparing Handles ===");
        
        ProcessHandle h1 = ProcessHandle.current();
        ProcessHandle h2 = ProcessHandle.current();
        
        // Same process
        System.out.println("Same process: " + h1.equals(h2));
        System.out.println("Same hash: " + (h1.hashCode() == h2.hashCode()));
        
        /**
         * PROCESSHANDLE EQUALITY:
         * 
         * Two ProcessHandles equal if:
         * - Same PID
         * - Same process
         * 
         * hashCode():
         * - Based on PID
         * - Can use in HashSet/HashMap
         * 
         * compareTo():
         * - Compares PIDs
         * - Can sort by PID
         */
    }
    
    public static void main(String[] args) throws IOException {
        currentProcess();
        processByPid();
        allProcesses();
        createdProcess();
        parentAndChildren();
    }
}
```

## 2. Process Information (PID, Arguments, etc.)

### Retrieving Detailed Process Information

```java
/**
 * PROCESS INFORMATION
 * 
 * Getting detailed info about processes
 */

import java.time.*;
import java.util.*;

public class ProcessInformation {
    
    /**
     * ProcessHandle.Info - Process metadata
     */
    public static void processInfo() {
        System.out.println("=== Process Info ===");
        
        ProcessHandle current = ProcessHandle.current();
        ProcessHandle.Info info = current.info();
        
        // Command
        info.command().ifPresent(cmd -> 
            System.out.println("Command: " + cmd));
        
        // Arguments
        info.arguments().ifPresent(args -> 
            System.out.println("Arguments: " + Arrays.toString(args)));
        
        // Command line (command + arguments)
        info.commandLine().ifPresent(cmdLine -> 
            System.out.println("Command line: " + cmdLine));
        
        // User
        info.user().ifPresent(user -> 
            System.out.println("User: " + user));
        
        // Start time
        info.startInstant().ifPresent(start -> 
            System.out.println("Started: " + start));
        
        // Total CPU time
        info.totalCpuDuration().ifPresent(cpu -> 
            System.out.println("CPU time: " + cpu));
        
        /**
         * PROCESSHANDLE.INFO:
         * 
         * All fields are Optional because:
         * - Platform-dependent availability
         * - Security restrictions
         * - Process state (zombie, etc.)
         * 
         * Available info:
         * - command(): Executable path
         * - arguments(): Command-line arguments
         * - commandLine(): Full command line
         * - user(): Process owner
         * - startInstant(): Start time
         * - totalCpuDuration(): CPU time used
         */
    }
    
    /**
     * Detailed info example
     */
    public static void detailedInfo() throws IOException {
        System.out.println("\n=== Detailed Info ===");
        
        // Create process with known arguments
        ProcessBuilder pb = new ProcessBuilder(
            "java", "-version"
        );
        Process process = pb.start();
        
        ProcessHandle.Info info = process.toHandle().info();
        
        System.out.println("=== Process Details ===");
        
        // Command
        String command = info.command().orElse("Unknown");
        System.out.println("Command: " + command);
        
        // Arguments (may not be available on all platforms)
        String[] args = info.arguments().orElse(new String[0]);
        System.out.println("Arguments: " + Arrays.toString(args));
        
        // Command line
        String cmdLine = info.commandLine().orElse("Unknown");
        System.out.println("Command line: " + cmdLine);
        
        // User
        String user = info.user().orElse("Unknown");
        System.out.println("User: " + user);
        
        // Start time
        Instant start = info.startInstant().orElse(Instant.now());
        System.out.println("Started: " + start);
        
        // CPU time
        Duration cpu = info.totalCpuDuration().orElse(Duration.ZERO);
        System.out.println("CPU time: " + cpu);
        
        process.waitFor();
        
        /**
         * PLATFORM DIFFERENCES:
         * 
         * Windows:
         * - command(): Usually available
         * - arguments(): Often empty
         * - user(): Usually available
         * 
         * Linux/Unix:
         * - command(): Usually available
         * - arguments(): Usually available
         * - user(): Usually available
         * 
         * macOS:
         * - Similar to Linux/Unix
         * 
         * Always use Optional methods!
         */
    } catch (InterruptedException e) {
        Thread.currentThread().interrupt();
    }
    
    /**
     * Finding processes by criteria
     */
    public static void findProcesses() {
        System.out.println("\n=== Find Processes ===");
        
        // Find Java processes
        System.out.println("Java processes:");
        ProcessHandle.allProcesses()
            .filter(p -> p.info().command()
                .orElse("")
                .contains("java"))
            .forEach(p -> {
                System.out.println("  PID: " + p.pid());
                p.info().commandLine().ifPresent(cmd ->
                    System.out.println("    Command: " + cmd));
            });
        
        /**
         * COMMON SEARCHES:
         * 
         * By command:
         * .filter(p -> p.info().command()
         *   .map(cmd -> cmd.contains("java"))
         *   .orElse(false))
         * 
         * By user:
         * .filter(p -> p.info().user()
         *   .map(user -> user.equals("admin"))
         *   .orElse(false))
         * 
         * By start time:
         * .filter(p -> p.info().startInstant()
         *   .map(start -> start.isAfter(cutoff))
         *   .orElse(false))
         */
    }
    
    /**
     * Process uptime
     */
    public static void processUptime() {
        System.out.println("\n=== Process Uptime ===");
        
        ProcessHandle current = ProcessHandle.current();
        ProcessHandle.Info info = current.info();
        
        info.startInstant().ifPresent(start -> {
            Duration uptime = Duration.between(start, Instant.now());
            System.out.println("Uptime: " + formatDuration(uptime));
        });
    }
    
    private static String formatDuration(Duration duration) {
        long hours = duration.toHours();
        long minutes = duration.toMinutes() % 60;
        long seconds = duration.getSeconds() % 60;
        return String.format("%d hours, %d minutes, %d seconds",
            hours, minutes, seconds);
    }
    
    /**
     * Process statistics
     */
    public static void processStatistics() {
        System.out.println("\n=== Process Statistics ===");
        
        // Count processes by user
        Map<String, Long> byUser = ProcessHandle.allProcesses()
            .map(p -> p.info().user().orElse("Unknown"))
            .collect(Collectors.groupingBy(
                user -> user,
                Collectors.counting()
            ));
        
        System.out.println("Processes by user:");
        byUser.forEach((user, count) -> 
            System.out.println("  " + user + ": " + count));
        
        // Average CPU time
        OptionalDouble avgCpu = ProcessHandle.allProcesses()
            .map(p -> p.info().totalCpuDuration().orElse(Duration.ZERO))
            .mapToLong(Duration::toMillis)
            .average();
        
        avgCpu.ifPresent(avg -> 
            System.out.println("Average CPU time: " + avg + " ms"));
    }
    
    /**
     * Complete process info display
     */
    public static void displayCompleteInfo(ProcessHandle handle) {
        System.out.println("\n=== Complete Info ===");
        
        System.out.println("PID: " + handle.pid());
        System.out.println("Alive: " + handle.isAlive());
        
        ProcessHandle.Info info = handle.info();
        
        System.out.println("\nDetails:");
        info.command().ifPresentOrElse(
            cmd -> System.out.println("  Command: " + cmd),
            () -> System.out.println("  Command: Not available")
        );
        
        info.arguments().ifPresentOrElse(
            args -> System.out.println("  Arguments: " + 
                Arrays.toString(args)),
            () -> System.out.println("  Arguments: Not available")
        );
        
        info.commandLine().ifPresentOrElse(
            cmdLine -> System.out.println("  Command line: " + cmdLine),
            () -> System.out.println("  Command line: Not available")
        );
        
        info.user().ifPresentOrElse(
            user -> System.out.println("  User: " + user),
            () -> System.out.println("  User: Not available")
        );
        
        info.startInstant().ifPresentOrElse(
            start -> {
                System.out.println("  Started: " + start);
                Duration uptime = Duration.between(start, Instant.now());
                System.out.println("  Uptime: " + formatDuration(uptime));
            },
            () -> System.out.println("  Start time: Not available")
        );
        
        info.totalCpuDuration().ifPresentOrElse(
            cpu -> System.out.println("  CPU time: " + cpu),
            () -> System.out.println("  CPU time: Not available")
        );
        
        // Parent
        handle.parent().ifPresentOrElse(
            parent -> System.out.println("\nParent PID: " + parent.pid()),
            () -> System.out.println("\nParent: None")
        );
        
        // Children
        long childCount = handle.children().count();
        System.out.println("Children: " + childCount);
        
        // Descendants
        long descendantCount = handle.descendants().count();
        System.out.println("Descendants: " + descendantCount);
    }
    
    public static void main(String[] args) throws IOException {
        processInfo();
        detailedInfo();
        findProcesses();
        processUptime();
        processStatistics();
        displayCompleteInfo(ProcessHandle.current());
    }
}
```

## 3. ProcessBuilder Improvements

### Enhanced Process Creation

```java
/**
 * PROCESSBUILDER IMPROVEMENTS
 * 
 * Java 9 enhancements to ProcessBuilder
 */

import java.io.*;
import java.util.*;
import java.util.concurrent.*;

public class ProcessBuilderImprovements {
    
    /**
     * Basic ProcessBuilder usage
     */
    public static void basicUsage() throws IOException {
        System.out.println("=== Basic Usage ===");
        
        ProcessBuilder pb = new ProcessBuilder("java", "-version");
        Process process = pb.start();
        
        // Read output
        try (BufferedReader reader = new BufferedReader(
                new InputStreamReader(process.getErrorStream()))) {
            reader.lines().forEach(System.out::println);
        }
        
        /**
         * PROCESSBUILDER BASICS:
         * 
         * Creation:
         * new ProcessBuilder("command", "arg1", "arg2")
         * new ProcessBuilder(List.of("command", "arg1", "arg2"))
         * 
         * Start:
         * Process process = pb.start();
         * 
         * Streams:
         * process.getInputStream()  - stdout
         * process.getErrorStream()  - stderr  
         * process.getOutputStream() - stdin
         */
    } catch (IOException e) {
        e.printStackTrace();
    }
    
    /**
     * Java 9: Pipeline support
     */
    public static void pipelineSupport() throws IOException {
        System.out.println("\n=== Pipeline Support ===");
        
        // Create pipeline: echo | grep
        List<ProcessBuilder> builders = Arrays.asList(
            new ProcessBuilder("echo", "Hello World"),
            new ProcessBuilder("grep", "World")
        );
        
        // Start pipeline
        List<Process> processes = ProcessBuilder.startPipeline(builders);
        
        // Read output from last process
        Process last = processes.get(processes.size() - 1);
        try (BufferedReader reader = new BufferedReader(
                new InputStreamReader(last.getInputStream()))) {
            reader.lines().forEach(System.out.println);
        }
        
        /**
         * PROCESSBUILDER.STARTPIPELINE():
         * 
         * Java 9 feature
         * 
         * Creates pipeline:
         * process1.stdout → process2.stdin → process3.stdin → ...
         * 
         * Example:
         * echo "Hello" | grep "H" | wc -l
         * 
         * List<ProcessBuilder> pipeline = List.of(
         *   new ProcessBuilder("echo", "Hello"),
         *   new ProcessBuilder("grep", "H"),
         *   new ProcessBuilder("wc", "-l")
         * );
         * List<Process> processes = ProcessBuilder.startPipeline(pipeline);
         * 
         * Benefits:
         * - Automatic pipe setup
         * - All processes started together
         * - Platform-independent
         */
    } catch (IOException e) {
        e.printStackTrace();
    }
    
    /**
     * Directory and environment
     */
    public static void directoryAndEnvironment() throws IOException {
        System.out.println("\n=== Directory and Environment ===");
        
        ProcessBuilder pb = new ProcessBuilder("pwd");
        
        // Set working directory
        pb.directory(new File("/tmp"));
        
        // Get environment
        Map<String, String> env = pb.environment();
        env.put("MY_VAR", "my_value");
        env.remove("UNWANTED_VAR");
        
        // Start and read output
        Process process = pb.start();
        try (BufferedReader reader = new BufferedReader(
                new InputStreamReader(process.getInputStream()))) {
            reader.lines().forEach(System.out.println);
        }
        
        /**
         * DIRECTORY:
         * pb.directory(new File("/path"))
         * - Sets working directory
         * - Default: current directory
         * 
         * ENVIRONMENT:
         * Map<String, String> env = pb.environment()
         * - Modifiable map
         * - Inherits parent environment
         * - Can add/remove/modify
         * 
         * env.put("VAR", "value")     - Add/replace
         * env.remove("VAR")           - Remove
         * env.clear()                 - Clear all
         */
    } catch (IOException e) {
        e.printStackTrace();
    }
    
    /**
     * Redirect I/O
     */
    public static void redirectIO() throws IOException {
        System.out.println("\n=== Redirect I/O ===");
        
        File outputFile = new File("output.txt");
        File errorFile = new File("error.txt");
        
        ProcessBuilder pb = new ProcessBuilder("java", "-version");
        
        // Redirect stdout to file
        pb.redirectOutput(outputFile);
        
        // Redirect stderr to file
        pb.redirectError(errorFile);
        
        // Or redirect both to same file
        // pb.redirectErrorStream(true);
        
        Process process = pb.start();
        process.waitFor();
        
        System.out.println("Output written to: " + outputFile.getAbsolutePath());
        System.out.println("Error written to: " + errorFile.getAbsolutePath());
        
        /**
         * REDIRECT OPTIONS:
         * 
         * redirectOutput(File file)
         * redirectOutput(ProcessBuilder.Redirect.INHERIT)
         * redirectOutput(ProcessBuilder.Redirect.PIPE)
         * 
         * redirectError(File file)
         * redirectError(ProcessBuilder.Redirect.INHERIT)
         * redirectError(ProcessBuilder.Redirect.PIPE)
         * 
         * redirectInput(File file)
         * 
         * redirectErrorStream(true)
         * - Merge stderr into stdout
         * 
         * INHERIT: Use parent process streams
         * PIPE: Use process.getInputStream() etc.
         */
    } catch (IOException | InterruptedException e) {
        e.printStackTrace();
    }
    
    /**
     * Inherit I/O (show in console)
     */
    public static void inheritIO() throws IOException {
        System.out.println("\n=== Inherit I/O ===");
        
        ProcessBuilder pb = new ProcessBuilder("echo", "Hello from child");
        
        // Inherit all I/O from parent
        pb.inheritIO();
        
        // Equivalent to:
        // pb.redirectOutput(ProcessBuilder.Redirect.INHERIT);
        // pb.redirectError(ProcessBuilder.Redirect.INHERIT);
        // pb.redirectInput(ProcessBuilder.Redirect.INHERIT);
        
        Process process = pb.start();
        process.waitFor();
        
        System.out.println("Process completed");
        
        /**
         * INHERITIO():
         * 
         * - Child uses parent's console
         * - Output appears directly in console
         * - No need to read streams
         * - Convenient for interactive processes
         * 
         * Use for:
         * - Interactive programs
         * - Debugging
         * - Scripts that need user input
         */
    } catch (IOException | InterruptedException e) {
        e.printStackTrace();
    }
    
    /**
     * Process example: Run command and capture output
     */
    public static String runCommand(String... command) throws IOException {
        ProcessBuilder pb = new ProcessBuilder(command);
        Process process = pb.start();
        
        StringBuilder output = new StringBuilder();
        try (BufferedReader reader = new BufferedReader(
                new InputStreamReader(process.getInputStream()))) {
            String line;
            while ((line = reader.readLine()) != null) {
                output.append(line).append("\n");
            }
        }
        
        return output.toString();
    } catch (IOException e) {
        throw e;
    }
    
    /**
     * Process example: Run with timeout
     */
    public static int runWithTimeout(long timeout, TimeUnit unit, 
            String... command) throws IOException {
        
        ProcessBuilder pb = new ProcessBuilder(command);
        Process process = pb.start();
        
        try {
            boolean finished = process.waitFor(timeout, unit);
            if (!finished) {
                process.destroy();
                throw new TimeoutException("Process timed out");
            }
            return process.exitValue();
            
        } catch (InterruptedException e) {
            process.destroy();
            Thread.currentThread().interrupt();
            throw new IOException("Interrupted", e);
        } catch (TimeoutException e) {
            throw new IOException(e);
        }
    }
    
    public static void main(String[] args) throws IOException {
        basicUsage();
        pipelineSupport();
        directoryAndEnvironment();
        redirectIO();
        inheritIO();
    }
}
```

## 4. Destroying Processes

### Terminating Processes Gracefully and Forcefully

```java
/**
 * DESTROYING PROCESSES
 * 
 * Different ways to terminate processes
 */

import java.io.*;
import java.util.concurrent.*;

public class DestroyingProcesses {
    
    /**
     * Graceful termination
     */
    public static void gracefulTermination() throws IOException {
        System.out.println("=== Graceful Termination ===");
        
        // Start long-running process
        ProcessBuilder pb = new ProcessBuilder("sleep", "60");
        Process process = pb.start();
        
        System.out.println("Process started, PID: " + process.pid());
        
        // Graceful termination
        process.destroy();
        
        try {
            // Wait for process to terminate
            boolean exited = process.waitFor(5, TimeUnit.SECONDS);
            if (exited) {
                System.out.println("Process terminated gracefully");
                System.out.println("Exit code: " + process.exitValue());
            } else {
                System.out.println("Process didn't exit in time");
            }
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        
        /**
         * DESTROY():
         * 
         * Platform-specific:
         * - Unix/Linux: Sends SIGTERM
         * - Windows: TerminateProcess()
         * 
         * Graceful:
         * - Process can handle signal
         * - Can cleanup resources
         * - May not terminate immediately
         * 
         * Not guaranteed to terminate!
         */
    }
    
    /**
     * Forceful termination
     */
    public static void forcefulTermination() throws IOException {
        System.out.println("\n=== Forceful Termination ===");
        
        // Start long-running process
        ProcessBuilder pb = new ProcessBuilder("sleep", "60");
        Process process = pb.start();
        
        System.out.println("Process started, PID: " + process.pid());
        
        // Forceful termination
        process.destroyForcibly();
        
        try {
            // Wait for process to terminate
            boolean exited = process.waitFor(2, TimeUnit.SECONDS);
            if (exited) {
                System.out.println("Process terminated forcefully");
                System.out.println("Exit code: " + process.exitValue());
            } else {
                System.out.println("Process still running!");
            }
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        
        /**
         * DESTROYFORCIBLY():
         * 
         * Java 8 addition
         * 
         * Platform-specific:
         * - Unix/Linux: Sends SIGKILL
         * - Windows: TerminateProcess()
         * 
         * Forceful:
         * - Process cannot handle signal
         * - No cleanup possible
         * - Terminates immediately
         * 
         * Almost always succeeds
         * (except for zombie processes)
         */
    }
    
    /**
     * Escalating termination strategy
     */
    public static void escalatingTermination(Process process) {
        System.out.println("\n=== Escalating Termination ===");
        
        try {
            // Try graceful first
            process.destroy();
            System.out.println("Sent termination signal");
            
            // Wait 5 seconds
            boolean exited = process.waitFor(5, TimeUnit.SECONDS);
            
            if (exited) {
                System.out.println("Process terminated gracefully");
                return;
            }
            
            // Still running, escalate to force
            System.out.println("Process didn't exit, forcing...");
            process.destroyForcibly();
            
            // Wait 2 more seconds
            exited = process.waitFor(2, TimeUnit.SECONDS);
            
            if (exited) {
                System.out.println("Process terminated forcefully");
            } else {
                System.out.println("Process still running (zombie?)");
            }
            
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        
        /**
         * BEST PRACTICE PATTERN:
         * 
         * 1. Try destroy() first (graceful)
         * 2. Wait reasonable time (5-10 seconds)
         * 3. If still running, destroyForcibly()
         * 4. Wait short time (1-2 seconds)
         * 5. If still running, log warning
         * 
         * Benefits:
         * - Gives process chance to cleanup
         * - Eventually terminates stubborn processes
         * - Handles most scenarios
         */
    }
    
    /**
     * Destroying process tree
     */
    public static void destroyProcessTree() throws IOException {
        System.out.println("\n=== Destroy Process Tree ===");
        
        // Start process that spawns children
        ProcessBuilder pb = new ProcessBuilder(
            "bash", "-c", "sleep 60 & sleep 60 & wait"
        );
        Process process = pb.start();
        
        Thread.sleep(1000);  // Let children spawn
        
        ProcessHandle handle = process.toHandle();
        
        // Count descendants before
        long beforeCount = handle.descendants().count();
        System.out.println("Descendants before: " + beforeCount);
        
        // Destroy all descendants
        handle.descendants().forEach(ProcessHandle::destroy);
        
        // Then destroy parent
        process.destroy();
        
        Thread.sleep(1000);
        
        // Count descendants after
        long afterCount = handle.descendants().count();
        System.out.println("Descendants after: " + afterCount);
        
        /**
         * DESTROYING TREE:
         * 
         * Process with children:
         * Parent
         *   └─ Child 1
         *   └─ Child 2
         * 
         * Option 1: Just destroy parent
         * process.destroy()
         * → May leave orphaned children!
         * 
         * Option 2: Destroy tree (Java 9)
         * handle.descendants().forEach(ProcessHandle::destroy)
         * handle.destroy()
         * → Terminates entire tree
         * 
         * Option 3: Forceful tree
         * handle.descendants().forEach(ProcessHandle::destroyForcibly)
         * handle.destroyForcibly()
         */
    } catch (InterruptedException e) {
        Thread.currentThread().interrupt();
    }
    
    /**
     * Java 9: supportsNormalTermination()
     */
    public static void supportsNormalTermination() throws IOException {
        System.out.println("\n=== Supports Normal Termination ===");
        
        ProcessBuilder pb = new ProcessBuilder("sleep", "60");
        Process process = pb.start();
        
        // Check if platform supports graceful termination
        boolean supports = process.supportsNormalTermination();
        System.out.println("Supports normal termination: " + supports);
        
        if (supports) {
            process.destroy();
            System.out.println("Using destroy()");
        } else {
            process.destroyForcibly();
            System.out.println("Using destroyForcibly()");
        }
        
        /**
         * SUPPORTSNORMALTERMINATION():
         * 
         * Java 9 addition
         * 
         * Returns true if:
         * - destroy() != destroyForcibly()
         * - Platform supports graceful termination
         * 
         * Returns false if:
         * - destroy() == destroyForcibly()
         * - No graceful termination support
         * 
         * Most modern systems: true
         * Some embedded systems: false
         */
    }
    
    /**
     * Timeout and destroy
     */
    public static void timeoutAndDestroy() throws IOException {
        System.out.println("\n=== Timeout and Destroy ===");
        
        ProcessBuilder pb = new ProcessBuilder("sleep", "60");
        Process process = pb.start();
        
        try {
            // Wait up to 3 seconds
            boolean finished = process.waitFor(3, TimeUnit.SECONDS);
            
            if (!finished) {
                System.out.println("Process timed out, destroying");
                process.destroyForcibly();
                process.waitFor();  // Wait for destruction
            }
            
            System.out.println("Exit code: " + process.exitValue());
            
        } catch (InterruptedException e) {
            process.destroyForcibly();
            Thread.currentThread().interrupt();
        }
        
        /**
         * TIMEOUT PATTERN:
         * 
         * boolean finished = process.waitFor(timeout, unit);
         * if (!finished) {
         *   process.destroyForcibly();
         *   process.waitFor();  // Wait for cleanup
         * }
         * 
         * Use for:
         * - Long-running processes
         * - Preventing hangs
         * - Resource cleanup
         * - Timeout enforcement
         */
    }
    
    /**
     * Exit value checking
     */
    public static void exitValueChecking() throws IOException {
        System.out.println("\n=== Exit Value Checking ===");
        
        // Success
        ProcessBuilder pb1 = new ProcessBuilder("echo", "hello");
        Process p1 = pb1.start();
        try {
            p1.waitFor();
            System.out.println("Echo exit: " + p1.exitValue());
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        
        // Failure
        ProcessBuilder pb2 = new ProcessBuilder("false");
        Process p2 = pb2.start();
        try {
            p2.waitFor();
            System.out.println("False exit: " + p2.exitValue());
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        
        /**
         * EXIT VALUES:
         * 
         * Convention (Unix/Linux):
         * 0 = Success
         * 1-255 = Error
         * 
         * Special:
         * 128+N = Killed by signal N
         * 137 = SIGKILL (128+9)
         * 143 = SIGTERM (128+15)
         * 
         * Windows:
         * 0 = Success
         * Non-zero = Error (application-specific)
         * 
         * Checking:
         * int exit = process.exitValue();
         * if (exit == 0) { /* success */ }
         * else { /* failure */ }
         */
    }
    
    public static void main(String[] args) throws IOException {
        gracefulTermination();
        forcefulTermination();
        supportsNormalTermination();
        timeoutAndDestroy();
        exitValueChecking();
    }
}
```

## 5. Monitoring Processes

### Watching Process State and Lifecycle

```java
/**
 * MONITORING PROCESSES
 * 
 * Watching process lifecycle and receiving notifications
 */

import java.io.*;
import java.util.concurrent.*;

public class MonitoringProcesses {
    
    /**
     * onExit() - CompletableFuture notification
     */
    public static void onExitNotification() throws IOException {
        System.out.println("=== onExit Notification ===");
        
        // Start process
        ProcessBuilder pb = new ProcessBuilder("sleep", "2");
        Process process = pb.start();
        
        // Get notification when process exits
        CompletableFuture<Process> onExit = process.onExit();
        
        System.out.println("Process started, PID: " + process.pid());
        
        onExit.thenAccept(p -> {
            System.out.println("Process exited!");
            System.out.println("Exit code: " + p.exitValue());
        });
        
        // Do other work while process runs
        System.out.println("Doing other work...");
        
        // Wait for process
        onExit.join();
        
        /**
         * ONEXIT():
         * 
         * Java 9 feature
         * 
         * Returns: CompletableFuture<Process>
         * 
         * Completes when:
         * - Process terminates (normally or forcefully)
         * 
         * Benefits:
         * - Asynchronous notification
         * - No blocking
         * - Composable with other futures
         * - Can chain operations
         * 
         * Use for:
         * - Non-blocking process monitoring
         * - Cleanup after process
         * - Process completion callbacks
         */
    }
    
    /**
     * Monitoring multiple processes
     */
    public static void monitorMultipleProcesses() throws IOException {
        System.out.println("\n=== Monitor Multiple Processes ===");
        
        // Start multiple processes
        ProcessBuilder pb = new ProcessBuilder("sleep", "2");
        Process p1 = pb.start();
        Process p2 = pb.start();
        Process p3 = pb.start();
        
        System.out.println("Started 3 processes");
        
        // Get futures for all
        CompletableFuture<Process> f1 = p1.onExit();
        CompletableFuture<Process> f2 = p2.onExit();
        CompletableFuture<Process> f3 = p3.onExit();
        
        // Wait for all to complete
        CompletableFuture.allOf(f1, f2, f3).thenRun(() -> {
            System.out.println("All processes completed");
        });
        
        // Or wait for first to complete
        CompletableFuture.anyOf(f1, f2, f3).thenAccept(p -> {
            System.out.println("First process completed: " + 
                ((Process)p).pid());
        });
        
        // Wait for completion
        CompletableFuture.allOf(f1, f2, f3).join();
        
        /**
         * MONITORING PATTERNS:
         * 
         * All processes:
         * CompletableFuture.allOf(futures...)
         *   .thenRun(() -> { /* all done */ })
         * 
         * Any process:
         * CompletableFuture.anyOf(futures...)
         *   .thenAccept(p -> { /* first done */ })
         * 
         * With timeout:
         * future.orTimeout(5, TimeUnit.SECONDS)
         *   .exceptionally(ex -> { /* timeout */ })
         */
    }
    
    /**
     * Process state polling
     */
    public static void processStatePolling() throws IOException {
        System.out.println("\n=== Process State Polling ===");
        
        ProcessBuilder pb = new ProcessBuilder("sleep", "5");
        Process process = pb.start();
        
        System.out.println("Monitoring process " + process.pid());
        
        // Poll every second
        ScheduledExecutorService executor = 
            Executors.newSingleThreadScheduledExecutor();
        
        executor.scheduleAtFixedRate(() -> {
            if (process.isAlive()) {
                System.out.println("Process still running...");
                
                // Get current info
                ProcessHandle.Info info = process.toHandle().info();
                info.totalCpuDuration().ifPresent(cpu ->
                    System.out.println("  CPU time: " + cpu));
            } else {
                System.out.println("Process terminated");
                executor.shutdown();
            }
        }, 0, 1, TimeUnit.SECONDS);
        
        // Wait for completion
        process.onExit().join();
        executor.shutdown();
        
        /**
         * POLLING PATTERN:
         * 
         * ScheduledExecutorService executor = ...
         * executor.scheduleAtFixedRate(() -> {
         *   if (process.isAlive()) {
         *     // Check status, collect metrics
         *   } else {
         *     // Cleanup
         *     executor.shutdown();
         *   }
         * }, initialDelay, period, unit);
         * 
         * Use for:
         * - Regular health checks
         * - Metrics collection
         * - Resource monitoring
         */
    }
    
    /**
     * Watching all processes
     */
    public static void watchAllProcesses() {
        System.out.println("\n=== Watch All Processes ===");
        
        // Snapshot current processes
        Set<Long> initialPids = ProcessHandle.allProcesses()
            .map(ProcessHandle::pid)
            .collect(Collectors.toSet());
        
        System.out.println("Initial processes: " + initialPids.size());
        
        // Poll for new processes
        ScheduledExecutorService executor = 
            Executors.newSingleThreadScheduledExecutor();
        
        executor.scheduleAtFixedRate(() -> {
            Set<Long> currentPids = ProcessHandle.allProcesses()
                .map(ProcessHandle::pid)
                .collect(Collectors.toSet());
            
            // Find new processes
            Set<Long> newPids = new HashSet<>(currentPids);
            newPids.removeAll(initialPids);
            
            if (!newPids.isEmpty()) {
                System.out.println("New processes: " + newPids);
            }
            
            // Find terminated processes
            Set<Long> terminatedPids = new HashSet<>(initialPids);
            terminatedPids.removeAll(currentPids);
            
            if (!terminatedPids.isEmpty()) {
                System.out.println("Terminated: " + terminatedPids);
            }
            
            // Update baseline
            initialPids.clear();
            initialPids.addAll(currentPids);
            
        }, 0, 2, TimeUnit.SECONDS);
        
        // Run for 10 seconds
        try {
            Thread.sleep(10000);
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        
        executor.shutdown();
    }
    
    /**
     * Process timeout with monitoring
     */
    public static void processTimeoutWithMonitoring() throws IOException {
        System.out.println("\n=== Process Timeout with Monitoring ===");
        
        ProcessBuilder pb = new ProcessBuilder("sleep", "60");
        Process process = pb.start();
        
        // Set timeout
        CompletableFuture<Process> onExit = process.onExit()
            .orTimeout(5, TimeUnit.SECONDS);
        
        onExit.whenComplete((p, ex) -> {
            if (ex != null) {
                System.out.println("Process timed out, destroying");
                process.destroyForcibly();
            } else {
                System.out.println("Process completed normally");
                System.out.println("Exit code: " + p.exitValue());
            }
        });
        
        try {
            onExit.join();
        } catch (CompletionException e) {
            // Timeout exception
            System.out.println("Caught: " + e.getCause());
        }
        
        /**
         * TIMEOUT PATTERN WITH COMPLETABLEFUTURE:
         * 
         * process.onExit()
         *   .orTimeout(duration, unit)
         *   .whenComplete((p, ex) -> {
         *     if (ex != null) {
         *       // Timeout - destroy process
         *       process.destroyForcibly();
         *     } else {
         *       // Completed normally
         *     }
         *   });
         * 
         * Benefits:
         * - Non-blocking
         * - Automatic cleanup
         * - Exception handling
         */
    }
    
    /**
     * Resource monitoring
     */
    public static void resourceMonitoring(ProcessHandle handle) {
        System.out.println("\n=== Resource Monitoring ===");
        
        ScheduledExecutorService executor = 
            Executors.newSingleThreadScheduledExecutor();
        
        executor.scheduleAtFixedRate(() -> {
            if (!handle.isAlive()) {
                executor.shutdown();
                return;
            }
            
            ProcessHandle.Info info = handle.info();
            
            System.out.println("=== Process " + handle.pid() + " ===");
            
            info.totalCpuDuration().ifPresent(cpu ->
                System.out.println("CPU time: " + cpu));
            
            info.startInstant().ifPresent(start -> {
                Duration uptime = Duration.between(start, Instant.now());
                System.out.println("Uptime: " + uptime);
            });
            
            long childCount = handle.children().count();
            System.out.println("Children: " + childCount);
            
        }, 0, 2, TimeUnit.SECONDS);
    }
    
    public static void main(String[] args) throws IOException {
        onExitNotification();
        monitorMultipleProcesses();
        processStatePolling();
        watchAllProcesses();
        processTimeoutWithMonitoring();
    }
}
```

**Key insights:**

- **ProcessHandle**: Platform-independent process management
- **Process info**: All fields are Optional (platform-dependent)
- **Pipelines**: Native process piping with startPipeline()
- **Destruction**: destroy() graceful, destroyForcibly() guaranteed
- **onExit()**: Non-blocking process completion via CompletableFuture

```
Before Java 9: Limited process control, no PID access
After Java 9:  Full process tree, rich metadata, lifecycle monitoring

From "start and hope" to "full control"
```
