# Pattern Matching for instanceof (Java 16)

## Overview

Java 16 finalized pattern matching for `instanceof`, a feature that simplifies the common pattern of testing an object's type and then casting it. Traditional `instanceof` required three steps: type test, explicit cast, and variable declaration. Pattern matching combines these into a single operation, introducing a pattern variable that's automatically cast and available within an appropriate scope. The compiler understands the flow of control to determine where the pattern variable is in scope, including within logical expressions. This feature improves code readability, eliminates redundant casts, and reduces the potential for ClassCastException errors. This guide covers how type testing and casting are unified, the scope rules for pattern variables, combining patterns with logical operators (&&, ||), nested instanceof checks, and the performance benefits both for developers and at runtime.

## 1. Type Test and Cast in One

### Eliminating Redundant Casting

```java
/**
 * TYPE TEST AND CAST
 * 
 * Pattern matching combines instanceof and cast
 */

public class TypeTestAndCast {
    
    /**
     * Traditional instanceof (before Java 16)
     */
    public static void traditionalInstanceof() {
        System.out.println("=== Traditional instanceof ===");
        
        Object obj = "Hello World";
        
        // Old way - three steps
        if (obj instanceof String) {           // 1. Type test
            String s = (String) obj;           // 2. Cast
            System.out.println("Length: " + s.length());  // 3. Use
        }
        
        /**
         * TRADITIONAL PATTERN:
         * 
         * 1. Type test:
         *    if (obj instanceof Type)
         * 
         * 2. Explicit cast:
         *    Type var = (Type) obj
         * 
         * 3. Use variable:
         *    var.method()
         * 
         * Problems:
         * - Redundant: test AND cast
         * - Verbose: 3 lines for simple task
         * - Error-prone: could cast to wrong type
         * - Repetitive: mention type twice
         */
    }
    
    /**
     * Pattern matching instanceof (Java 16+)
     */
    public static void patternMatchingInstanceof() {
        System.out.println("\n=== Pattern Matching instanceof ===");
        
        Object obj = "Hello World";
        
        // New way - one step!
        if (obj instanceof String s) {
            System.out.println("Length: " + s.length());
        }
        
        /**
         * PATTERN MATCHING:
         * 
         * Syntax:
         * if (obj instanceof Type variable)
         * 
         * Combines:
         * 1. Type test (instanceof)
         * 2. Cast (automatic)
         * 3. Variable binding (s)
         * 
         * Benefits:
         * ✓ Concise: 1 line instead of 3
         * ✓ Safe: no ClassCastException possible
         * ✓ Clear: intent obvious
         * ✓ Type only mentioned once
         */
    }
    
    /**
     * Side-by-side comparison
     */
    public static void comparison() {
        System.out.println("\n=== Comparison ===");
        
        Object obj = 42;
        
        // Traditional
        if (obj instanceof Integer) {
            Integer i = (Integer) obj;
            System.out.println("Traditional: " + (i * 2));
        }
        
        // Pattern matching
        if (obj instanceof Integer i) {
            System.out.println("Pattern: " + (i * 2));
        }
        
        /**
         * CODE REDUCTION:
         * 
         * Before:
         * if (obj instanceof String) {
         *   String s = (String) obj;
         *   System.out.println(s.toUpperCase());
         * }
         * 
         * After:
         * if (obj instanceof String s) {
         *   System.out.println(s.toUpperCase());
         * }
         * 
         * From 3 lines → 2 lines
         * From 2 mentions of String → 1 mention
         */
    }
    
    /**
     * Multiple type checks
     */
    public static void multipleTypes() {
        System.out.println("\n=== Multiple Types ===");
        
        Object obj = "Hello";
        
        // Traditional - verbose
        if (obj instanceof String) {
            String s = (String) obj;
            System.out.println("String: " + s.length());
        } else if (obj instanceof Integer) {
            Integer i = (Integer) obj;
            System.out.println("Integer: " + (i * 2));
        } else if (obj instanceof Double) {
            Double d = (Double) obj;
            System.out.println("Double: " + (d / 2));
        } else {
            System.out.println("Other type");
        }
        
        // Pattern matching - cleaner
        if (obj instanceof String s) {
            System.out.println("String: " + s.length());
        } else if (obj instanceof Integer i) {
            System.out.println("Integer: " + (i * 2));
        } else if (obj instanceof Double d) {
            System.out.println("Double: " + (d / 2));
        } else {
            System.out.println("Other type");
        }
        
        /**
         * MULTIPLE CHECKS:
         * 
         * Pattern matching shines with multiple types:
         * - Each branch has own variable
         * - No repeated casting
         * - Much cleaner code
         * 
         * Note:
         * For many types, consider switch expression
         * with pattern matching (Java 17+)
         */
    }
    
    /**
     * Null handling
     */
    public static void nullHandling() {
        System.out.println("\n=== Null Handling ===");
        
        Object obj = null;
        
        // instanceof returns false for null
        if (obj instanceof String s) {
            System.out.println("String: " + s);
        } else {
            System.out.println("Not a String (or null)");
        }
        
        /**
         * NULL BEHAVIOR:
         * 
         * instanceof returns false for null:
         * null instanceof String → false
         * 
         * Pattern matching same:
         * if (obj instanceof String s)  // false if null
         * 
         * No NullPointerException
         * No special null check needed
         * 
         * This is safe:
         * if (obj instanceof String s) {
         *   s.length()  // s cannot be null here
         * }
         */
    }
    
    /**
     * Pattern variable cannot shadow
     */
    public static void noShadowing() {
        /**
         * SHADOWING PREVENTION:
         * 
         * Pattern variable cannot shadow existing variable:
         * 
         * String s = "existing";
         * if (obj instanceof String s) {  // COMPILATION ERROR!
         *   // Cannot declare s, already exists
         * }
         * 
         * Error: "variable s is already defined"
         * 
         * Must use different name:
         * if (obj instanceof String str) {  // OK
         *   System.out.println(str);
         * }
         * 
         * This prevents confusion and errors
         */
    }
    
    /**
     * Real-world example
     */
    public static void realWorldExample() {
        System.out.println("\n=== Real World Example ===");
        
        // Process different shapes
        Object shape = new Circle(5.0);
        
        double area;
        if (shape instanceof Circle c) {
            area = Math.PI * c.radius * c.radius;
        } else if (shape instanceof Rectangle r) {
            area = r.width * r.height;
        } else if (shape instanceof Triangle t) {
            area = 0.5 * t.base * t.height;
        } else {
            area = 0;
        }
        
        System.out.println("Area: " + area);
        
        /**
         * REAL USAGE:
         * 
         * Common in:
         * - Visitor pattern
         * - Type dispatch
         * - Polymorphic operations
         * - Heterogeneous collections
         * - Event handling
         * - Deserialization
         * 
         * Example:
         * void handle(Event event) {
         *   if (event instanceof ClickEvent e) {
         *     handleClick(e.x(), e.y());
         *   } else if (event instanceof KeyEvent e) {
         *     handleKey(e.keyCode());
         *   }
         * }
         */
    }
    
    // Helper classes
    static class Circle {
        final double radius;
        Circle(double radius) { this.radius = radius; }
    }
    
    static class Rectangle {
        final double width, height;
        Rectangle(double width, double height) {
            this.width = width;
            this.height = height;
        }
    }
    
    static class Triangle {
        final double base, height;
        Triangle(double base, double height) {
            this.base = base;
            this.height = height;
        }
    }
    
    public static void main(String[] args) {
        traditionalInstanceof();
        patternMatchingInstanceof();
        comparison();
        multipleTypes();
        nullHandling();
        realWorldExample();
    }
}
```

## 2. Scope of Pattern Variables

### Understanding Where Pattern Variables Are Available

```java
/**
 * PATTERN VARIABLE SCOPE
 * 
 * Flow-sensitive scoping of pattern variables
 */

public class PatternVariableScope {
    
    /**
     * Basic scope - if statement
     */
    public static void basicIfScope() {
        System.out.println("=== Basic If Scope ===");
        
        Object obj = "Hello";
        
        if (obj instanceof String s) {
            // s is in scope here
            System.out.println("Inside if: " + s.toUpperCase());
            System.out.println("Length: " + s.length());
        }
        
        // s is NOT in scope here
        // System.out.println(s);  // COMPILATION ERROR
        
        /**
         * BASIC SCOPE:
         * 
         * Pattern variable in scope when:
         * - Test is true
         * - Within if block
         * 
         * NOT in scope:
         * - After if block
         * - In else block
         * 
         * Compiler tracks where variable is safe to use
         */
    }
    
    /**
     * Scope in if-else
     */
    public static void ifElseScope() {
        System.out.println("\n=== If-Else Scope ===");
        
        Object obj = 42;
        
        if (obj instanceof String s) {
            // s in scope - definitely String
            System.out.println("String: " + s);
        } else {
            // s NOT in scope - not String
            // System.out.println(s);  // ERROR
            System.out.println("Not a String");
        }
        
        /**
         * IF-ELSE:
         * 
         * if (obj instanceof Type var) {
         *   // var in scope
         * } else {
         *   // var NOT in scope
         * }
         * 
         * Reason:
         * In else, instanceof was false
         * So object is NOT that type
         * Variable would be unsafe
         */
    }
    
    /**
     * Scope with negation (!)
     */
    public static void negationScope() {
        System.out.println("\n=== Negation Scope ===");
        
        Object obj = "Hello";
        
        // Negated pattern
        if (!(obj instanceof String s)) {
            // s NOT in scope - test was negated!
            System.out.println("Not a String");
        } else {
            // s IS in scope - must be String
            System.out.println("String: " + s.toUpperCase());
        }
        
        /**
         * NEGATION (!):
         * 
         * if (!(obj instanceof String s)) {
         *   // s NOT in scope (instanceof false)
         * } else {
         *   // s IN scope (instanceof true)
         * }
         * 
         * Scopes are reversed!
         * 
         * This is flow-sensitive scoping:
         * Compiler understands control flow
         */
    }
    
    /**
     * Scope with && (AND)
     */
    public static void andScope() {
        System.out.println("\n=== AND Scope ===");
        
        Object obj = "Hello";
        
        // Pattern in AND expression
        if (obj instanceof String s && s.length() > 3) {
            System.out.println("Long string: " + s);
        }
        
        /**
         * AND (&&) SCOPE:
         * 
         * if (obj instanceof String s && s.length() > 3)
         *                                 ↑
         *                    s in scope here!
         * 
         * Left-to-right evaluation:
         * 1. instanceof test
         * 2. If true, s in scope for rest of &&
         * 3. Can use s in right side
         * 
         * Short-circuit:
         * If instanceof false, right side not evaluated
         */
    }
    
    /**
     * Scope with || (OR)
     */
    public static void orScope() {
        System.out.println("\n=== OR Scope ===");
        
        Object obj = 42;
        
        // Pattern NOT in scope after ||
        if (obj instanceof String s || obj instanceof Integer) {
            // s NOT in scope!
            // Could be String OR Integer
            // System.out.println(s);  // ERROR
            System.out.println("String or Integer");
        }
        
        /**
         * OR (||) SCOPE:
         * 
         * if (obj instanceof String s || other_test)
         *                                 ↑
         *                    s NOT in scope after ||
         * 
         * Reason:
         * - If first test true, second not evaluated
         * - If first test false, s doesn't exist
         * - Either way, unsafe to use s after ||
         * 
         * Pattern variable only in scope when
         * test guarantees it's safe
         */
    }
    
    /**
     * Complex logical expressions
     */
    public static void complexLogic() {
        System.out.println("\n=== Complex Logic ===");
        
        Object obj = "Hello World";
        
        // Multiple conditions with AND
        if (obj instanceof String s && 
            s.length() > 5 && 
            s.contains("World")) {
            System.out.println("Long string with 'World': " + s);
        }
        
        // Nested conditions
        Object obj2 = 100;
        if (obj2 instanceof Integer i) {
            if (i > 50 && i < 150) {
                System.out.println("In range: " + i);
            }
        }
        
        /**
         * COMPLEX EXPRESSIONS:
         * 
         * Chaining with &&:
         * if (obj instanceof String s && 
         *     s.length() > 5 &&
         *     s.startsWith("A")) {
         *   // s available throughout
         * }
         * 
         * Mixing operators:
         * if ((obj instanceof String s && s.length() > 5) ||
         *     (obj instanceof Integer i && i > 0)) {
         *   // Neither s nor i in scope!
         *   // Could be either branch
         * }
         * 
         * Nested ifs:
         * if (obj instanceof String s) {
         *   if (s.length() > 5) {
         *     // s still in scope
         *   }
         * }
         */
    }
    
    /**
     * Ternary operator scope
     */
    public static void ternaryScope() {
        System.out.println("\n=== Ternary Scope ===");
        
        Object obj = "Hello";
        
        // Pattern in ternary
        String result = obj instanceof String s 
            ? s.toUpperCase()  // s in scope
            : "Not a string";  // s not in scope
        
        System.out.println("Result: " + result);
        
        /**
         * TERNARY OPERATOR:
         * 
         * obj instanceof String s ? s.toUpperCase() : "default"
         *                           ↑               ↑
         *                    in scope       not in scope
         * 
         * Pattern variable in scope:
         * - In true branch (after ?)
         * - NOT in false branch (after :)
         * 
         * Similar to if-else
         */
    }
    
    /**
     * Scope in while loop
     */
    public static void whileLoopScope() {
        System.out.println("\n=== While Loop Scope ===");
        
        Object[] objects = {"Hello", 42, "World"};
        int i = 0;
        
        while (i < objects.length && 
               objects[i] instanceof String s &&
               s.length() > 0) {
            System.out.println("String: " + s);
            i++;
        }
        
        /**
         * WHILE LOOP:
         * 
         * while (obj instanceof String s && s.length() > 0) {
         *   // s in scope in loop body
         * }
         * 
         * Re-evaluated each iteration:
         * - Pattern variable re-bound
         * - Scope renewed each time
         */
    }
    
    /**
     * Scope summary
     */
    public static void scopeSummary() {
        /**
         * SCOPE RULES SUMMARY:
         * 
         * IN SCOPE:
         * ✓ if (obj instanceof Type v) { v }
         * ✓ if (!(obj instanceof Type v)) { } else { v }
         * ✓ if (obj instanceof Type v && v.test()) { }
         * ✓ obj instanceof Type v ? v : other
         * 
         * NOT IN SCOPE:
         * ✗ if (obj instanceof Type v) { } // after block
         * ✗ if (obj instanceof Type v) { } else { v }
         * ✗ if (obj instanceof Type v || other) { v }
         * ✗ obj instanceof Type v ? other : v
         * 
         * FLOW-SENSITIVE:
         * Compiler analyzes control flow
         * Variable in scope only when safe
         * 
         * PRINCIPLE:
         * "Definite assignment when true"
         * Pattern variable exists when
         * control flow guarantees test passed
         */
    }
    
    public static void main(String[] args) {
        basicIfScope();
        ifElseScope();
        negationScope();
        andScope();
        orScope();
        complexLogic();
        ternaryScope();
        whileLoopScope();
    }
}
```

## 3. Combining with Logical Operators

### Patterns in Boolean Expressions

```java
/**
 * COMBINING WITH LOGICAL OPERATORS
 * 
 * Using pattern matching with &&, ||, !
 */

public class LogicalOperators {
    
    /**
     * AND operator (&&)
     */
    public static void andOperator() {
        System.out.println("=== AND Operator ===");
        
        Object obj = "Hello World";
        
        // Pattern variable available in right side
        if (obj instanceof String s && s.length() > 5) {
            System.out.println("Long string: " + s);
        }
        
        // Multiple conditions
        if (obj instanceof String s && 
            !s.isBlank() && 
            s.startsWith("H")) {
            System.out.println("Valid string: " + s);
        }
        
        /**
         * AND (&&) USAGE:
         * 
         * if (obj instanceof Type var && var.condition())
         *                                 ↑
         *                    var available here
         * 
         * Pattern:
         * 1. Test type
         * 2. If true, var in scope
         * 3. Test additional conditions on var
         * 
         * Short-circuit:
         * If instanceof false, right side skipped
         * 
         * Common pattern:
         * if (obj instanceof String s && s.length() > 0) {
         *   // Process non-empty string
         * }
         */
    }
    
    /**
     * Null-safe operations with &&
     */
    public static void nullSafeOperations() {
        System.out.println("\n=== Null-Safe Operations ===");
        
        Object obj = "Hello";
        
        // Traditional null check + type check
        if (obj != null && obj instanceof String s) {
            System.out.println("Non-null string: " + s);
        }
        
        // But instanceof already handles null!
        if (obj instanceof String s) {  // Simpler!
            System.out.println("String: " + s);
        }
        
        /**
         * NULL SAFETY:
         * 
         * instanceof returns false for null:
         * null instanceof String → false
         * 
         * So this is redundant:
         * if (obj != null && obj instanceof String s)
         *     ↑
         *   unnecessary
         * 
         * Just use:
         * if (obj instanceof String s)
         * 
         * Cleaner and safer!
         */
    }
    
    /**
     * OR operator (||)
     */
    public static void orOperator() {
        System.out.println("\n=== OR Operator ===");
        
        Object obj = 42;
        
        // Pattern variable NOT available after ||
        if (obj instanceof String s || obj instanceof Integer) {
            // s NOT in scope - could be String OR Integer
            System.out.println("String or Integer, but can't use s");
        }
        
        // Separate checks
        String result;
        if (obj instanceof String s) {
            result = "String: " + s;
        } else if (obj instanceof Integer i) {
            result = "Integer: " + i;
        } else {
            result = "Other";
        }
        System.out.println(result);
        
        /**
         * OR (||) LIMITATION:
         * 
         * if (obj instanceof String s || other_condition)
         *                                 ↑
         *                    s NOT in scope
         * 
         * Reason:
         * - First test could be false
         * - Second test could be true
         * - Can't guarantee s exists
         * 
         * Solution:
         * Use separate if-else for different types
         */
    }
    
    /**
     * NOT operator (!)
     */
    public static void notOperator() {
        System.out.println("\n=== NOT Operator ===");
        
        Object obj = "Hello";
        
        // Negated pattern
        if (!(obj instanceof String s)) {
            System.out.println("Not a String");
            // s NOT in scope here
        } else {
            System.out.println("String: " + s);
            // s IS in scope here
        }
        
        /**
         * NOT (!) OPERATOR:
         * 
         * if (!(obj instanceof String s)) {
         *   // s NOT in scope (test was false)
         * } else {
         *   // s IN scope (test was true)
         * }
         * 
         * Inverted logic:
         * Scopes are reversed
         * 
         * Use for:
         * - Early returns
         * - Guard clauses
         * - Validation
         */
    }
    
    /**
     * Guard clauses
     */
    public static void guardClauses() {
        System.out.println("\n=== Guard Clauses ===");
        
        processObject("Hello World");
        processObject(42);
        processObject(null);
    }
    
    private static void processObject(Object obj) {
        // Guard clause with negation
        if (!(obj instanceof String s)) {
            System.out.println("Not a string, skipping");
            return;
        }
        
        // s is in scope for rest of method
        System.out.println("Processing string: " + s.toUpperCase());
        System.out.println("Length: " + s.length());
        
        /**
         * GUARD CLAUSE PATTERN:
         * 
         * void method(Object obj) {
         *   if (!(obj instanceof Type var)) {
         *     return;  // Early exit
         *   }
         *   
         *   // var in scope for rest of method
         *   var.method1();
         *   var.method2();
         * }
         * 
         * Benefits:
         * - Reduces nesting
         * - Clear validation
         * - Fail fast
         * - Cleaner code
         */
    }
    
    /**
     * Complex conditions
     */
    public static void complexConditions() {
        System.out.println("\n=== Complex Conditions ===");
        
        Object obj = "Hello World";
        
        // Multiple type checks with OR
        if (obj instanceof String || 
            obj instanceof Integer || 
            obj instanceof Double) {
            System.out.println("Number or String");
        }
        
        // Type check with conditions
        if (obj instanceof String s && 
            s.length() > 5 && 
            s.length() < 100 &&
            !s.isBlank()) {
            System.out.println("Valid string: " + s);
        }
        
        // Type check with negations
        if (obj instanceof String s && 
            !s.isEmpty() && 
            !s.startsWith(" ")) {
            System.out.println("Good string: " + s);
        }
        
        /**
         * COMPLEX PATTERNS:
         * 
         * Chaining conditions:
         * if (obj instanceof String s &&
         *     s.length() >= MIN &&
         *     s.length() <= MAX &&
         *     s.matches(PATTERN)) {
         *   // Process valid string
         * }
         * 
         * Multiple types (OR):
         * if (obj instanceof String s) {
         *   handleString(s);
         * } else if (obj instanceof Integer i) {
         *   handleInteger(i);
         * } else if (obj instanceof Double d) {
         *   handleDouble(d);
         * }
         * 
         * Don't use OR for different pattern variables:
         * if (obj instanceof String s || 
         *     obj instanceof Integer i) {
         *   // Neither s nor i in scope!
         * }
         */
    }
    
    /**
     * Combining patterns with method calls
     */
    public static void methodCalls() {
        System.out.println("\n=== Method Calls ===");
        
        Object obj = "Hello";
        
        // Pattern with method predicate
        if (obj instanceof String s && isValid(s)) {
            System.out.println("Valid string: " + s);
        }
        
        // Pattern with multiple methods
        if (obj instanceof String s && 
            isValid(s) && 
            isAllowed(s)) {
            System.out.println("Allowed string: " + s);
        }
        
        /**
         * METHOD INTEGRATION:
         * 
         * if (obj instanceof String s && validate(s)) {
         *                                 ↑
         *                    s passed to method
         * 
         * Pattern:
         * 1. Check type
         * 2. Pass to validation method
         * 3. Use if valid
         * 
         * Benefits:
         * - Separation of concerns
         * - Reusable validation
         * - Clean code
         */
    }
    
    private static boolean isValid(String s) {
        return s.length() > 0 && !s.isBlank();
    }
    
    private static boolean isAllowed(String s) {
        return !s.contains("forbidden");
    }
    
    public static void main(String[] args) {
        andOperator();
        nullSafeOperations();
        orOperator();
        notOperator();
        guardClauses();
        complexConditions();
        methodCalls();
    }
}
```

## 4. Nested Patterns

### Layered instanceof Checks

```java
/**
 * NESTED PATTERNS
 * 
 * Checking types within types
 */

import java.util.*;

public class NestedPatterns {
    
    /**
     * Basic nesting
     */
    public static void basicNesting() {
        System.out.println("=== Basic Nesting ===");
        
        Object obj = Optional.of("Hello");
        
        // Nested type checks
        if (obj instanceof Optional opt) {
            if (opt.isPresent()) {
                Object value = opt.get();
                if (value instanceof String s) {
                    System.out.println("String in Optional: " + s);
                }
            }
        }
        
        /**
         * BASIC NESTING:
         * 
         * Check outer type, then inner:
         * if (obj instanceof Container c) {
         *   if (c.getValue() instanceof Type t) {
         *     // Use t
         *   }
         * }
         * 
         * Common with:
         * - Optionals
         * - Collections
         * - Wrappers
         * - Nested structures
         */
    }
    
    /**
     * Combined with logical operators
     */
    public static void combinedWithLogical() {
        System.out.println("\n=== Combined with Logical ===");
        
        Object obj = Optional.of("Hello World");
        
        // Flatten nesting with &&
        if (obj instanceof Optional<?> opt && 
            opt.isPresent() &&
            opt.get() instanceof String s) {
            System.out.println("String value: " + s.toUpperCase());
        }
        
        /**
```

## Summary

### Quick Reference

**Basic Pattern Matching:**

```java
// Before
if (obj instanceof String) {
    String s = (String) obj;
    System.out.println(s.toUpperCase());
}

// After
if (obj instanceof String s) {
    System.out.println(s.toUpperCase());
}
```

**Scope Rules:**

```java
// In scope
if (obj instanceof String s) {
    s.length();  // ✓ s in scope
}

// Not in scope
if (obj instanceof String s) {
}
s.length();  // ✗ s not in scope

// With &&
if (obj instanceof String s && s.length() > 5) {
    // ✓ s in scope
}

// With ||
if (obj instanceof String s || other) {
    // ✗ s not in scope
}

// With !
if (!(obj instanceof String s)) {
    // ✗ s not in scope
} else {
    // ✓ s in scope
}
```

**Guard Clauses:**

```java
void process(Object obj) {
    if (!(obj instanceof String s)) {
        return;  // Early exit
    }
    
    // s in scope for rest of method
    System.out.println(s.toUpperCase());
}
```

**Multiple Types:**

```java
if (obj instanceof String s) {
    process(s);

    process(i);

    process(d);
}
```
