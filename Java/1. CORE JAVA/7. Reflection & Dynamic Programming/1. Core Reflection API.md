# Core Reflection API

## Overview

Java Reflection allows programs to inspect and manipulate classes, methods, fields, and constructors at runtime. While powerful for frameworks and libraries, reflection has performance overhead and security implications. This guide covers Class.forName(), member access, private member manipulation, method invocation, dynamic instantiation, generic type inspection, and performance considerations essential for using reflection effectively.

## 1. Class.forName() and Classloading

### Dynamic Class Loading

```java
/**
 * CLASS.FORNAME() AND CLASSLOADING
 * 
 * Loading classes dynamically at runtime
 */

import java.lang.reflect.*;

public class ClassForNameAndLoading {
    
    /**
     * Basic Class.forName()
     */
    public static void basicClassForName() throws Exception {
        // Load class by name
        Class<?> stringClass = Class.forName("java.lang.String");
        System.out.println("Loaded: " + stringClass.getName());
        
        // Get Class object
        Class<?> integerClass = Integer.class;
        System.out.println("Class literal: " + integerClass.getName());
        
        // Get Class from instance
        String str = "Hello";
        Class<?> instanceClass = str.getClass();
        System.out.println("From instance: " + instanceClass.getName());
        
        /**
         * THREE WAYS TO GET CLASS:
         * 
         * 1. Class.forName("ClassName")
         *    - Load by string name
         *    - Can throw ClassNotFoundException
         *    - Initializes static blocks
         * 
         * 2. ClassName.class
         *    - Class literal
         *    - Compile-time safe
         *    - No initialization
         * 
         * 3. object.getClass()
         *    - From instance
         *    - Runtime type
         *    - Already initialized
         */
    }
    
    /**
     * Class.forName() initialization
     */
    static class InitializationTest {
        static {
            System.out.println("Static block executed");
        }
        
        static int value = initValue();
        
        static int initValue() {
            System.out.println("Static initializer called");
            return 42;
        }
    }
    
    public static void classForNameInitialization() throws Exception {
        System.out.println("=== Using Class.forName() ===");
        Class<?> clazz1 = Class.forName(
            "ClassForNameAndLoading$InitializationTest");
        
        System.out.println("\n=== Using class literal ===");
        Class<?> clazz2 = InitializationTest.class;
        
        /**
         * OUTPUT:
         * === Using Class.forName() ===
         * Static block executed
         * Static initializer called
         * 
         * === Using class literal ===
         * (no output - already initialized)
         * 
         * 
         * CLASS.FORNAME() BEHAVIOR:
         * 
         * Class.forName(name):
         * - Loads class
         * - Initializes static blocks
         * - Runs static initializers
         * 
         * Class.forName(name, initialize, loader):
         * - Control initialization
         * - specify ClassLoader
         * - initialize=false skips static blocks
         */
    }
    
    /**
     * Class.forName() with initialization control
     */
    public static void controlledInitialization() throws Exception {
        ClassLoader loader = ClassLoader.getSystemClassLoader();
        
        // Load without initialization
        Class<?> clazz = Class.forName(
            "ClassForNameAndLoading$InitializationTest",
            false,  // don't initialize
            loader
        );
        
        System.out.println("Loaded class: " + clazz.getName());
        System.out.println("Not initialized yet");
        
        // Force initialization
        clazz.getDeclaredConstructor().newInstance();
        System.out.println("Now initialized");
        
        /**
         * CONTROLLED INITIALIZATION:
         * 
         * Class.forName(name, initialize, loader)
         * 
         * initialize=false:
         * - Loads class definition
         * - Doesn't run static blocks
         * - Lazy initialization
         * 
         * initialize=true:
         * - Runs static blocks
         * - Same as Class.forName(name)
         */
    }
    
    /**
     * Primitive and array types
     */
    public static void primitiveAndArrayTypes() throws Exception {
        // Primitive types
        Class<?> intClass = int.class;
        System.out.println("int.class: " + intClass.getName());
        
        // Can't use Class.forName() for primitives directly
        // Class<?> intClass2 = Class.forName("int");  // ClassNotFoundException
        
        // But can for wrappers
        Class<?> integerClass = Class.forName("java.lang.Integer");
        
        // Array types
        Class<?> stringArray = String[].class;
        System.out.println("String[].class: " + stringArray.getName());
        
        // Array with Class.forName()
        Class<?> intArray = Class.forName("[I");  // int[]
        System.out.println("int[]: " + intArray.getName());
        
        Class<?> stringArray2 = Class.forName("[Ljava.lang.String;");
        System.out.println("String[]: " + stringArray2.getName());
        
        /**
         * ARRAY TYPE NAMES:
         * 
         * [I - int[]
         * [D - double[]
         * [J - long[]
         * [Z - boolean[]
         * [C - char[]
         * [B - byte[]
         * [S - short[]
         * [F - float[]
         * [Lpackage.ClassName; - ClassName[]
         * 
         * [[I - int[][]
         * [[[Ljava.lang.String; - String[][][]
         */
    }
    
    /**
     * Class information
     */
    static class Parent {}
    static class Child extends Parent implements Serializable {}
    
    public static void classInformation() {
        Class<?> clazz = Child.class;
        
        System.out.println("=== Class Information ===");
        
        // Basic info
        System.out.println("Name: " + clazz.getName());
        System.out.println("Simple name: " + clazz.getSimpleName());
        System.out.println("Canonical name: " + clazz.getCanonicalName());
        
        // Package
        System.out.println("Package: " + clazz.getPackage());
        
        // Modifiers
        int modifiers = clazz.getModifiers();
        System.out.println("Modifiers: " + Modifier.toString(modifiers));
        System.out.println("Is public: " + Modifier.isPublic(modifiers));
        System.out.println("Is static: " + Modifier.isStatic(modifiers));
        
        // Type checks
        System.out.println("\n=== Type Checks ===");
        System.out.println("Is interface: " + clazz.isInterface());
        System.out.println("Is enum: " + clazz.isEnum());
        System.out.println("Is array: " + clazz.isArray());
        System.out.println("Is primitive: " + clazz.isPrimitive());
        System.out.println("Is annotation: " + clazz.isAnnotation());
        
        // Hierarchy
        System.out.println("\n=== Hierarchy ===");
        System.out.println("Superclass: " + clazz.getSuperclass());
        System.out.println("Interfaces: ");
        for (Class<?> iface : clazz.getInterfaces()) {
            System.out.println("  " + iface.getName());
        }
        
        /**
         * CLASS INFORMATION METHODS:
         * 
         * getName() - fully qualified name
         * getSimpleName() - class name only
         * getCanonicalName() - canonical name
         * 
         * getModifiers() - int bitmask
         * Modifier.toString() - string representation
         * Modifier.isPublic/isStatic/etc.
         * 
         * isInterface/isEnum/isArray/isPrimitive
         * 
         * getSuperclass() - parent class
         * getInterfaces() - direct interfaces
         */
    }
    
    /**
     * Class loading and ClassLoader
     */
    public static void classLoadingAndClassLoader() {
        Class<?> clazz = String.class;
        
        System.out.println("=== ClassLoader Hierarchy ===");
        
        ClassLoader loader = clazz.getClassLoader();
        System.out.println("String loader: " + loader);  // null = bootstrap
        
        ClassLoader myLoader = ClassForNameAndLoading.class.getClassLoader();
        System.out.println("My loader: " + myLoader);
        
        // Loader hierarchy
        ClassLoader current = myLoader;
        while (current != null) {
            System.out.println("  " + current);
            current = current.getParent();
        }
        System.out.println("  Bootstrap ClassLoader (null)");
        
        /**
         * CLASSLOADER HIERARCHY:
         * 
         * Bootstrap ClassLoader (null)
         * └─ Platform/Extension ClassLoader
         *    └─ Application/System ClassLoader
         *       └─ Custom ClassLoaders
         * 
         * Bootstrap:
         * - Loads JDK classes
         * - java.lang.*, java.util.*, etc.
         * - Native implementation
         * 
         * Platform (Java 9+):
         * - Loads platform classes
         * - java.sql.*, javax.*, etc.
         * 
         * Application:
         * - Loads classpath classes
         * - Your application code
         * 
         * Custom:
         * - Web containers
         * - Plugin systems
         * - Hot reload
         */
    }
    
    /**
     * Custom ClassLoader
     */
    static class SimpleClassLoader extends ClassLoader {
        @Override
        protected Class<?> findClass(String name) throws ClassNotFoundException {
            // Load class bytes (from file, network, etc.)
            // This is simplified - real implementation loads actual bytes
            throw new ClassNotFoundException(name);
        }
        
        /**
         * CUSTOM CLASSLOADER:
         * 
         * Override findClass(name)
         * Load class bytes
         * Call defineClass()
         * 
         * Use cases:
         * - Hot reload
         * - Plugin systems
         * - Bytecode generation
         * - Security isolation
         * - OSGi containers
         */
    }
    
    /**
     * Class loading best practices
     */
    public static void bestPractices() {
        /**
         * BEST PRACTICES:
         * 
         * 1. CACHE CLASS OBJECTS:
         *    Class.forName() is expensive
         *    Cache and reuse
         * 
         * 2. HANDLE ClassNotFoundException:
         *    Always in try-catch
         *    Provide meaningful error
         * 
         * 3. USE CLASS LITERALS WHEN POSSIBLE:
         *    String.class vs Class.forName("java.lang.String")
         *    Compile-time safe
         *    No exception
         * 
         * 4. SPECIFY CLASSLOADER:
         *    In complex environments
         *    Plugin systems
         *    Web containers
         * 
         * 5. AVOID INITIALIZATION SIDE EFFECTS:
         *    Static blocks should be safe
         *    No dependencies on runtime state
         * 
         * 6. SECURITY:
         *    Untrusted class names
         *    Validate before loading
         *    SecurityManager checks
         */
    }
    
    public static void main(String[] args) throws Exception {
        System.out.println("=== Basic Class.forName() ===");
        basicClassForName();
        
        System.out.println("\n=== Class Information ===");
        classInformation();
    }
}
```

## 2. Getting Fields, Methods, Constructors

### Inspecting Class Members

```java
/**
 * GETTING FIELDS, METHODS, CONSTRUCTORS
 * 
 * Accessing class members via reflection
 */

import java.lang.reflect.*;
import java.util.*;

public class GettingMembers {
    
    /**
     * Example class for reflection
     */
    static class Person {
        public String name;
        private int age;
        protected String address;
        String phone;  // package-private
        
        public static final String SPECIES = "Homo sapiens";
        
        public Person() {}
        
        public Person(String name) {
            this.name = name;
        }
        
        private Person(String name, int age) {
            this.name = name;
            this.age = age;
        }
        
        public String getName() { return name; }
        public void setName(String name) { this.name = name; }
        
        private int getAge() { return age; }
        private void setAge(int age) { this.age = age; }
        
        public static void staticMethod() {
            System.out.println("Static method");
        }
    }
    
    /**
     * Getting fields
     */
    public static void gettingFields() {
        Class<?> clazz = Person.class;
        
        System.out.println("=== Public Fields ===");
        Field[] publicFields = clazz.getFields();
        for (Field field : publicFields) {
            System.out.println(field.getName() + " - " + field.getType());
        }
        
        System.out.println("\n=== All Declared Fields ===");
        Field[] allFields = clazz.getDeclaredFields();
        for (Field field : allFields) {
            System.out.println(field.getName() + " - " + 
                Modifier.toString(field.getModifiers()));
        }
        
        System.out.println("\n=== Specific Field ===");
        try {
            Field nameField = clazz.getField("name");
            System.out.println("Found: " + nameField.getName());
            
            Field ageField = clazz.getDeclaredField("age");
            System.out.println("Found: " + ageField.getName());
        } catch (NoSuchFieldException e) {
            e.printStackTrace();
        }
        
        /**
         * FIELD METHODS:
         * 
         * getFields():
         * - Public fields only
         * - Includes inherited
         * - Returns Field[]
         * 
         * getDeclaredFields():
         * - All fields (public/private/protected/package)
         * - This class only (no inherited)
         * - Returns Field[]
         * 
         * getField(name):
         * - Single public field
         * - Throws NoSuchFieldException
         * 
         * getDeclaredField(name):
         * - Single field (any visibility)
         * - This class only
         * - Throws NoSuchFieldException
         */
    }
    
    /**
     * Getting methods
     */
    public static void gettingMethods() {
        Class<?> clazz = Person.class;
        
        System.out.println("=== Public Methods ===");
        Method[] publicMethods = clazz.getMethods();
        for (Method method : publicMethods) {
            if (method.getDeclaringClass() == Person.class) {
                System.out.println(method.getName() + " - " + 
                    Arrays.toString(method.getParameterTypes()));
            }
        }
        
        System.out.println("\n=== All Declared Methods ===");
        Method[] allMethods = clazz.getDeclaredMethods();
        for (Method method : allMethods) {
            System.out.println(method.getName() + " - " + 
                Modifier.toString(method.getModifiers()));
        }
        
        System.out.println("\n=== Specific Method ===");
        try {
            Method getter = clazz.getMethod("getName");
            System.out.println("Found: " + getter.getName());
            
            Method setter = clazz.getMethod("setName", String.class);
            System.out.println("Found: " + setter.getName() + 
                " with params: " + Arrays.toString(setter.getParameterTypes()));
            
            Method privateMethod = clazz.getDeclaredMethod("getAge");
            System.out.println("Found private: " + privateMethod.getName());
        } catch (NoSuchMethodException e) {
            e.printStackTrace();
        }
        
        /**
         * METHOD METHODS:
         * 
         * getMethods():
         * - Public methods
         * - Includes inherited
         * - Returns Method[]
         * 
         * getDeclaredMethods():
         * - All methods (any visibility)
         * - This class only
         * - Returns Method[]
         * 
         * getMethod(name, paramTypes...):
         * - Single public method
         * - Match by name and parameters
         * - Throws NoSuchMethodException
         * 
         * getDeclaredMethod(name, paramTypes...):
         * - Single method (any visibility)
         * - This class only
         * - Throws NoSuchMethodException
         */
    }
    
    /**
     * Getting constructors
     */
    public static void gettingConstructors() {
        Class<?> clazz = Person.class;
        
        System.out.println("=== Public Constructors ===");
        Constructor<?>[] publicCtors = clazz.getConstructors();
        for (Constructor<?> ctor : publicCtors) {
            System.out.println("Constructor with params: " + 
                Arrays.toString(ctor.getParameterTypes()));
        }
        
        System.out.println("\n=== All Declared Constructors ===");
        Constructor<?>[] allCtors = clazz.getDeclaredConstructors();
        for (Constructor<?> ctor : allCtors) {
            System.out.println(Modifier.toString(ctor.getModifiers()) + 
                " constructor with params: " + 
                Arrays.toString(ctor.getParameterTypes()));
        }
        
        System.out.println("\n=== Specific Constructor ===");
        try {
            Constructor<?> defaultCtor = clazz.getConstructor();
            System.out.println("Found: default constructor");
            
            Constructor<?> stringCtor = clazz.getConstructor(String.class);
            System.out.println("Found: constructor(String)");
            
            Constructor<?> privateCtor = clazz.getDeclaredConstructor(
                String.class, int.class);
            System.out.println("Found private: constructor(String, int)");
        } catch (NoSuchMethodException e) {
            e.printStackTrace();
        }
        
        /**
         * CONSTRUCTOR METHODS:
         * 
         * getConstructors():
         * - Public constructors
         * - Returns Constructor<?>[]
         * 
         * getDeclaredConstructors():
         * - All constructors (any visibility)
         * - Returns Constructor<?>[]
         * 
         * getConstructor(paramTypes...):
         * - Single public constructor
         * - Match by parameters
         * 
         * getDeclaredConstructor(paramTypes...):
         * - Single constructor (any visibility)
         */
    }
    
    /**
     * Member details
     */
    public static void memberDetails() throws Exception {
        Class<?> clazz = Person.class;
        
        // Field details
        Field field = clazz.getDeclaredField("age");
        System.out.println("=== Field Details ===");
        System.out.println("Name: " + field.getName());
        System.out.println("Type: " + field.getType());
        System.out.println("Generic type: " + field.getGenericType());
        System.out.println("Modifiers: " + Modifier.toString(field.getModifiers()));
        System.out.println("Declaring class: " + field.getDeclaringClass());
        
        // Method details
        Method method = clazz.getDeclaredMethod("setAge", int.class);
        System.out.println("\n=== Method Details ===");
        System.out.println("Name: " + method.getName());
        System.out.println("Return type: " + method.getReturnType());
        System.out.println("Parameter types: " + 
            Arrays.toString(method.getParameterTypes()));
        System.out.println("Generic return: " + method.getGenericReturnType());
        System.out.println("Exception types: " + 
            Arrays.toString(method.getExceptionTypes()));
        System.out.println("Modifiers: " + 
            Modifier.toString(method.getModifiers()));
        
        // Constructor details
        Constructor<?> ctor = clazz.getDeclaredConstructor(String.class, int.class);
        System.out.println("\n=== Constructor Details ===");
        System.out.println("Parameter types: " + 
            Arrays.toString(ctor.getParameterTypes()));
        System.out.println("Modifiers: " + 
            Modifier.toString(ctor.getModifiers()));
        
        /**
         * MEMBER INFORMATION:
         * 
         * All members:
         * - getName()
         * - getModifiers()
         * - getDeclaringClass()
         * 
         * Field:
         * - getType()
         * - getGenericType()
         * 
         * Method:
         * - getReturnType()
         * - getParameterTypes()
         * - getGenericReturnType()
         * - getExceptionTypes()
         * 
         * Constructor:
         * - getParameterTypes()
         * - getExceptionTypes()
         */
    }
    
    /**
     * Finding methods by pattern
     */
    public static void findMethodsByPattern() {
        Class<?> clazz = Person.class;
        
        System.out.println("=== Getters ===");
        for (Method method : clazz.getDeclaredMethods()) {
            if (method.getName().startsWith("get") && 
                method.getParameterCount() == 0 &&
                method.getReturnType() != void.class) {
                System.out.println(method.getName());
            }
        }
        
        System.out.println("\n=== Setters ===");
        for (Method method : clazz.getDeclaredMethods()) {
            if (method.getName().startsWith("set") && 
                method.getParameterCount() == 1 &&
                method.getReturnType() == void.class) {
                System.out.println(method.getName());
            }
        }
        
        System.out.println("\n=== Static Methods ===");
        for (Method method : clazz.getDeclaredMethods()) {
            if (Modifier.isStatic(method.getModifiers())) {
                System.out.println(method.getName());
            }
        }
        
        /**
         * FINDING PATTERNS:
         * 
         * Common patterns:
         * - Getters: get*, 0 params, non-void
         * - Setters: set*, 1 param, void
         * - Boolean getters: is*, 0 params, boolean
         * - Static methods: Modifier.isStatic()
         * - Public methods: Modifier.isPublic()
         */
    }
    
    /**
     * Best practices
     */
    public static void bestPractices() {
        /**
         * MEMBER ACCESS BEST PRACTICES:
         * 
         * 1. CACHE REFLECTION OBJECTS:
         *    Field, Method, Constructor objects
         *    Lookup is expensive
         *    Reuse across calls
         * 
         * 2. HANDLE NoSuchMethodException:
         *    Method might not exist
         *    Graceful degradation
         * 
         * 3. CHECK PARAMETER TYPES:
         *    Method overloading
         *    Exact match needed
         * 
         * 4. USE getDeclared* FOR PRIVATE:
         *    Private members need getDeclared*
         *    get* only returns public
         * 
         * 5. CONSIDER INHERITANCE:
         *    get* includes inherited
         *    getDeclared* only this class
         * 
         * 6. PERFORMANCE:
         *    Reflection is slow
         *    Cache and optimize
         *    Consider alternatives
         */
    }
    
    public static void main(String[] args) throws Exception {
        System.out.println("=== Getting Fields ===");
        gettingFields();
        
        System.out.println("\n=== Getting Methods ===");
        gettingMethods();
        
        System.out.println("\n=== Getting Constructors ===");
        gettingConstructors();
    }
}
```

## 3. Accessing Private Members (setAccessible)

### Breaking Encapsulation

```java
/**
 * ACCESSING PRIVATE MEMBERS
 * 
 * Using setAccessible to access private members
 */

import java.lang.reflect.*;

public class AccessingPrivateMembers {
    
    /**
     * Example class with private members
     */
    static class SecretData {
        private String secret = "confidential";
        private int code = 12345;
        
        private SecretData() {
            System.out.println("Private constructor called");
        }
        
        private String getSecret() {
            return "The secret is: " + secret;
        }
        
        private void setSecret(String secret) {
            System.out.println("Setting secret");
            this.secret = secret;
        }
    }
    
    /**
     * Accessing private fields
     */
    public static void accessingPrivateFields() throws Exception {
        SecretData obj = new SecretData();
        
        Class<?> clazz = obj.getClass();
        Field secretField = clazz.getDeclaredField("secret");
        
        System.out.println("=== Before setAccessible ===");
        try {
            String value = (String) secretField.get(obj);
            System.out.println("This won't print");
        } catch (IllegalAccessException e) {
            System.out.println("Cannot access private field: " + e.getMessage());
        }
        
        System.out.println("\n=== After setAccessible ===");
        secretField.setAccessible(true);
        String value = (String) secretField.get(obj);
        System.out.println("Secret value: " + value);
        
        // Modify private field
        secretField.set(obj, "new secret");
        System.out.println("Modified value: " + secretField.get(obj));
        
        /**
         * SETACCESSIBLE():
         * 
         * setAccessible(true):
         * - Suppresses Java access checking
         * - Allows access to private members
         * - Required for private/protected/package members
         * 
         * Security:
         * - Can be blocked by SecurityManager
         * - Breaks encapsulation
         * - Use with caution
         */
    }
    
    /**
     * Accessing private methods
     */
    public static void accessingPrivateMethods() throws Exception {
        SecretData obj = new SecretData();
        
        Class<?> clazz = obj.getClass();
        Method getSecret = clazz.getDeclaredMethod("getSecret");
        Method setSecret = clazz.getDeclaredMethod("setSecret", String.class);
        
        // Enable access
        getSecret.setAccessible(true);
        setSecret.setAccessible(true);
        
        // Call private getter
        String result = (String) getSecret.invoke(obj);
        System.out.println("Result: " + result);
        
        // Call private setter
        setSecret.invoke(obj, "modified secret");
        
        // Verify
        result = (String) getSecret.invoke(obj);
        System.out.println("After modification: " + result);
        
        /**
         * PRIVATE METHOD ACCESS:
         * 
         * 1. Get method with getDeclaredMethod()
         * 2. Call setAccessible(true)
         * 3. Invoke with invoke()
         * 
         * Common use case:
         * - Testing private methods
         * - Frameworks (DI, ORM)
         * - Serialization libraries
         */
    }
    
    /**
     * Accessing private constructors
     */
    public static void accessingPrivateConstructors() throws Exception {
        Class<?> clazz = SecretData.class;
        Constructor<?> ctor = clazz.getDeclaredConstructor();
        
        System.out.println("Constructor modifiers: " + 
            Modifier.toString(ctor.getModifiers()));
        
        // Can't use without setAccessible
        try {
            SecretData obj = (SecretData) ctor.newInstance();
        } catch (IllegalAccessException e) {
            System.out.println("Cannot access private constructor");
        }
        
        // Enable access
        ctor.setAccessible(true);
        SecretData obj = (SecretData) ctor.newInstance();
        System.out.println("Created instance via private constructor");
        
        /**
         * PRIVATE CONSTRUCTOR ACCESS:
         * 
         * Use cases:
         * - Singleton pattern testing
         * - Utility class instantiation
         * - Frameworks (Spring, Hibernate)
         * 
         * Singletons can be broken:
         * - Reflection bypasses getInstance()
         * - Can create multiple instances
         * - Enum singletons are safer
         */
    }
    
    /**
     * Final field modification
     */
    static class FinalFieldTest {
        private final String finalField = "original";
        private static final String STATIC_FINAL = "static original";
    }
    
    public static void modifyingFinalFields() throws Exception {
        FinalFieldTest obj = new FinalFieldTest();
        
        Class<?> clazz = obj.getClass();
        Field field = clazz.getDeclaredField("finalField");
        field.setAccessible(true);
        
        System.out.println("Original: " + field.get(obj));
        
        // Modify final field (works for instance fields)
        field.set(obj, "modified");
        System.out.println("Modified: " + field.get(obj));
        
        // Direct access still shows original (compiler optimization)
        System.out.println("Direct access: " + obj.finalField);
        
        /**

/**
```

```java
import java.lang.reflect.*;
import java.util.*;

public class MethodInvocation {
    
    /**
     * Example class for method invocation
     */
    static class Calculator {
        public int add(int a, int b) {
            return a + b;
        }
        
        public String concat(String a, String b) {
            return a + b;
        }
        
        public static int multiply(int a, int b) {
            return a * b;
        }
        
        private int subtract(int a, int b) {
            return a - b;
        }
        
        public void voidMethod() {
            System.out.println("Void method called");
        }
        
        public int varargs(int... numbers) {
            return Arrays.stream(numbers).sum();
        }
    }
    
    /**
     * Basic method invocation
     */
    public static void basicMethodInvocation() throws Exception {
        Calculator calc = new Calculator();
        Class<?> clazz = calc.getClass();
        
        // Get method
        Method addMethod = clazz.getMethod("add", int.class, int.class);
        
        // Invoke method
        Object result = addMethod.invoke(calc, 5, 3);
        System.out.println("5 + 3 = " + result);
        
        // Another method
        Method concatMethod = clazz.getMethod("concat", String.class, String.class);
        result = concatMethod.invoke(calc, "Hello", "World");
        System.out.println("Concat: " + result);
        
        /**
         * METHOD.INVOKE():
         * 
         * invoke(instance, args...)
         * 
         * instance:
         * - Object to invoke on
         * - null for static methods
         * 
         * args:
         * - Method arguments
         * - Must match parameter types
         * - Primitive types boxed
         * 
         * Returns:
         * - Object (return value)
         * - null for void methods
         * - Primitives are boxed
         */
    }
    
    /**
     * Static method invocation
     */
    public static void staticMethodInvocation() throws Exception {
        Class<?> clazz = Calculator.class;
        
        Method multiplyMethod = clazz.getMethod("multiply", int.class, int.class);
        
        // Static method - pass null as instance
        Object result = multiplyMethod.invoke(null, 4, 5);
        System.out.println("4 * 5 = " + result);
        
        /**
         * STATIC METHODS:
         * 
         * invoke(null, args...)
         * 
         * Instance parameter is null
         * Method must be static
         * Otherwise IllegalArgumentException
         */
    }
    
    /**
     * Private method invocation
     */
    public static void privateMethodInvocation() throws Exception {

            int.class, int.class);

        subtractMethod.setAccessible(true);
        
        Object result = subtractMethod.invoke(calc, 10, 3);
        System.out.println("10 - 3 = " + result);
        
        /**
         * PRIVATE METHODS:
         * 
         * 1. getDeclaredMethod() to get private method
         * 2. setAccessible(true)
         * 3. invoke() as normal

    }

        System.out.println("Void method result: " + result);  // null

         * VOID METHODS:

        System.out.println("Sum: " + result);

         * VARARGS:
         * 
         * Varargs parameter is actually an array

         * Method signature: void method(int... args)
         * Reflection sees: void method(int[] args)

         * Must pass array, not individual values

    /**
     * Exception handling in invoke
     */
    static class ThrowingClass {
        public void throwsCheckedException() throws IOException {
            throw new IOException("Checked exception");
        }
        
        public void throwsRuntimeException() {
            throw new RuntimeException("Runtime exception");
        }
    }
    
    public static void exceptionHandlingInInvoke() throws Exception {
        ThrowingClass obj = new ThrowingClass();
        Class<?> clazz = obj.getClass();
        
        // Method that throws checked exception
        Method checkedMethod = clazz.getMethod("throwsCheckedException");
        
        try {
            checkedMethod.invoke(obj);
        } catch (InvocationTargetException e) {
            System.out.println("InvocationTargetException caught");
            System.out.println("Cause: " + e.getCause());
            System.out.println("Cause type: " + e.getCause().getClass());
        }
        
        // Method that throws runtime exception
        Method runtimeMethod = clazz.getMethod("throwsRuntimeException");
        
        try {
            runtimeMethod.invoke(obj);
        } catch (InvocationTargetException e) {
            System.out.println("\nInvocationTargetException caught again");
            System.out.println("Cause: " + e.getCause().getMessage());
        }
        
        /**
         * EXCEPTION HANDLING:
         * 
         * invoke() wraps exceptions in InvocationTargetException
         * 
         * Original exception:
         * - e.getCause()
         * - Always check getCause()
         * 
         * Other exceptions:
         * - IllegalAccessException (access issues)
         * - IllegalArgumentException (wrong args)
         * - NullPointerException (null instance for non-static)
         */
    }
    
    /**
     * Method handle caching
     */
    static class MethodCache {
        private static final Map<String, Method> cache = new HashMap<>();
        
        public static Method getMethod(Class<?> clazz, String name, 
                Class<?>... paramTypes) throws NoSuchMethodException {
            String key = clazz.getName() + "#" + name + "#" + 
                Arrays.toString(paramTypes);
            
            return cache.computeIfAbsent(key, k -> {
                try {
                    Method method = clazz.getMethod(name, paramTypes);
                    method.setAccessible(true);  // If needed
                    return method;
                } catch (NoSuchMethodException e) {
                    throw new RuntimeException(e);
                }
            });
        }
        
        /**
         * METHOD CACHING:
         * 
         * Reflection lookup is expensive
         * Cache Method objects
         * Reuse across calls
         * 
         * Benefits:
         * - Much faster
         * - Avoid repeated lookups
         * - One-time cost
         */
    }
    
    /**
     * Bulk method invocation
     */
    public static void bulkMethodInvocation() throws Exception {
        Calculator calc = new Calculator();
        Class<?> clazz = calc.getClass();
        
        // Get all public methods
        Method[] methods = clazz.getDeclaredMethods();
        
        System.out.println("=== Invoking All No-Arg Methods ===");
        for (Method method : methods) {
            if (method.getParameterCount() == 0 && 
                !Modifier.isStatic(method.getModifiers())) {
                
                try {
                    Object result = method.invoke(calc);
                    System.out.println(method.getName() + " -> " + result);
                } catch (Exception e) {
                    System.out.println(method.getName() + " failed: " + 
                        e.getCause());
                }
            }
        }
        
        /**
         * BULK INVOCATION:
         * 
         * Useful for:
         * - Testing all getters
         * - Calling lifecycle methods
         * - Method discovery and execution
         * - Generic framework code
         */
    }
    
    /**
     * Best practices
     */
    public static void bestPractices() {
        /**
         * METHOD INVOCATION BEST PRACTICES:
         * 
         * 1. CACHE METHOD OBJECTS:
         *    Lookup is expensive
         *    Invoke is relatively cheap
         *    Cache and reuse
         * 
         * 2. HANDLE InvocationTargetException:
         *    Always catch
         *    Check getCause()
         *    Re-throw or handle
         * 
         * 3. VALIDATE ARGUMENTS:
         *    Check parameter count
         *    Check parameter types
         *    Prevent IllegalArgumentException
         * 
         * 4. CHECK RETURN TYPE:
         *    void methods return null
         *    Primitives are boxed
         *    Cast carefully
         * 
         * 5. SETACCESSIBLE ONCE:
         *    Not needed for public methods
         *    Call once, cache result
         *    Don't call on every invoke
         * 
         * 6. CONSIDER METHODHANDLES:
         *    Java 7+ alternative
         *    Better performance
         *    Type-safe
         */
    }
    
    public static void main(String[] args) throws Exception {

        System.out.println("\n=== Static Method Invocation ===");
        staticMethodInvocation();
        
        System.out.println("\n=== Exception Handling ===");
        exceptionHandlingInInvoke();
    }
```

```java
/**
```

```java
import java.lang.reflect.*;
import java.util.*;

public class GenericTypeInspection {
    
    /**
     * Type erasure basics
     */
    public static void typeErasureBasics() {
        /**
         * TYPE ERASURE:
         * 
         * Generics are compile-time only
         * Erased at runtime
         * 
         * List<String> → List
         * Map<String, Integer> → Map
         * 
         * Can't determine generic type from instance:
         * List<String> list = new ArrayList<>();
         * list.getClass() → ArrayList (no <String>)
         * 
         * 
         * EXCEPTIONS:
         * 
         * Generic type info preserved in:
         * 1. Field declarations
         * 2. Method signatures
         * 3. Superclass type parameters
         * 4. Anonymous inner classes
         */
    }
    
    /**
     * Example classes with generics
     */
    static class Container<T> {
        private T value;
        private List<String> stringList;
        private Map<String, Integer> stringIntMap;
        
        public T getValue() { return value; }
        public void setValue(T value) { this.value = value; }
        
        public List<String> getStringList() { return stringList; }
        public Map<String, Integer> getMap() { return stringIntMap; }
    }
    
    /**
     * Inspecting field generic types
     */
    public static void inspectingFieldGenericTypes() throws Exception {
        Class<?> clazz = Container.class;
        
        Field valueField = clazz.getDeclaredField("value");
        Field listField = clazz.getDeclaredField("stringList");
        Field mapField = clazz.getDeclaredField("stringIntMap");
        
        System.out.println("=== Field Generic Types ===");
        
        // Get generic type
        Type valueType = valueField.getGenericType();
        System.out.println("value type: " + valueType);
        System.out.println("  Type class: " + valueType.getClass());
        
        // List field
        Type listType = listField.getGenericType();
        System.out.println("\nstringList type: " + listType);
        
        if (listType instanceof ParameterizedType) {
            ParameterizedType pt = (ParameterizedType) listType;
            System.out.println("  Raw type: " + pt.getRawType());
            System.out.println("  Type arguments: " + 
                Arrays.toString(pt.getActualTypeArguments()));
        }
        
        // Map field
        Type mapType = mapField.getGenericType();
        System.out.println("\nstringIntMap type: " + mapType);
        
        if (mapType instanceof ParameterizedType) {
            ParameterizedType pt = (ParameterizedType) mapType;
            System.out.println("  Raw type: " + pt.getRawType());
            System.out.println("  Type arguments: " + 
                Arrays.toString(pt.getActualTypeArguments()));
        }
        
        /**
         * GENERIC FIELD TYPES:
         * 
         * getGenericType() returns Type
         * 
         * Type subtypes:
         * - Class (regular class)
         * - ParameterizedType (List<String>)
         * - TypeVariable (T)
         * - GenericArrayType (T[])
         * - WildcardType (? extends Number)
         * 
         * ParameterizedType:
         * - getRawType() → List.class
         * - getActualTypeArguments() → [String.class]
         */
    }
    
    /**
     * Inspecting method generic types
     */
    static class GenericMethods {
        public <T> T identity(T value) { return value; }
        
        public <T extends Number> void bounded(T number) {}
        
        public <T> List<T> createList() { return new ArrayList<>(); }

     * TypeToken pattern (like Gson/Guice)

        private final Type type;
        
        protected TypeToken() {
            Type superclass = getClass().getGenericSuperclass();
            if (superclass instanceof ParameterizedType) {
                ParameterizedType pt = (ParameterizedType) superclass;
                this.type = pt.getActualTypeArguments()[0];
            } else {
                throw new IllegalStateException("Must use anonymous class");
            }
        }
        
        public Type getType() {
            return type;
        }
    }
```

```java
/**
 * PERFORMANCE OVERHEAD
 * 
 * Measuring and minimizing reflection cost
 */

import java.lang.reflect.*;
import java.util.*;

public class PerformanceOverhead {
    
    /**
     * Example class for benchmarking
     */
    static class Target {
        private int value = 0;
        
        public int getValue() {
            return value;
        }
        
        public void setValue(int value) {
            this.value = value;
        }
        
        public int add(int a, int b) {
            return a + b;
        }
    }
    
    /**
     * Direct access vs reflection
     */
    public static void directVsReflection() throws Exception {
        int iterations = 1_000_000;
        Target target = new Target();
        
        // Direct access
        long start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            target.setValue(i);
            int value = target.getValue();
        }
        long directTime = System.nanoTime() - start;
        
        // Reflection access
        Method getter = Target.class.getMethod("getValue");
        Method setter = Target.class.getMethod("setValue", int.class);
        
        start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            setter.invoke(target, i);
            int value = (int) getter.invoke(target);
        }
        long reflectionTime = System.nanoTime() - start;
        
        System.out.println("=== Direct vs Reflection (" + iterations + " iterations) ===");
        System.out.println("Direct access:     " + directTime / 1_000_000 + " ms");
        System.out.println("Reflection access: " + reflectionTime / 1_000_000 + " ms");
        System.out.println("Slowdown: " + (reflectionTime / (double) directTime) + "x");
        
        /**
         * TYPICAL RESULTS:
         * 
         * Direct: ~5 ms
         * Reflection: ~50-100 ms
         * Slowdown: 10-20x
         * 
         * Reflection is much slower
         * Especially for simple operations
         */
    }
    
    /**
     * Method lookup vs invocation
     */
    public static void lookupVsInvocation() throws Exception {
        int iterations = 1_000_000;
        Target target = new Target();
        
        // Lookup every time (anti-pattern)
        long start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            Method method = Target.class.getMethod("add", int.class, int.class);
            method.invoke(target, 1, 2);
        }
        long lookupTime = System.nanoTime() - start;
        
        // Cached method (best practice)
        Method cachedMethod = Target.class.getMethod("add", int.class, int.class);
        
        start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            cachedMethod.invoke(target, 1, 2);
        }
        long cachedTime = System.nanoTime() - start;
        
        System.out.println("=== Lookup vs Cached (" + iterations + " iterations) ===");
        System.out.println("Lookup every time: " + lookupTime / 1_000_000 + " ms");
        System.out.println("Cached method:     " + cachedTime / 1_000_000 + " ms");
        System.out.println("Speedup: " + (lookupTime / (double) cachedTime) + "x");
        
        /**
         * TYPICAL RESULTS:
         * 
         * Lookup: ~500 ms
         * Cached: ~50 ms
         * Speedup: 10x
         * 
         * LESSON:
         * Always cache Method/Field/Constructor objects
         * Lookup is very expensive
         * Invocation is relatively cheap
         */
    }
    
    /**
     * setAccessible impact
     */
    public static void setAccessibleImpact() throws Exception {
        int iterations = 1_000_000;
        Target target = new Target();
        
        Method method = Target.class.getDeclaredMethod("add", int.class, int.class);
        
        // Without setAccessible (public method)
        long start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            method.invoke(target, 1, 2);
        }
        long withoutTime = System.nanoTime() - start;
        
        // With setAccessible
        method.setAccessible(true);
        
        start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            method.invoke(target, 1, 2);
        }
        long withTime = System.nanoTime() - start;
        
        System.out.println("=== setAccessible Impact (" + iterations + " iterations) ===");
        System.out.println("Without setAccessible: " + withoutTime / 1_000_000 + " ms");
        System.out.println("With setAccessible:    " + withTime / 1_000_000 + " ms");
        System.out.println("Speedup: " + (withoutTime / (double) withTime) + "x");
        
        /**
         * TYPICAL RESULTS:
         * 
         * Without: ~60 ms
         * With: ~50 ms
         * Speedup: 1.2x
         * 
         * setAccessible(true):
         * - Skips access checking
         * - Slight performance improvement
         * - More important for private members
         */
    }
    
    /**
     * Optimization techniques
     */
    static class OptimizedReflection {
        private static final Map<String, Method> methodCache = new HashMap<>();
        
        public static Object invoke(Object target, String methodName, 
                Object... args) throws Exception {
            // Cache key
            String key = target.getClass().getName() + "#" + methodName;
            
            // Get cached or lookup
            Method method = methodCache.computeIfAbsent(key, k -> {
                try {
                    Method m = target.getClass().getMethod(methodName);
                    m.setAccessible(true);  // Once
                    return m;
                } catch (NoSuchMethodException e) {
                    throw new RuntimeException(e);
                }
            });
            
            // Invoke
            return method.invoke(target, args);
        }
        
        /**
         * OPTIMIZATION TECHNIQUES:
         * 
         * 1. CACHE Method objects
         * 2. Call setAccessible() once
         * 3. Minimize reflection scope
         * 4. Use direct access where possible
         * 5. Consider MethodHandle (Java 7+)
         */
    }
    
    /**
     * MethodHandle alternative (Java 7+)
     */
    public static void methodHandleAlternative() throws Throwable {
        int iterations = 1_000_000;
        Target target = new Target();
        
        // Reflection
        Method method = Target.class.getMethod("add", int.class, int.class);
        
        long start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            method.invoke(target, 1, 2);
        }
        long reflectionTime = System.nanoTime() - start;
        
        // MethodHandle
        MethodHandles.Lookup lookup = MethodHandles.lookup();
        MethodHandle mh = lookup.findVirtual(Target.class, "add",
            MethodType.methodType(int.class, int.class, int.class));
        
        start = System.nanoTime();
        for (int i = 0; i < iterations; i++) {
            int result = (int) mh.invoke(target, 1, 2);
        }
        long handleTime = System.nanoTime() - start;
        
        System.out.println("=== MethodHandle vs Reflection (" + iterations + " iterations) ===");
        System.out.println("Reflection:   " + reflectionTime / 1_000_000 + " ms");
        System.out.println("MethodHandle: " + handleTime / 1_000_000 + " ms");
        System.out.println("Speedup: " + (reflectionTime / (double) handleTime) + "x");
        
        /**
         * METHODHANDLE:
         * 
         * Faster than reflection
         * Type-safe
         * JVM can optimize better
         * 
         * TYPICAL SPEEDUP: 2-5x
         * 
         * Use when:
         * - Performance critical
         * - Java 7+
         * - Can handle complexity
         */
    }
```

### Best Practices Summary

**✅ DO:**

- Cache Method/Field/Constructor objects
- Use setAccessible() once on cached objects
- Handle InvocationTargetException properly
- Check getCause() for actual exception
- Use getDeclaredX() for private members
- Consider MethodHandle for performance
- Validate class names before loading
- Document why reflection is needed

**❌ DON'T:**

- Use Class.newInstance() (deprecated)
- Lookup methods in loops
- Modify final fields (undefined behavior)
- Ignore SecurityException
- Use reflection in hot paths
- Trust untrusted class names
- Break encapsulation unnecessarily
