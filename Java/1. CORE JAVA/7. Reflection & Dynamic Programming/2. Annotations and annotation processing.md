# Annotations and Annotation Processing

## Overview

Annotations are metadata that provide data about code but don't directly affect program execution. They enable declarative programming, reduce boilerplate, and power modern Java frameworks. This guide covers built-in annotations, meta-annotations, creating custom annotations, compile-time processing (APT), runtime processing, inheritance behavior, and real-world annotation frameworks like Spring and JPA.

## 1. Built-in Annotations

### Standard Java Annotations

```java
/**
 * BUILT-IN ANNOTATIONS
 * 
 * Core Java annotations
 */

import java.util.*;

public class BuiltInAnnotations {
    
    /**
     * @Override - Method overriding
     */
    static class Parent {
        public void doSomething() {
            System.out.println("Parent method");
        }
    }
    
    static class Child extends Parent {
        @Override
        public void doSomething() {
            System.out.println("Child method");
        }
        
        // Compile error if uncommented
        // @Override
        // public void doesntExist() {}
        
        /**
         * @OVERRIDE BENEFITS:
         * 
         * 1. Compiler checking - catches typos
         * 2. Documentation - makes intent clear
         * 3. Refactoring safety
         * 
         * ALWAYS use @Override when overriding!
         */
    }
    
    /**
     * @Deprecated - Marking deprecated code
     */
    static class DeprecatedExample {
        @Deprecated
        public void oldMethod() {}
        
        @Deprecated(since = "2.0", forRemoval = true)
        public void veryOldMethod() {}
        
        public void newMethod() {}
    }
    
    /**
     * @FunctionalInterface - SAM interfaces
     */
    @FunctionalInterface
    interface Calculator {
        int calculate(int a, int b);
        
        default int add(int a, int b) { return a + b; }
        static int multiply(int a, int b) { return a * b; }
    }
    
    /**
     * @SuppressWarnings - Suppress compiler warnings
     */
    @SuppressWarnings("unchecked")
    public static void suppressWarnings() {
        List rawList = new ArrayList();
        List<String> stringList = (List<String>) rawList;
    }
    
    /**
     * @SafeVarargs - Varargs with generics
     */
    @SafeVarargs
    public static <T> void printAll(T... items) {
        for (T item : items) {
            System.out.println(item);
        }
    }
}
```

```java
 * META-ANNOTATIONS

import java.lang.annotation.*;

public class MetaAnnotations {
    
    /**
     * @Target - Where annotation can be used
     */
    @Target(ElementType.METHOD)

    /**
     * @Retention - How long annotation is retained
     */
    @Retention(RetentionPolicy.SOURCE)
    @interface SourceOnly {}  // Discarded by compiler
    
    @Retention(RetentionPolicy.CLASS)
    @interface ClassLevel {}  // In .class file (default)
    
    @Retention(RetentionPolicy.RUNTIME)
    @interface RuntimeAvailable {}  // Available via reflection
    
    /**
     * @Documented - Include in Javadoc
     */
    @Documented
    @Retention(RetentionPolicy.RUNTIME)
    @interface PublicAPI {
        String since();
    }
    
    /**
     * @Inherited - Annotation inheritance
     */
    @Inherited
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.TYPE)
    @interface InheritedAnnotation {}
    
    /**
     * @Repeatable - Repeating annotations (Java 8+)
     */
    @Retention(RetentionPolicy.RUNTIME)
    @interface Authors {
        Author[] value();
    }
    
    @Repeatable(Authors.class)
    @Retention(RetentionPolicy.RUNTIME)

        String name();
    }
```

```java
/**
 * CUSTOM ANNOTATIONS

import java.lang.annotation.*;

public class CustomAnnotations {
    
    /**
     * Simple marker annotation
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.METHOD)
    @interface Test {}
    
    /**
     * Single-value annotation
     */
    @Retention(RetentionPolicy.RUNTIME)
    @interface Description {

    }
    
    /**
     * Multi-attribute annotation
     */
    @Retention(RetentionPolicy.RUNTIME)
    @interface Retry {
        int times() default 3;
        long delay() default 1000;
        Class<? extends Exception>[] on() default {Exception.class};
    }
    
    /**
     * Complex annotation
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.TYPE)
    @interface Configuration {
        int timeout() default 5000;
        String name();
        Level level() default Level.INFO;
        Class<?> handler() default Object.class;
        String[] tags() default {};
    }
    
    enum Level { DEBUG, INFO, WARN, ERROR }
    
    /**
     * Usage examples
     */
    static class AnnotatedClass {
        @Test
        public void testMethod() {}
        
        @Description("Important test")
        @Test
        public void describedTest() {}
        
        @Retry(times = 5, delay = 2000)
        public void retryableMethod() {}
    }
    
    /**
     * Nested annotations
     */
    @Retention(RetentionPolicy.RUNTIME)
    @interface Entity {
        String table();
        Index[] indexes() default {};
    }
    
    @Retention(RetentionPolicy.RUNTIME)
    @interface Index {
        String name();
        String[] columns();
        boolean unique() default false;
    }
    
    @Entity(
        table = "users",
        indexes = {
            @Index(name = "idx_email", columns = {"email"}, unique = true)
        }
    )
    static class User {}
}
```

## 5. Runtime Annotation Processing

### Reflection-Based Processing

```java
/**
 * RUNTIME ANNOTATION PROCESSING
 */

import java.lang.annotation.*;
import java.lang.reflect.*;
import java.util.*;

public class RuntimeAnnotationProcessing {
    
    /**
     * Validation framework example
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.FIELD)
    @interface NotNull {
        String message() default "Field cannot be null";
    }
    
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.FIELD)
    @interface Range {
        int min();
        int max();
        String message() default "Value out of range";
    }
    
    static class User {
        @NotNull
        private String name;
        
        @NotNull
        @Range(min = 0, max = 150)
        private Integer age;
        
        @NotNull
        private String email;
        
        public User(String name, Integer age, String email) {
            this.name = name;
            this.age = age;
            this.email = email;
        }
    }
    
    /**
     * Simple validator
     */
    static class Validator {
        public static List<String> validate(Object obj) {
            List<String> errors = new ArrayList<>();
            Class<?> clazz = obj.getClass();
            
            for (Field field : clazz.getDeclaredFields()) {
                field.setAccessible(true);
                
                try {
                    Object value = field.get(obj);
                    
                    // Check @NotNull
                    if (field.isAnnotationPresent(NotNull.class)) {
                        if (value == null) {
                            NotNull ann = field.getAnnotation(NotNull.class);
                            errors.add(field.getName() + ": " + ann.message());
                        }
                    }
                    
                    // Check @Range
                    if (field.isAnnotationPresent(Range.class) && value != null) {
                        Range range = field.getAnnotation(Range.class);
                        int intValue = (Integer) value;
                        
                        if (intValue < range.min() || intValue > range.max()) {
                            errors.add(field.getName() + ": " + range.message() +
                                " [" + range.min() + "-" + range.max() + "]");
                        }
                    }
                    
                } catch (IllegalAccessException e) {
                    errors.add("Cannot access field: " + field.getName());
                }
            }
            
            return errors;
        }
    }
    
    public static void validationExample() {
        System.out.println("=== Validation Example ===");
        
        // Valid user
        User validUser = new User("Alice", 30, "alice@example.com");
        List<String> errors = Validator.validate(validUser);
        System.out.println("Valid user errors: " + errors);
        
        // Invalid user
        User invalidUser = new User(null, 200, null);
        errors = Validator.validate(invalidUser);
        System.out.println("Invalid user errors:");
        for (String error : errors) {
            System.out.println("  " + error);
        }
    }
    
    /**
     * Dependency injection example
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.FIELD)
    @interface Inject {}
    
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.TYPE)
    @interface Component {}
    
    @Component
    static class DatabaseService {
        public void query() {
            System.out.println("Querying database");
        }
    }
    
    @Component
    static class UserService {
        @Inject
        private DatabaseService database;
        
        public void findUser() {
            System.out.println("Finding user...");
            database.query();
        }
    }
    
    static class SimpleDI {
        private static Map<Class<?>, Object> instances = new HashMap<>();
        
        public static <T> T getInstance(Class<T> clazz) {
            if (!clazz.isAnnotationPresent(Component.class)) {
                throw new IllegalArgumentException("Not a component");
            }
            
            @SuppressWarnings("unchecked")
            T instance = (T) instances.computeIfAbsent(clazz, k -> {
                try {
                    T newInstance = clazz.getDeclaredConstructor().newInstance();
                    injectDependencies(newInstance);
                    return newInstance;
                } catch (Exception e) {
                    throw new RuntimeException(e);
                }
            });
            
            return instance;
        }
        
        private static void injectDependencies(Object obj) throws Exception {
            for (Field field : obj.getClass().getDeclaredFields()) {
                if (field.isAnnotationPresent(Inject.class)) {
                    field.setAccessible(true);
                    Object dependency = getInstance(field.getType());
                    field.set(obj, dependency);
                }
            }
        }
    }
    
    public static void dependencyInjectionExample() {
        System.out.println("\n=== Dependency Injection Example ===");
        
        UserService service = SimpleDI.getInstance(UserService.class);
        service.findUser();
    }
    
    /**
     * Test framework example
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.METHOD)
    @interface Test {}
    
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.METHOD)
    @interface BeforeEach {}
    
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.METHOD)
    @interface AfterEach {}
    
    static class TestRunner {
        public static void run(Class<?> testClass) throws Exception {
            Object testInstance = testClass.getDeclaredConstructor().newInstance();
            
            // Find lifecycle methods
            Method beforeEach = findAnnotatedMethod(testClass, BeforeEach.class);
            Method afterEach = findAnnotatedMethod(testClass, AfterEach.class);
            
            // Find and run test methods
            for (Method method : testClass.getDeclaredMethods()) {
                if (method.isAnnotationPresent(Test.class)) {
                    System.out.println("Running: " + method.getName());
                    
                    if (beforeEach != null) beforeEach.invoke(testInstance);
                    
                    try {
                        method.invoke(testInstance);
                        System.out.println("  ✓ PASSED");
                    } catch (InvocationTargetException e) {
                        System.out.println("  ✗ FAILED: " + e.getCause().getMessage());
                    }
                    
                    if (afterEach != null) afterEach.invoke(testInstance);
                }
            }
        }
        
        private static Method findAnnotatedMethod(Class<?> clazz, 
                Class<? extends Annotation> annotation) {
            for (Method method : clazz.getDeclaredMethods()) {
                if (method.isAnnotationPresent(annotation)) {
                    return method;
                }
            }
            return null;
        }
    }
    
    static class SampleTest {
        @BeforeEach
        public void setup() {
            System.out.println("  Setup");
        }
        
        @Test
        public void testSuccess() {
            System.out.println("  Test execution");
        }
        
        @Test
        public void testFailure() {
            throw new RuntimeException("Test failed!");
        }
        
        @AfterEach
        public void teardown() {
            System.out.println("  Teardown");
        }
    }
    
    public static void testFrameworkExample() throws Exception {
        System.out.println("\n=== Test Framework Example ===");
        TestRunner.run(SampleTest.class);
    }
    
    public static void main(String[] args) throws Exception {
        validationExample();
        dependencyInjectionExample();
        testFrameworkExample();
    }
}
```

## 6. Annotation Inheritance

### How Annotations are Inherited

```java
/**
 * ANNOTATION INHERITANCE
 */

import java.lang.annotation.*;

public class AnnotationInheritance {
    
    /**
     * Inherited annotation
     */
    @Inherited
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.TYPE)
    @interface Transactional {
        String value() default "";
    }
    
    /**
     * Non-inherited annotation
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.TYPE)
    @interface Loggable {
        String value() default "";
    }
    
    @Transactional("parent-tx")
    @Loggable("parent-log")
    static class Parent {}
    
    static class Child extends Parent {
        // Inherits @Transactional
        // Does NOT inherit @Loggable
    }
    
    public static void inheritanceDemo() {
        System.out.println("=== Annotation Inheritance ===");
        
        // Check annotations on Child
        Annotation[] annotations = Child.class.getAnnotations();
        System.out.println("Child class annotations: " + annotations.length);
        
        for (Annotation ann : annotations) {
            System.out.println("  " + ann);
        }
        
        // Check specific annotations
        System.out.println("\n@Transactional present: " + 
            Child.class.isAnnotationPresent(Transactional.class));
        System.out.println("@Loggable present: " + 
            Child.class.isAnnotationPresent(Loggable.class));
        
        /**
         * INHERITANCE RULES:
         * 
         * @Inherited only works for:
         * - CLASS annotations
         * - Direct superclass
         * 
         * Does NOT work for:
         * - METHOD annotations
         * - FIELD annotations
         * - INTERFACE implementations
         * - Multiple levels (grandparent)
         */
    }
    
    /**
     * Method annotation inheritance
     */
    @Retention(RetentionPolicy.RUNTIME)
    @Target(ElementType.METHOD)
    @interface Audited {}
    
    static class BaseClass {
        @Audited
        public void auditedMethod() {}
        
        public void normalMethod() {}
    }
    
    static class DerivedClass extends BaseClass {
        @Override
        public void auditedMethod() {
            // Does NOT inherit @Audited
        }
        
        @Audited
        @Override
        public void normalMethod() {
            // Must re-annotate
        }
    }
    
    public static void methodInheritanceDemo() {
        System.out.println("\n=== Method Annotation Inheritance ===");
        
        try {
            Method baseMethod = BaseClass.class.getMethod("auditedMethod");
            Method derivedMethod = DerivedClass.class.getMethod("auditedMethod");
            
            System.out.println("Base method @Audited: " + 
                baseMethod.isAnnotationPresent(Audited.class));
            System.out.println("Derived method @Audited: " + 
                derivedMethod.isAnnotationPresent(Audited.class));
            
        } catch (NoSuchMethodException e) {
            e.printStackTrace();
        }
        
        /**
         * METHOD ANNOTATIONS:
         * 
         * Never inherited automatically
         * Even with @Inherited
         * 
         * To check parent method:
         * 1. Get parent class method
         * 2. Check annotation on parent
         * 3. Manual propagation if needed
         */
    }
    
    /**
     * Interface annotation inheritance
     */
    @Transactional("interface-tx")
    interface MyInterface {
        void doSomething();
    }
    
    static class Implementation implements MyInterface {
        @Override
        public void doSomething() {
            // Does NOT inherit @Transactional
        }
    }
    
    public static void interfaceInheritanceDemo() {
        System.out.println("\n=== Interface Annotation Inheritance ===");
        
        System.out.println("Interface @Transactional: " + 
            MyInterface.class.isAnnotationPresent(Transactional.class));
        System.out.println("Implementation @Transactional: " + 
            Implementation.class.isAnnotationPresent(Transactional.class));
        
        /**
         * INTERFACE ANNOTATIONS:
         * 
         * NOT inherited by implementing classes
         * Even with @Inherited
         * 
         * @Inherited only for classes extending classes
         */
    }
    
    public static void main(String[] args) {
        inheritanceDemo();
        methodInheritanceDemo();
        interfaceInheritanceDemo();
    }
}
```

```java
    }
    
    /**
     * Framework implementation pattern
     */
    static class SimpleFramework {
        public static void register(Class<?> controllerClass) {
            if (!controllerClass.isAnnotationPresent(Controller.class)) {
                throw new IllegalArgumentException("Not a controller");
            }
            
            Controller controller = controllerClass.getAnnotation(Controller.class);
            String basePath = controller.path();
            
            System.out.println("Registering controller: " + basePath);
            
            for (var method : controllerClass.getDeclaredMethods()) {
                if (method.isAnnotationPresent(Route.class)) {
                    Route route = method.getAnnotation(Route.class);
                    String fullPath = basePath + route.path();
                    
                    System.out.println("  " + route.method() + " " + fullPath + 
                        " -> " + method.getName() + "()");
                }
            }
        }
    }
    
    public static void main(String[] args) {
        System.out.println("=== Custom Framework Example ===");
        SimpleFramework.register(UserController.class);
    }
}
```

## Summary

### Quick Reference

**Built-in Annotations:**

```java
@Override           // Method overrides parent
@Deprecated         // Marked for removal
@FunctionalInterface // SAM interface
@SuppressWarnings   // Suppress compiler warnings
@SafeVarargs        // Safe varargs
```

**Meta-Annotations:**

```java
@Target(ElementType.METHOD)     // Where annotation applies
@Retention(RetentionPolicy.RUNTIME) // How long it's kept
@Documented                     // Include in Javadoc
@Inherited                      // Inherited by subclasses
@Repeatable(Container.class)    // Can repeat
```

**Custom Annotation:**

```java
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
@interface MyAnnotation {
    String value();
    int priority() default 0;
}
```

**Reading Annotations:**

```java
// Check presence
if (method.isAnnotationPresent(MyAnnotation.class)) {
    // Get annotation
    MyAnnotation ann = method.getAnnotation(MyAnnotation.class);
    String value = ann.value();
}

// Get all annotations
Annotation[] annotations = clazz.getAnnotations();

// Get repeating annotations
Author[] authors = clazz.getAnnotationsByType(Author.class);
```

### Comparison Tables

**Retention Policies:**

```
┌─────────┬────────┬─────────┬──────────────┐
│ Policy  │ Source │ Bytecode│ Runtime      │
├─────────┼────────┼─────────┼──────────────┤
│ SOURCE  │ ✓      │ ✗       │ ✗            │
│ CLASS   │ ✓      │ ✓       │ ✗            │
│ RUNTIME │ ✓      │ ✓       │ ✓            │
└─────────┴────────┴─────────┴──────────────┘
```

**When to Use:**

```
Compile-time (APT)    → Retention.SOURCE or CLASS
Runtime reflection    → Retention.RUNTIME
Documentation         → @Documented
Class inheritance     → @Inherited
Method/field          → No inheritance mechanism
```

### Best Practices Summary

**✅ DO:**

- Use @Override for all overridden methods
- Provide sensible defaults for annotation attributes
- Use RetentionPolicy.RUNTIME for framework annotations
- Document annotations with Javadoc
- Use specific @Target to prevent misuse
- Cache annotation lookups for performance
- Validate annotation usage at compile-time (APT)
- Use existing frameworks before rolling your own

**❌ DON'T:**

- Forget @Override (catches refactoring errors)
- Use @SuppressWarnings("all")
- Rely on annotation inheritance for methods
- Make annotations too complex
- Use reflection in hot paths
- Ignore compile-time validation opportunities
- Recreate what exists (validation, DI, etc.)
